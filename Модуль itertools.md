## Бесконечные итераторы
### `itertools.count()`
создаёт объект-итератор с бесконечной последовательностью чисел, с определённой последовательностью и началом.
`itertools.count(start, step)`, где `start` начальное значение, а `step` — интервал до следующего значения.

### `itertools.cycle()`
создаёт объект-итератор с бесконечным перебором всех значений в итерируемом множестве. принимает значение множества для перебора

### `itertools.repeat()`
создаёт объект-итератор, возвращающий один-единственный элемент бесконечное количество раз, если не было указано точное количество итераций. Синтаксис функции следующий `itertools.repeat(object, times)`, где `object` возвращаемое значение, а `times` — количество итераций, по умолчанию — бесконечное значение.

## Комбинаторные итераторы
### `itertools.product()`
создаёт объект-итератор, определяющий все возможные последовательности элементов двух множеств, реализуя декартово произведение множеств. `itertools.product(*iterables, repeat=1)`, где на месте `*iterables` через запятую перечислены все коллекции (списки, кортежи, множества или другие итерируемые объекты), а параметр `repeat` указывает, какое количество раз возможно задействовать исходные множества. По умолчанию это значение равно 1.

### `itertools.permutations()`
создаёт объект-итератор, содержащий все возможные уникальные варианты сочетания элементов одного множества с учётом их последовательности и уникальности. В одной последовательности не может быть несколько одних и тех же элементов множества. Функция имеет следующий синтаксис: `itertools.permutations(iterable, r=None)`, где `iterable` — множество значений, а параметр `r` определяет длину будущих сочетаний множеств. По умолчанию этот параметр = `None` и соответствует количеству элементов во множестве.

### `itertools.combinations()`
создаёт объект-итератор, содержащий все возможные уникальные варианты сочетания элементов множества заданной длины без дублирования значений и учёта их последовательности, в порядке их расположения в исходном множестве. Функция имеет следующий синтаксис: `itertools.combinations(iterable, r)`, где `iterable` — множество значений, а параметр `r` определяет длину будущих сочетаний множеств и должен быть обязательно указан. Это значение не может быть больше, чем общее количество элементов в исходном множестве.

### `itertools.combinations_with_replacement()`
(комбинации с замещением) создаёт объект-итератор, содержащий все возможные варианты сочетания элементов множества заданной длины с возможностью дублирования друг друга. Синтаксис функции `combinations_with_replacement()` такой же, как функции `combinations()`.
