В булевом типе данных хранят логические значения и представляют результаты логических выражений.

Логические значения часто представляют как числа: `True` — это 1, а `False` — 0. В Python логические значения можно явно преобразовать в целочисленные (`int`):
```python
print(int(False), int(True)) 

0 1 
```

В других операциях логические значения тоже ведут себя как числа.
```python
is_buyer = [True, True, False, True, False, False, False, False, True, True, False]
print(f'Всего посетителей: {len(is_buyer)}, из них совершили покупку {sum(is_buyer)}') 

Всего посетителей: 11, из них совершили покупку 5
```

## Преобразование из других типов данных
Функция `bool()` возвращает значение `False`, если в исходных данных:
- число 0;
- что-то пустое: строка, список или пустой кортеж;
- значение None.

В остальных случаях функция `bool()` вернёт `True`.

## Функции-предикаты
**Функции-предикаты** возвращают булево значение (`True` или `False`) и помогают проверить условия или выполнить логические операции.

Примеры функций-предикатов:
- `islower()` — возвращает `True`, если в строке нет прописных букв;
- `isdigit()` — возвращает `True`, если строка состоит только из числовых знаков;
- `isalpha()` — возвращает `True`, если в строке только буквы, без пробелов и знаков препинания;
- `isspace()` — возвращает `True`, если в строке только пробелы.

Если условие не соблюдено, эти функции вернут `False`.

Ещё примеры функций-предикатов

Строковые методы, которые проверяют соответствие начала и окончания строки заданному набору символов:

- `startswith(prefix)` — возвращает `True`, если строка начинается с указанного набором символов `prefix`;
- `endswith(suffix)` — возвращает `True`, если строка заканчивается указанным набором символов `suffix`;

```
print('hello'.startswith('h')) # проверяем, начинается ли hello с h
print('hello'.endswith('O')) # проверяем, заканчивается ли hello символом O 

True
False 
```

Хотя `hello` заканчивается `o`, но в функции мы уточняем про другой символ — заглавную `O`. Эта функция чувствительна к регистру.

Функции `all()` и `any()` работают с пустыми значениями в списках и других множествах:

- `all()` — возвращает `True`, если все значения списка `True`. Например, функция поможет проверить, что заполнены все обязательные поля в форме регистрации. Она вернёт `True`, если всё заполнено, и `False`, если чего-то недостаёт.
- `any()` — возвращает `True`, если хотя бы одно значение списка `True`. Например, клиент может указать разные способы связи: номер телефона, имейл, ник в Телеграме. Эта функция поможет проверить, что указан хоть один способ связи. Она вернёт `True`, если клиент указал, как с ним можно связаться, и `False`, если клиент не указал ни одного способа.

```
print(all(['A','B',None,'D'])) 
print(any(['A','B',None,'D'])) 

False
True 
```
