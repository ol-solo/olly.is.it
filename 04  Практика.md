# Расчёт по группам. Практика

Группировка данных позволяет сегментировать данные по одной или нескольким переменным и применять к ним различные агрегирующие функции. В pandas для группировки данных используется метод `groupby()`. Вот основные концепты и советы по использованию группировки данных:

1. Основной синтаксис:

```
grouped = df.groupby('колонка группировки') 
```

Результат — это объект `GroupBy`, к которому можно применить агрегирующие функции.

2. Агрегирующие функции:
    - `mean()`, `sum()`, `min()`, `max()` и другие — позволяют рассчитать среднее, сумму, минимум, максимум и другие агрегаты для каждой группы;
    - `agg()` — используется для выполнения нескольких агрегаций одновременно или при необходимости применить пользовательские функции.
3. Группировка по нескольким ключам: Можно группировать данные по нескольким столбцам, передавая список:

```
grouped = df.groupby(['колонка группировки1', 'колонка группировки2']) 
```

4. Параметры `groupby()`:
    - `as_index`: если `True` (по умолчанию), ключи группировки используются как индексы нового объекта. Если `False`, ключи группировки добавляются как обычные столбцы.
    - `sort`: контролирует, следует ли сортировать результаты группировки по ключам. По умолчанию `True`.
5. Сброс индексов: После группировки и агрегации результат часто возвращается с мультииндексом. Использование `reset_index()` позволяет преобразовать мультииндекс в столбцы и сделать данные более читаемыми:

```
    result = df.groupby('колонка группировки')\
               .agg({'колонка расчётов': 'метод агрегации'})\
               .reset_index()
     
```

6. Использование `agg()` с словарями: Метод `agg()` можно использовать со словарями для указания агрегаций для разных столбцов:

```
    result = df.groupby('колонка группировки')\
               .agg({'column1': 'sum', 'column2': 'mean'})
     
```

7. Использование `agg()` с кортежем: Метод `agg()` можно использовать с кортежами для указания агрегаций для разных столбцов:

```
    result = df.groupby('колонка группировки')\
               .agg(новая_колонка=('column1', 'mean'))
     
```

# Сводные таблицы. Практика

Каждая сводная таблица в pandas базируется на следующих ключевых компонентах:

1. `Индексные колонки (Index columns)` — одна или несколько колонок, значения которых используются для формирования строк результирующей таблицы. Каждая уникальная комбинация значений в этих колонках формирует отдельную строку.
2. `Колонки значений (Values columns)` — колонка или колонки, данные которых агрегируются. Чаще всего это числовые колонки, но в зависимости от функции агрегации, могут использоваться и нечисловые данные.
3. `Функция агрегации (Aggregation function)` — функция, применяемая к данным колонки значений. В случае сводных таблиц могут быть использованы различные функции (`'mean'`, `'min'`, `'max'`, `'sum'` и т.д.) для вычисления агрегированных значений по каждой группе.
4. `Колонки для группировки (Columns)` — опциональный параметр, который позволяет дополнительно разделить агрегированные данные по другому измерению, формируя столбцы результирующей таблицы.

```
df.pivot_table(
    values= 'колонка значений', 
    index=  'индексные колонки', 
    columns='колонки для группировки', 
    aggfunc='функция агрегации') 
```

Когда мы опускаем один из параметров, мы получаем разные виды агрегаций:

- без указания `aggfunc` применяется метод `mean`;
- без `values` pandas попытается агрегировать все числовые столбцы;
- без `columns` данные будут агрегированы только по индексам;
- без `index` данные будут агрегированы в общую сумму по столбцам.

**Переименование столбцов**

```
# Переименование столбцов
pivot_renamed = pivot_example\
                 .rename(columns={
                   'колонка1': 'новая_колонка1',
                   'колонка2': 'новая_колонка1',
                   'колонка3': 'новая_колонка1'
                   }) 
```

**Сброс мультииндекса**

```
# Объединяем названия уровней столбцов в одну строку
multi_level_pivot.columns = ['_'.join(col).strip() 
                             for col in multi_level_pivot.columns.values]

# Избавляемся от мультииндекса
multi_level_df = multi_level_pivot.reset_index() 
```
