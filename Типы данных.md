# Что такое тип данных
Тип данных в Python — это характеристика значения (объекта), которая определяет, какого рода данные оно хранит и какие операции с ним можно выполнять.

Более формально: в Python тип данных — это класс, а конкретные значения являются экземплярами этих классов; именно тип задаёт структуру значения, допустимые методы и поведение в выражениях и операциях.

Узнать, какой тип Python использует для хранения объекта, поможет функция `type()`:

```python
number_int = 123 #Это целое число
number_float = 1.2345 #Это число с плавающей точкой

print(type(number_int))
print(type(number_float)) 

<class 'int'>
<class 'float'> 
```

В Python не нужно специально указывать, какой тип данных ему использовать — он сам справляется с этой задачей, когда создаёт переменную. Это свойство называют динамической типизацией. Благодаря этой особенности писать код на Python удобнее.

|**Тип**|**Примеры**|**Как обозначается**|
|---|---|---|
|Вещественный|`-10.0`, `.1`, `1.23`|`float`|
|Целочисленный|`-1`, `12`, `123`|`int`|
|Логический|`True` / `False`|`bool`|

|Словарь|`{'a': 1}, {1: 'one'}`|`dict`|

|Массив байт|`bytearray(b'abc')`|`bytearray`|
|Комплексное число|`1+2j, -0.5j`|`complex`|

## Вещественный
### Что входит
Дробные числа  хранят в типе `float`. Этот тип называют ещё **числом с плавающей точкой**.

### Как работать
1. Арифметические операции:
    - `+` — сложение: `1.5 + 2.5`;
    - `-` — вычитание: `3.0 - 0.5`;
    - `*` — умножение: `2.0 * 3.5`;
    - `/` — деление (результат всегда `float`): `5 / 2`;
    - `**` — степень: `2.0 ** 3`;
    - `//` — целочисленное деление: `5.5 // 2`;
    - `%` — остаток от деления: `5.5 % 2`.

2. Сравнения: `>`, `<`, `>=`, `<=`, `==`, `!=`.

## Целочисленный
### Что входит
Целые числа хранят в типе `int`.

### Как работать
1. Арифметические операции:
    - `+` — сложение: `1.5 + 2.5`;
    - `-` — вычитание: `3.0 - 0.5`;
    - `*` — умножение: `2.0 * 3.5`;
    - `/` — деление (результат всегда `float`): `5 / 2`;
    - `**` — степень: `2.0 ** 3`;
    - `//` — целочисленное деление: `5.5 // 2`;
    - `%` — остаток от деления: `5.5 % 2`.

2. Сравнения: `>`, `<`, `>=`, `<=`, `==`, `!=`.

3. Операции сдвига и битовые: `<<`, `>>`, `&`, `|`, `^`, `~`.

## Логический тип
### Что входит
Объекты логического типа (булевый, bool) хранят значения `True` или `False`. Обычно такие данные — флаг для какого-то признака.

### Как работать
1. Логические операции: 

|**Оператор**|**Что означает**|**Как работает**|
|---|---|---|
|and|И|даёт истину, только если и справа и слева от него истина|
|or|ИЛИ|даёт в результате истину, если справа или слева от него есть истина|
|not|НЕ|меняет истину на ложь, и наоборот|

2. Сравнения:
`==`, `!=` (часто как результат выражений, например `3 > 2` даёт `True`).

3. Арифметические операции:
В таком случае Python считает, что `True` это `1`, а `False` это `0`.

## Строка
### Что входит
Внутри строки могут быть любые символы: буквы, цифры и прочие символы. Строки всегда записывают в кавычках.
Если текст не умещается в одной строке кода, можно использовать тройные кавычки:

```python
pushkin_text = """Я помню чудное мгновенье:
Передо мной явилась ты,
Как мимолетное виденье,
Как гений чистой красоты.
"""
```

### Как работать
1. “Арифметические” операции:
    - **Конкатенация** (сложение строк):
    `+` — соединяет строки
    - **Повторение** строки:
    `* `— умножение на целое число

```python
title = 'Название магазина'
shop_name = 'Моторизированный котокраб'

print(title + shop_name)

print("=" * 20) # Повторим символ 20 раз
```

> [!important]
> Невозможно, однако, сложить строку и число!

2. Индексация и срезы: `s[0], s[1:4]` — доступ к символам и подстрокам.

3. Сравнения: `==, !=, <, >` — лексикографически (по алфавиту/коду символов).

4. Методы: lower(), upper(), replace(), split(), join() и т.д.

# Какие бывают типы данных в datetime (стандартная библиотека)

|**Тип**|**Примеры**|**Как обозначается**|
|---|---|---|
|Дата и время|Полная метка времени (дата + время)|`datetime.datetime`|
|Дата|Только дата (год, месяц, день)|`datetime.date`|
|Время|Только время (часы, минуты, секунды)|`datetime.time`|
|Разность во времени|Интервал между датами/временем|`datetime.timedelta`|
