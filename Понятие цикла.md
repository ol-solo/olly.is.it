## Цикл
Цикл - это конструкция для повторения действий в коде. 
Итерация цикла - шаги действий со следующим элементом списка. Общее количество итераций цикла будет равно количеству элементов в списке.
Скрипт - набор команд или инструкций, прописанных в коде для решения задач.

## Элементы цикла
Кортеж - последовательность элементов, перечисленных через запятую и заключённых в круглые скобки, например, `tuple = 'A','B','C')`. Кортеж похож на список — его элементы могут иметь различный тип. Однако кортеж, в отличие от списка, представляет неизменяемую структуру данных — в нём нельзя добавлять или удалять элементы.

Цикл `for` повторяет заданную последовательность инструкций для каждого итерируемого объекта, например, списка, кортежа, строки.
**Заголовок цикла** — его первая строка, которая должна содержать:
- Ключевое слово `for`;
- Переменную-итератор, которая перемещается по множеству значений слева направо, принимая на каждом шаге значение очередного элемента списка; 
- Ключевое слово `in`;
- Множество значений, по которому движется итератор; 
- Знак `:`. Без него будет синтаксическая ошибка, а сам цикл не будет объявлен.
**Тело цикла** — набор инструкций, которые выполняются на каждом шаге цикла. Каждая строка тела цикла обязательно отделяется отступом в четыре пробела. Так Python сможет определить, где начинается блок инструкций и где он заканчивается. А вот если тело цикла не отделить четырьмя пробелами, то инструкции выполнятся вне цикла. Если же количество пробелов будет иным — возникнет синтаксическая ошибка.
Тело цикла обязательно должно содержать хотя бы одну инструкцию. Если необходимо оставить цикл пустым, используют команду `pass`, которая не выполняет никаких действий, но и не позволяет всему коду остановиться из-за ошибки выполнения.

Имя переменной-итератора в цикле можно задать любое. Традиционно эти имена образуют от имени списка элементов в единственном числе: например, если имя списка `friends`, то имя итератора дают `friend`. Если переменная-итератор перебирает числовые последовательности, то в качестве имени итератора обычно используют латинские буквы `i`, `j` или `k`, поскольку ряд чисел не означает количество чего-либо и является служебными значениями, необходимыми для работы цикла.
```python
friends = ['Юля', 'Гена', 'Гоша', 'Алиса', 'Слава'] # список друзей

for friend in friends: # заголовок цикла
    print('Имя друга:', friend) # тело цикла
```
Циклы можно применять не только к спискам. Например, можно перебирать строки. В этом случае переменная-итератор будет последовательно принимать значение каждого элемента строки:
```python

    print(symbol)
```

## Итератор цикла и количество итераций
Итератор последовательно перемещается по элементам списка, перебирая их слева направо. Цикл автоматически прекратит работу, когда переберёт все элементы списка.
![[Pasted image 20260303103638.png]]
В этом случае количество итераций цикла равно количеству элементов в списке.

Если в теле цикла происходит изменение значения итератора, то это не изменит сам элемент исходного списка.
```python
prices = [1500, 3100, 7500, 4200]
for price in prices:
     price += 50
     print(f'Новая цена товара: {price}')
print('Значения списка prices:', prices) 

Новая цена товара: 1550
Новая цена товара: 3150
Новая цена товара: 7550
Новая цена товара: 4250
Значения списка prices: [1500, 3100, 7500, 4200] # изменения остались внутри цикла и не повлияли на другие переменные основного кода
```

Внутри цикла `for` лучше избегать изменения итератора, поскольку в сложных циклах это может привести к различным ошибкам в работе кода. Это позволит использовать значение итератора несколько раз за одну итерацию:
```python
prices = [1500, 3100, 7500, 4200]
for price in prices:
     new_price = price + 50
     print(f'Старая цена товара: {price}, новая цена товара: {new_price}') 

Старая цена товара: 1500, новая цена товара: 1550
Старая цена товара: 3100, новая цена товара: 3150
Старая цена товара: 7500, новая цена товара: 7550
Старая цена товара: 4200, новая цена товара: 4250 
```
В теле цикла мы выполнили последовательно несколько инструкций и назначили новую переменную `new_price`. При этом эта переменная будет доступна только внутри тела цикла и будет задаваться каждый раз заново на новой итерации цикла. В таком случае переменную `new_price` называют локальной переменной цикла.

По видимости переменные могут быть локальными и глобальными, а выбор видимости зависит от области видимости. Область видимости определяет, в какой части программы переменные могут быть назначены, изменены и прочитаны. Существует две основные области:
- глобальная, в которой задаются переменные, доступные в любой части кода, и
- локальные — которые выполняются в строго определённой части кода, например, в цикле или функциях.
Глобальные переменные можно использовать везде внутри программы, а локальные переменные будут доступны только в пределах локальной области, в которой они были заданы.

## Функция `range()`
Чтобы задать конкретное количество итераций цикла, вызывают функцию `range()` -  по мере обращения к ней возвращает последовательность целых чисел длиной `n` в диапазоне от `0` до `n-1`. Правая граница — значение `n` исключено из диапазона значений, возвращаемого функцией.
```python
price = 100 # исходная стоимость товара
coeff = 1.05 # коэффициент увеличения стоимости на 5%

for i in range(0,4): # в заголовке установим количество итераций, равное 4
    price *= coeff # увеличиваем исходное значение на 5% на каждом шаге цикла
    print(f'Стоимость товара в квартале {i+1} равна {round(price,2)} руб.') 

Стоимость товара в квартале 1 равна 105.0 руб.
Стоимость товара в квартале 2 равна 110.25 руб.
Стоимость товара в квартале 3 равна 115.76 руб.
Стоимость товара в квартале 4 равна 121.55 руб. 
```

Цикл `for` на каждом шаге берёт значение `price` и увеличивает его на заданный коэффициент. Поскольку переменная `price` глобальная, то есть была задана в основной части кода, то изменения переменной сохраняются: в последующей итерации цикл будет обрабатывать изменённое на предыдущей итерации значение `price`.

## Перебор индексов элементов списка в цикле
Если в функцию `range()` передать длину списка — результат работы функции `len()`, то последовательность чисел будет соответствовать индексам списка.
```python
friends = ['Юля', 'Гена', 'Гоша', 'Алиса', 'Слава'] 

for i in range(len(friends)): 
    print('Имя друга:', friends[i]) 

Имя друга: Юля
Имя друга: Гена
Имя друга: Гоша
Имя друга: Алиса
Имя друга: Слава 
```

Итератор `i` перемещается по последовательности целых чисел от `0` до `len(friends) - 1`. А в теле цикла по значению итератора выводится элемент с индексом итератора — `i`: `friends[i]`.

Использование индекса в качестве итератора позволяет изменять значения элементов самого списка.
```python
prices = [1500, 3100, 7500, 4200]
print('Значения списка prices до изменения:', prices)

# уменьшим каждое значение списка prices на 100 единиц:
for i in range(len(prices)): 
     prices[i] -= 100
print('Результат изменения значений списка prices:', prices) 

Значения списка prices до изменения: [1500, 3100, 7500, 4200]
Результат изменения значений списка prices: [1400, 3000, 7400, 4100] 
```
Здесь цикл `for` на каждом шаге хранит очередной индекс элемента в переменной-итераторе `i`. В теле цикла алгоритм берёт элемент списка `prices` с индексом `i` и уменьшает его значение с помощью оператора `-=`.

## Перебор определённого интервала индексов элементов списка в цикле
С помощью функции `range()` можно перебрать заданный интервал индексов элементов множества. Для этого нужно передать в функцию `range()` два аргумента — начальное значение и конечное.
```python
friends = ['Юля', 'Гена', 'Гоша', 'Алиса', 'Слава'] 

for i in range(0,3): 
    print('Имя друга:', friends[i]) 

Имя друга: Юля
Имя друга: Гена
Имя друга: Гоша 
```

## Перебор индексов элементов строки в цикле
можно обращаться к буквам или любым другим символам строки через их индексы.
```python
string = '001642 Кизильник цельнокрайний (обыкновенный), 50 кустов'
for i in range(6): # переберём первые 6 элементов строки
    print(string[i], end="") 

001642 
```

Такой перебор значений строки можно использовать для создания новой строки:
```python
string = '001642 Кизильник цельнокрайний (обыкновенный), 50 кустов'
order_id = '' # зададим пустую строку
for i in range(6): # переберём первые 6 элементов строки
    order_id += string[i]
print(f'Номер заказа: {order_id}') 

Номер заказа: 001642 
```
В результате работы цикла сформировалась новая строка, которая содержит только номер заказа. Обратите внимание, что перед циклом мы создали пустую строку `order_id`, к которой на каждой итерации цикла добавляли значение строки по индексу — значению переменной-итератора от 0 до 5.
```python
string = '164200, г. Няндома, Няндомский р-н, Архангельская обл.' 
result = '' 
for i in range(11,18): result += string[i] 
print(result)

Няндома
```

## Изменение значений строки с помощью цикла
Строки — это уникальные неизменяемые объекты, и мы не можем менять символы в строке, используя индекс элемента, как это делали в случае со списками. Однако, чтобы изменить конкретные значения исходной строки, можно создать новую строку, которая будет содержать изменённые значения.
```python
string = 'АП ЧЕХОВ' # исходная строка
new_string = '' # создадим пустую строку
new_string += string[3].upper() # добавим к новой строке первый символ фамилии

for i in range(4, len(string)): # переберём в цикле остальную часть фамилии
	new_string += string[i].lower() # переведём символы в нижний регистр

new_string += ' ' # добавим разделитель фамилии и инициалов

for i in range(0, 2): # добавим инициалы и после них точку 
	new_string += string[i].upper() + '.'

print(f'Исходная строка: {string}') 
print(f'Новая строка: {new_string}')

Исходная строка: АП ЧЕХОВ 
Новая строка: Чехов А.П.
```

## Изменение значения исходной строки после определённого символа
Бывают ситуации, когда строки имеют разное количество символов. Например, есть строки, где указаны только имена и фамилии авторов:
```python
author = ['Джон Стейнбек',
          'Дэниел Киз'] 
```
Мы не знаем точное расположение фамилии автора в каждой строке. Однако, чтобы провести какие-то преобразования исходной строки, мы можем найти определённый символ в строке и использовать его индекс.

Например, чтобы выделить фамилию автора в отдельную строку, найдём положение разделяющего слова пробела. Зная его индекс, мы сможем выделить фамилию в отдельную строку.
```python
author = 'Джон Стейнбек'

space_position = ''
for i in range(len(author)): # объявим цикл, перебирающий исходную строку по индексу
	if author[i] == ' ': # зададим условие, сравнивающее значение символа с пробелом 
		space_position = i 
		
print(space_position)

4
```

Цикл выше перебрал все элементы строки, сравнивая их с пробелом, и установил значение переменной `space_position`, равное индексу пробела. Обратите внимание, что если бы в исходной строке было бы несколько пробелов, то значение переменной `space_position` приобрело бы значение индекса последнего пробела в исходной строке. Но поскольку в примере выше количество пробелов единичное, то будем использовать `space_position` дальше для сохранения в новой строке фамилии автора.
```python
author = 'Джон Стейнбек'

space_position = ''
for i in range(len(author)):
    if author[i] == ' ': 
        space_position = i
        
last_name = '' # создадим новую пустую строку
for i in range(space_position+1, len(author)): # объявим новый цикл
    last_name += author[i] # сохраним значения исходной строки в новую строку

print(f'Фамилия автора: {last_name}')

Фамилия автора: Стейнбек
```

Отобразить необходимую часть строки после определённого символа по индексу можно, используя срез строки по индексу:
```python
author = 'Джон Стейнбек'

space_position = ''
for i in range(len(author)):
    if author[i] == ' ': 
        space_position = i
        
last_name = author[space_position+1:] # используем срез строки 
```

## Вложенные циклы
У нас есть список инициалов и фамилий, который нужно отформатировать в едином стиле. В таком случае можно использовать несколько циклов, которые будут вложены один в другой. Первый цикл будет перебирать строки в списке `names`, а вложенные — изменять написание имени, то есть работать со строками
```python
names = ['АП ЧЕХОВ','ФМ ДосТОЕВский','ма шолохов'] # исходный список
new_names = [] # создадим пустой список, в который будут помещены новые элементы 

for name in names: # объявим первый цикл, перебирающий строки исходного цикла
	new_string = '' # создадим пустую строку 
	new_string += name[3].upper() # добавим к новой строке первый символ фамилии

		# объявим первый вложенный цикл: 
	for i in range(4, len(name)): # переберём в цикле остальную часть фамилии
			new_string += name[i].lower() # переведём символы в нижний регистр
		
	new_string += ' ' #добавим разделитель фамилии и инициал

	for i in range(0, 2): # добавим инициалы и после них точку
		new_string += name[i].upper() + '.'
	
	new_names.append(new_string) # в теле основного цикла добавим строку в новый список

print(f'Исходный список: {names}') 
print(f'Новый список: {new_names}')

Исходный список: ['АП ЧЕХОВ', 'ФМ ДосТОЕВский', 'ма шолохов'] 
Новый список: ['Чехов А.П.', 'Достоевский Ф.М.', 'Шолохов М.А.']
```

Цикл, который содержит в себе другой цикл (или несколько других циклов), называется **внешним**, или циклом первого уровня. Вложенные циклы также могут называться вложенными или **внутренними** циклами, или циклами второго, третьего и т. д. уровней.
![[Pasted image 20260303140231.png]]

> [!important]
> В Python нет ограничения на количество вложенных списков. Однако необходимо помнить, что много вложенных циклов делает код сложным для понимания.

За одну итерацию внешнего цикла, внутренние циклы совершают все свои итерации. Поэтому общее количество итераций при использовании вложенных циклов будет равно произведению числа самостоятельных итераций всех циклов.

Для создания нового списка мы использовали метод `append()`. Этот метод позволяет добавлять элемент в конец списка, к которому метод применяется. Поэтому в начале кода мы создали новый пустой список `new_names`, а в теле цикла последовательно добавляли новые значения в этот список.
