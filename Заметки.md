`products['price_per_one']`  
Это объект `Series` — **один столбец как вектор значений**.  
Можно думать: «просто колонка чисел».
```python
import pandas as pd
df = pd.DataFrame({'price_per_one': [10, 20, 30, 40, 50]})
s = df['price_per_one']
print(s)
print(type(s))
print(s.mean())

0    10
1    20
2    30
3    40
4    50
Name: price_per_one, dtype: int64
<class 'pandas.core.series.Series'>
30.0
```

`products[['price_per_one']]`  
Это объект `DataFrame` — **таблица с одним столбцом**.  
Формально всё ещё двумерная структура (строки × столбцы).
```python
import pandas as pd
df = pd.DataFrame({'price_per_one': [10, 20, 30, 40, 50]})
df1 = df[['price_per_one']]
print(df1)
print(type(df1))
print(df1.mean())

   price_per_one
0             10
1             20
2             30
3             40
4             50
<class 'pandas.core.frame.DataFrame'>
price_per_one    30.0
dtype: float64
```
