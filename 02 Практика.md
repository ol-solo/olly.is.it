# Циклы и списки. Практика
### Шпаргалка
Встроенные функции:
- `sum()` используется для суммирования значений: она принимает в качестве аргумента список и возвращает сумму его элементов,

Факториал числа `n` — произведение всех целых чисел от 1 до `n`. В математике факториал обозначают знаком `!`.

# Циклы с генераторами. Практика
### Шпаргалка
Для изменения исходных значений в списке, создания отдельных списков или словарей на основе существующих, для ранжирования данных или проведения сложных расчётов используется функция `enumerate()`.

Используя функцию `enumerate()`, можно изменять значения исходного списка по индексу по аналогии с функцией `range()`.

Функция `enumerate()` возвращает индекс и значение всех элементов списка или другого итерируемого объекта. Это позволяет уже в самом начале цикла определить как индекс элемента, так и его значение.

Функция `zip()` позволяет получать пары, тройки и так далее значений из нескольких списков. В функции через запятую можно перечислить списки. Первым элементом возвращаемой генератором последовательности будут первые значения во всех списках, вторым — вторые и так далее.

### Задание 1
Анализируя стоимость товаров в онлайн-магазине, сотрудники обнаружили ошибку. В сохранённых ценах есть два лишних нуля, а значит, стоимость товаров завышена в 100 раз.
Необходимо исправить эту ошибку: используя цикл `for` и функцию `enumerate`, измените значения исходного списка `prices` так, чтобы скорректировать значения выше 10000, то есть поделить их на 100. В конце выведите на экран результат преобразования списка `prices`.
```python
prices = [850, 700, 95000, 120000, 890, 990, 1150]

# напишите ваш код здесь
for i, price in enumerate(prices):
    if price > 10000:
        prices[i] = price / 100

print(prices)
```

# Циклы while. Практика
### Шпаргалка
Цикл `while` будет повторять последовательность действий, пока условие будет истинным. Остановка цикла `while` произойдёт, когда условие станет ложным. Для этого в теле цикла изменяют используемые в условии переменные.

Цикл `while`, так же как и цикл `for`, состоит из заголовка и тела, но есть и другие элементы:
- Ключевое слово `while`;
- Условие, по которому цикл выбирает продолжить итерации или остановиться; в примере выше проверяется условие `n < 5`, и пока это условие истинное, цикл будет совершать итерации;
- Знак `:`. Без него будет синтаксическая ошибка, а сам цикл не будет объявлен.

Важно, что при этом тело цикла содержит действие, которое будет изменять переменную, используемую в условии.

### Задание 1
Вычислите, как тратится топливо на обогрев. Всего есть 15 единиц топлива. Задайте цикл `while` до тех пор, пока счётчик `i` меньше или равен `15`. В теле цикла сначала выводите значение, затем добавляйте к счётчику `i` 3.
```python
# напишите ваш код здесь
i = 0

while i <= 15:
    print(i)
    i += 3
```

# Библиотеки. Практика
### Шпаргалка
**Библиотека** — коллекция готовых функций и методов, которые помогают в решении специализированных задач.

Для загрузки библиотек часто используют систему `pip`. Как работать с `pip` — на примере библиотеки Seaborn.

- Установить библиотеку:
```
pip install seaborn  
```
- Посмотреть все установленные библиотеки:
```
pip list 
```
- Установить фиксированную версию библиотеки:
```
pip install seaborn==0.13.0   
```
- Обновить версию библиотеки до последней:
```
pip install --upgrade seaborn  
```

Импортировать библиотеку можно разными способами:
- `import название_библиотеки`. Пример:
```
import math 
```
- `import название_библиотеки as сокращённое_название`. Пример:
```
import math as m 
```
- Если заранее известно, какие функции из библиотеки будут нужны, и их количество невелико, можно оптимизировать память и время работы с помощью синтаксиса: `from название_библиотеки import название_функции`. Пример:
```
from math import sin, cos, pi 
```

Получить версию импортированной библиотеки:
```
import math 

print(math.__version__) 
```

```
weather_forecast = [(17, False), (21, False), (-3, True), (10, True)] 
```
Напишите функцию `suggest_clothing`, которая принимает прогноз погоды в таком формате и выдаёт рекомендации по одежде с такими условиями:
- Если температура выше 15 градусов, предлагается футболка.
- Если температура ниже 0 градусов, предлагается тёплая куртка.
- В промежутке между 0 и 15 градусами предлагается лёгкая куртка.
- Если ожидаются осадки, то дополнительно рекомендуется взять зонт.
Функция проанализирует температуру и вероятность осадков и предложит оптимальный варианты одежды для каждого дня недели.
```python
def suggest_clothing(weather_forecast):
    suggestions = []
    for day, (temperature, precipitation) in enumerate(weather_forecast):
        outfit = "Футболка" if temperature > 15 else "Лёгкая куртка"
        if temperature < 0:
            outfit = "Тёплая куртка"
        if precipitation is True:
            outfit += " и зонт"
        suggestions.append((day + 1, outfit))
    return suggestions
```
