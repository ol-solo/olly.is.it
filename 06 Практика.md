### Задание 1.

Первым делом в исследовательском анализе данных нужно загрузить данные, вывести общую информацию о них и несколько строк датафрейма. Загрузите данные о клиентах банка `/datasets/data.csv` в переменную `df` и выведите на экран:

- основную информацию о данных с помощью метода `info()`;
- три случайные строки.

### Задание 2.

Теперь подсчитайте количество и долю явных дубликатов в данных. Используйте прекод для вывода информации на экран.

```python
# Импортируем библиотеку pandas и загрузим данные в переменную df:
import pandas as pd
df = pd.read_csv('/datasets/data.csv')
  
# Подсчитаем общее количество дубликатов:
total_dupl = df.duplicated().sum()
  
# Подсчитаем долю дубликатов:
share_dupl = df.duplicated().mean()
  
# Выведем результат:
print(f'Количество дубликатов в данных = {total_dupl}, что составляет {share_dupl:.2%} от всех данных.')
```

### Задание 4.

В прошлом задании вы нашли, что в столбце `education` содержатся неявные дубликаты - результат написания категорий в различном регистре. Чтобы избежать этой проблемы следует стандартизировать данные - привести их к единому виду. В нашем случае будет достаточно привести их к единому регистру, например, к нижнему.

Приведите данные в столбце `education` к нижнему регистру. После этого проверьте, как стандартизация данных сказалась на количестве полных дубликатов в данных.

```python
# Импортируем библиотеку pandas и загрузим данные в переменную df:
import pandas as pd
df = pd.read_csv('/datasets/data.csv')
  
# Изменим регистр значений в столбце education на нижний:
def standardize_column(column):
    column = column.str.lower()
    return column
  
df['education'] = standardize_column(df['education'])

# ИЛИ
# df['education'] = df['education'].str.lower()

# Подсчитаем общее количество дубликатов:
total_dupl = df.duplicated().sum()
  
# Подсчитаем долю дубликатов:
share_dupl = df.duplicated().mean()
  
# Выведем результат:
print(f'Количество дубликатов в данных: {total_dupl}, что составляет {share_dupl:.2%} от всех данных.')
```

### Задание 5.

В результате стандартизации данных в одном из столбцов количество явных дубликатов увеличилось. Трудно сказать, с чем могут быть связаны дубликаты. Это может быть ошибка при сборе данных: например, если данные собирались из разных баз, где фигурировали одни и те же клиенты, но в базах были приняты разные стили написания уровня образования. Или дубликаты — это клиенты с полностью одинаковыми данными.

Поскольку в данных нет номера клиента или другой персональной информации, то этот вопрос решить сложно. Однако количество таких дубликатов невысокое (менее 1%), поэтому их можно отфильтровать.

Отфильтруйте явные дубликаты в данных и сохраните результат в исходном датасете.

```python
# Импортируем библиотеку pandas и загрузим данные в переменную df:
import pandas as pd
df = pd.read_csv('/datasets/data.csv')
  
# Изменим регистр значений в столбце education на нижний:
df['education'] = df['education'].str.lower()
  
# Удалим явные дубликаты:
df = df.drop_duplicates()
  
# Подсчитаем общее количество дубликатов:
total_dupl = df.duplicated().sum()
  
# Выведем результат:
print(f'Количество дубликатов в данных: {total_dupl}.')
```
