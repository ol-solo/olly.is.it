### Задание 5.

В результате стандартизации данных в одном из столбцов количество явных дубликатов увеличилось. Трудно сказать, с чем могут быть связаны дубликаты. Это может быть ошибка при сборе данных: например, если данные собирались из разных баз, где фигурировали одни и те же клиенты, но в базах были приняты разные стили написания уровня образования. Или дубликаты — это клиенты с полностью одинаковыми данными.

Поскольку в данных нет номера клиента или другой персональной информации, то этот вопрос решить сложно. Однако количество таких дубликатов невысокое (менее 1%), поэтому их можно отфильтровать.

Отфильтруйте явные дубликаты в данных и сохраните результат в исходном датасете.

```python
# Импортируем библиотеку pandas и загрузим данные в переменную df:
import pandas as pd
df = pd.read_csv('/datasets/data.csv')
  
# Изменим регистр значений в столбце education на нижний:
df['education'] = df['education'].str.lower()
  
# Удалим явные дубликаты:
df = df.drop_duplicates()
  
# Подсчитаем общее количество дубликатов:
total_dupl = df.duplicated().sum()
  
# Выведем результат:
print(f'Количество дубликатов в данных: {total_dupl}.')
```

В датасете с информацией о продаже квартир не так много категориальных данных. Одно из них — название населённого пункта, к которому квартира относится. Проверьте возможные аномалии в этом столбце. Для этого:

import pandas as pd

Создайте новый столбец `room_cat`, который будет содержать название категории с учётом количества комнат для обычных квартир и площади для квартир-студий. После этого оцените распределение объявлений по этим категориям с помощью метода `.value_counts()`.

```python
# Импортируем библиотеку pandas и загрузим данные в переменную df:
import pandas as pd
df = pd.read_csv('https://code.s3.yandex.net/datasets/real_estate_data_n_missing.csv')
  
def room_cat(row):
    """
    Функция возвращает категорию квартиры по количеству комнат, или по
    общей площади для квартир-студий:
    - однокомнатная — с одной жилой комнатой, кухней и санузлом;
    - двухкомнатная — с двумя жилыми комнатами, кухней и санузлом;
    - трёхкомнатная — с тремя жилыми комнатами, кухней и санузлом;
    - многокомнатная — с четырьмя и более жилыми комнатами, кухней и санузлом;
    - мини-студия — это небольшие квартиры-студии, с площадью менее 25 кв. м.;
    - квартира-студия – остальные студии с площадью 25 и более кв. м.
    """
  
    # Создадим переменную, которая будет содержать информацию о том,
    # является ли квартира студией или нет:
    is_studio = row['studio']
  
    # Создадим переменную, в которой сохраним количество комнат:
    rooms = row['rooms']
  
    # Создадим переменную, в которой сохраним общую площадь:
    total_area = row['total_area']
  
    # Зададим условную конструкцию, которая будет разделять квартиры от студий:
    # В начале пропишем условие, если квартира не студия:
    if is_studio == False:
  
        # Категоризация по количеству комнат:
        if rooms <= 0:
            return 'без категории'
        elif rooms == 1:
            return 'однокомнатная'
        elif rooms == 2:
            return 'двухкомнатная'
        elif rooms == 3:
            return 'трёхкомнатная'
        elif rooms >= 4:
            return 'многокомнатная'
  
    # Пропишем условия для квартир-студий:
    elif is_studio == True:
  
        # Категоризация по общей площади:
        if total_area <= 0:
            return 'без категории'
        elif total_area < 25:
            return 'мини-студия'
        elif total_area >= 25:
            return 'квартира-студия'
  
# Применим функцию для категоризации данных:
df['room_cat'] = df.apply(room_cat, axis=1)
  
# Выведем распределение значений по новой категории:
print(df['room_cat'].value_counts(normalize=True))
```
