# Снова о данных, пропусках и дубликатах. Практика

В этом и в следующих уроках вы попрактикуете действия, которые аналитик совершает при проведении исследовательского анализа данных. Практиковаться вы будете на разных данных. В этом уроке - на данных по информации о клиентах кредитного отдела банка - датасет `/datasets/data.csv`. Датасет содержит следующие столбцы:

- `children`
    - количество детей
- `days_employed`
    - трудовой стаж в днях
- `dob_years`
    - возраст клиента
- `education`
    - ступень образования
- `education_id`
    - ступень образования, id
- `family_status`
    - семейное положение
- `family_status_id`
    - семейное положение, id
- `gender`
    - пол
- `income_type`
    - тип занятости
- `debt`
    - факт невозврата кредита в срок
- `total_income`
    - общий годовой доход
- `purpose`
    - цель кредита

В этом уроке вы начнете исследовательский анализ данных и проведете знакомство с данными и изучите дубликаты в них.

### Задание 1.

Первым делом в исследовательском анализе данных нужно загрузить данные, вывести общую информацию о них и несколько строк датафрейма. Загрузите данные о клиентах банка `/datasets/data.csv` в переменную `df` и выведите на экран:

- основную информацию о данных с помощью метода `info()`;
- три случайные строки.
