## Что такое функция
**Функция** — маленькая программа, созданная для решения некоторой задачи. Её пишут один раз и затем при необходимости применяют одной командой.

**Основные преимущества применения функций:**

- **Повышается скорость работы.** Аналитик применяет свои наработки для решения похожих задач, а это быстрее, чем писать код заново.
- **Удобнее поддерживать код.** С функциями код становится структурированным, детали выносятся из основного алгоритма. Код проще анализировать, дорабатывать и переиспользовать.
- **Код проще отлаживать.** Опять же потому, что код более чистый, и всё логично разделено на блоки.

Подведём итоги:

- В коде приходится сталкиваться с повтором написанного кода. Повторяющиеся блоки кода стоит выносить в функции. Функции упрощают работу и делают программу понятнее.
- Функция — маленькая программа, созданная для решения задачи. Её создают один раз и применяют при необходимости.
- Оформление части кода в функцию — первый шаг к автоматизации процесса работы.

# Встроенные функции
В Python много **встроенных** функций — таких, которые есть в языке по умолчанию. Чтобы их использовать, нужно просто указать их название, дополнительно ничего писать не нужно.
Часто используют функцию `len()` — в неё передаётся аргумент: строка, список, таблица или др. — а на выходе функции получается длина этого аргумента, то есть количество символов в строке, элементов в массиве, строк в таблице.
В работе с численными данными полезны, например, функции:

- `round()` — округляет число до заданного значения. В неё передают через запятую исходное число и количество цифр после запятой после округления.
- `int()` — возвращает целую часть числа.
Встроенных функций много, их полный список можно найти [в документации](https://docs.python.org/3/library/functions.html).

## Методы
Некоторые функции работают с разными объектами. Под объектами в Python понимают строки, массивы, числа, списки, словари и другие инструменты.

Другие же функции привязаны к конкретному объекту и выполняют специфические операции. Например, бывает нужно преобразовать строку к нижнему регистру. Такая операция актуальна только для одного объекта — строки.

**Метод** — это функция, которая привязана к определённому объекту и выполняет операции, нужные в работе именно с этим объектом.

Метод вызывается в формате `объект.метод()`, то есть метод указывается через точку после имени объекта.

Примеры часто встречаемых операций в работе со строками:

- Разделить строку. Используем метод `.split()`, по умолчанию он делит строку по пробелам, но можно указать разделитель самостоятельно:

```
print("Есть три вида лжи: ложь, наглая ложь и статистика".split())
print("кто, где, когда".split(", ")) 
```

```
['Есть', 'три', 'вида', 'лжи:', 'ложь,', 'наглая', 'ложь', 'и', 'статистика']
['кто', 'где', 'когда'] 
```

Обратите внимание, если во втором случае мы бы указали разделителем запятую без пробела, то пробелы стали бы частью строк в результирующем списке.

- Проверить, состоит ли строка только из букв. Здесь помогает метод `.isalpha()`:

```
example_1 = 'ЭтаСтрокаСостоитТолькоИзБукв'
example_2 = 'а в этой уже будут не только буквы, но пробелы, спец.символы и даже цифры: 123654789'
# проверим, состоят ли строки только из букв, используя метод .isalpha()
print(example_1.isalpha())
print(example_2.isalpha()) 
```

```
> True    # в первой строке действительно содержатся исключительно буквы
> False   # во второй строке есть не только буквы 
```

- Преобразовать строку к верхнему регистру — помогает метод `.upper()`:

```
# приведём все символы строки к верхнему регистру, используя метод .upper()
print('я программирую на python, могу обойтись без caps lock'.upper()) 
```

```
Я ПРОГРАММИРУЮ НА PYTHON, МОГУ ОБОЙТИСЬ БЕЗ CAPS LOCK
# все символы строки перешли в верхний регистр -
# caps lock действительно не нужен! 
```

- Преобразовать строку к нижнему регистру — используют метод `.lower()`:

```
# приведём все символы строки к нижнему регистру, используя метод .lower()
print('КАЖЕТСЯ, Я ОПЯТЬ ЗАБЫЛ ВЫКЛЮЧИТЬ CAPS LOCK ПЕРЕД ТЕМ, КАК НАБРАТЬ ТЕКСТ'.lower()) 
```

```
кажется, я опять забыл выключить caps lock перед тем, как набрать текст
# все символы строки перешли в нижний регистр - отлично! 
```
```
mylist = list('террариум')
print(mylist) 
```

```
['т', 'е', 'р', 'р', 'а', 'р', 'и', 'у', 'м'] 
```

```
mylist.reverse()
print(mylist) 
```

```
['м', 'у', 'и', 'р', 'а', 'р', 'р', 'е', 'т'] 
```

Иногда такая операция пригодилась бы и для строки, но этот метод работает только со списком. Если попытаться применить его к строке, то выйдет ошибка с сообщением о том, что для строки этот метод не доступен:

```
mystr = 'террариум'
print(mystr.reverse()) 
```

```
AttributeError: 'str' object has no attribute 'reverse' 
```

`AttributeError` — это ошибка атрибута, то есть в метод передан не тот параметр.

Обращать строку придётся другими способами.

Подведём итоги.

- Решить наиболее частые задачи в Python помогают встроенные функции. Чтобы их использовать, достаточно просто вызвать функцию по названию.
- Метод — это функция, которая привязана к определённому объекту и выполняет операции, нужные в работе именно с этим объектом. Например, работает только со строками или только со списками.
- Методы закреплены за своими объектами, и метод для работы с одним объектом не подойдёт для работы с другими.

Встроенные функции и методы решают не все возможные задачи. Когда нет подходящей готовой функции, аналитики создают собственные функции для решения конкретных задач. О пользовательских функциях мы поговорим в следующем уроке.
