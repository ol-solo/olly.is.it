# Как компьютер нас понимает?
Чтобы сообщить компьютеру, что ему нужно сделать, — необходимо передать команду (инструкцию).
Если мы объединим инструкции вместе, то это уже программа. Сам текст программы называют программным кодом. 
А если мы объединим несколько программ, которые помогают решать одну задачу, то это уже будет модулем.
Компьютер понимает только машинный код. Чтобы перевести код  программы в машинный код, используют интерпретатор. Интерпретатор — это программа, которая последовательно читает по строчкам код и переводит каждую в машинный код с набором инструкций.

## Как компьютер хранит данные?
Компьютер хранит информацию в памяти. Память компьютера похожа на хранилище в банке — всё хранится по своим ячейкам.
В Python такая ячейка в памяти называется объектом, а номер ячейки называется ссылкой на объект. 
![[Pasted image 20260226123535.png]]
Использовать такие ссылки в явном виде неудобно: они длинные, ими тяжело пользоваться, а ещё они могут меняться в процессе выполнения программы. Было бы гораздо удобнее, если мы могли дать имя ячейке, которое не будет меняться. Именно для этого нужны **переменные**.

## Что такое переменная?
Переменная — это имя для объекта.
Когда вы первый раз указываете название переменной, это называется _объявить переменную_. В Python переменную всегда объявляют, присваивая ей какое-нибудь значение. Присвоить переменной значение — значит сообщить компьютеру, что в переменной содержится это значение.

## Как выбрать имя переменной
Для Python есть целый стандарт языка — [PEP-8](https://peps.python.org/pep-0008/). Это собрание правил хорошего стиля, которого стараются придерживаться все, кто пишут на Python.

## Вывод переменной на экран

```python
price = 399
product = 'pizza'

print(price)
```
`print()` - функция
То, что находится в круглых скобках `( )` - аргументы функции. Их может быть несколько.
`end` — аргумент, в котором можно задать текст, который будет добавлен в конце вывода строки.
`sep` — аргумент, в котором можно задать разделитель между значениями.

# Арифметические операции
Арифметические знаки в Python называют операторами, их принято отделять пробелами — так легче читать код.
- `+` — сложение,
- `-` — вычитание,
- `*` — умножение,
- `/` — деление;
- `//` — деление нацело;
- `%` — остаток от деления;
- `**` — возведение в степень.

## Порядок действий
Порядок выполнения арифметических операций в Python соответствует привычному «школьному»:
1. Действия в скобках.
2. Умножение, деление.
3. Сложение, вычитание.

## Используем переменные
Результаты арифметических операций можно сохранять в переменных.
Сначала выполняется всё, что в правой части, а затем результат присваивается переменной.
