# Как проводить статистические тесты. Практика

## Шпаргалка

В этом уроке мы будем практиковать методы и функции Python для проведения одновыборочного t-теста:

- `scipy.stats.ttest_1samp` — функция для проведения одновыборочного t-теста с проверкой того, как среднее выборки отличается от предполагаемого значения генеральной совокупности и насколько это статистически значимо;
    - `popmean` — предполагаемое среднее значение генеральной совокупности, с которым сравнивается выборочное среднее;
- `print()` — функция для вывода результатов на экран.
- Статистические понятия:
    - `t-статистика` — показывает, насколько далеко среднее выборочных данных находится от предполагаемого среднего в единицах стандартного отклонения;
    - `p-value` — вероятность получить такие или более крайние значения t-статистики, если нулевая гипотеза верна. Малые значения p-value (например, меньше 0.05) указывают на то, что различия статистически значимы.


## Задание 1.

Представим, что вы продуктовый аналитик в команде, которая недавно запустила рекламную кампанию для увеличения пользовательской активности на новом лендинге.

Вам нужно оценить, достигла ли кампания своей цели по повышению вовлечённости пользователей. Один из ключевых показателей успеха — увеличение среднего времени, которое пользователи проводят на сайте.

Среднее время, которое люди проводили на сайте раньше, равно пяти минутам. Определите, увеличилось ли время, проведённое пользователями на лендинге, после запуска маркетинговой кампании.

```python
from scipy.stats import ttest_1samp
  
time_spent = [6, 7, 5, 7, 6, 8, 5, 6, 7, 5, 4, 5, 6, 4, 5]
popmean = 5
  
t_statistic, p_value = ttest_1samp(time_spent, popmean)
print(f"t-статистика: {t_statistic:.2f}, p-value: {p_value:.4f}")
```

## Задание 2.

Мы получили значение p-value. Теперь определите, стоит ли отвергать нулевую гипотезу на основе результатов t-теста из Задания 1. Для этого используйте стандартный уровень значимости (0.05) для большинства подобных исследований (`alpha`).

```python
from scipy.stats import ttest_1samp
  
time_spent = [6, 7, 5, 7, 6, 8, 5, 6, 7, 5, 4, 5, 6, 4, 5]
popmean = 5
  
t_statistic, p_value = ttest_1samp(time_spent, popmean)
  
alpha = 0.05
if p_value < alpha:
    print("Отвергаем нулевую гипотезу: есть значимые статистические различия.")
else:
    print("Не отвергаем нулевую гипотезу: статистически значимых различий нет.")
```

# Сравнение средних. Практика

`scipy.stats.ttest_ind(a, b, equal_var=False)` — функция для проведения t-теста для двух независимых выборок, где `a` и `b` — списки данных для каждой из групп. Параметр `equal_var` указывает, равны ли дисперсии данных выборок (по умолчанию предполагается, что дисперсии равны).

`sum(list)` / `len(list)` — один из способов расчёта среднего значения списка в Python.

## Задание 1.

Розничная торговая сеть провела масштабную рекламную кампанию для увеличения продаж в двух ключевых магазинах. Используя данные о продажах до (`sales_before`) и после (`sales_after`) кампании, нужно анализировать изменение средних ежемесячных продаж. Исходя из уровня значимости `0.05` определите, привела ли кампания к значимому увеличению продаж, с использованием t-теста.

```python
from scipy.stats import ttest_ind
  
sales_before = [200, 210, 190, 205, 195, 220, 230, 215, 185, 195, 210, 220]
sales_after = [220, 230, 215, 225, 210, 235, 245, 225, 215, 205, 220, 230]
  
t_stat, p_value = ttest_ind(sales_before, sales_after, equal_var=False)
print(f"t-statistic: {t_stat}, p-value: {p_value}")
```

## Задание 2.

После анализа результатов рекламной кампании нужно оценить качество обслуживания клиентов в тех же двух магазинах. Данные по времени обслуживания собраны для трёх различных временных сегментов: утро (`service_times_morning_before` и `service_times_morning_after`), день (`service_times_afternoon_before` и `service_times_afternoon_after`) и вечер (`service_times_evening_before` и `service_times_evening_after`). Необходимо оценить, были ли улучшения в обслуживании клиентов в определённые части дня после кампании, используя отдельные t-тесты для каждого временного сегмента.

```python
from scipy.stats import ttest_ind
  
service_times_morning_before = [15, 16, 15, 14, 16, 15, 16, 15, 14, 15]
service_times_morning_after = [13, 14, 13, 12, 13, 14, 12, 13, 14, 12]
service_times_afternoon_before = [16, 15, 14, 16, 17, 16, 15, 16, 17, 15]
service_times_afternoon_after = [14, 13, 12, 14, 13, 14, 13, 12, 14, 13]
service_times_evening_before = [17, 15, 17, 16, 15, 16, 17, 15, 16, 17]
service_times_evening_after = [16, 15, 16, 15, 16, 15, 16, 15, 15, 16]
  
t_stat_morning, p_value_morning = ttest_ind(service_times_morning_before, service_times_morning_after, equal_var=False)
t_stat_afternoon, p_value_afternoon = ttest_ind(service_times_afternoon_before, service_times_afternoon_after, equal_var=False)
t_stat_evening, p_value_evening = ttest_ind(service_times_evening_before, service_times_evening_after, equal_var=False)
  
print(f"Morning - t-statistic: {t_stat_morning}, p-value: {p_value_morning}")
print(f"Afternoon - t-statistic: {t_stat_afternoon}, p-value: {p_value_afternoon}")
print(f"Evening - t-statistic: {t_stat_evening}, p-value: {p_value_evening}")
```

## Задание 3.

После анализа продаж и качества обслуживания розничная сеть также заинтересовалась анализом влияния рекламной кампании на продажи разных категорий товаров. Компания выделила две категории — бытовая техника и товары для дома — и хочет узнать, увеличились ли продажи в них после рекламной кампании по сравнению с продажами до неё.

Используя данные о продажах до (sales_appliances_before, sales_home_goods_before) и после (sales_appliances_after, sales_home_goods_after) кампании, определите с помощью t-теста для независимых выборок, привела ли кампания к значимому увеличению продаж в каждой из категорий отдельно.

```python
import pandas as pd
from scipy.stats import ttest_ind
  
sales_df = pd.DataFrame({
    'sales': [110, 120, 115, 118, 105, 125, 130, 119, 117, 112, 123, 115, 120, 116, 118,
              100, 105, 98, 103, 102, 107, 95, 110, 104, 97, 99, 105, 101, 96, 108, 165,
              170, 175, 180, 160, 165, 172, 169, 167, 175, 168, 163, 171, 178, 166, 150,
              145, 160, 155, 150, 148, 152, 153, 147, 160, 149, 154, 145, 158, 151],
    'time': ['after'] * 15 + ['before'] * 15 + ['after'] * 15 + ['before'] * 15,
    'category': ['appliances'] * 30 + ['home_goods'] * 30
})
  
sales_appliances_before = sales_df.query('category == "appliances" and time == "before"')['sales']
sales_appliances_after = sales_df.query('category == "appliances" and time == "after"')['sales']
sales_home_goods_before = sales_df.query('category == "home_goods" and time == "before"')['sales']
sales_home_goods_after = sales_df.query('category == "home_goods" and time == "after"')['sales']
  
t_stat_appliances, p_value_appliances = ttest_ind(sales_appliances_before, sales_appliances_after, equal_var=True)
print(f"Appliances - t-statistic: {t_stat_appliances}, p-value: {p_value_appliances}")
t_stat_home_goods, p_value_home_goods = ttest_ind(sales_home_goods_before, sales_home_goods_after, equal_var=True)
print(f"Home Goods - t-statistic: {t_stat_home_goods}, p-value: {p_value_home_goods}")
```

# Асимметричные проверки. Практика

Для проведения односторонних t-тестов в Python методам `ttest_1samp` и `ttest_ind` нужно передать аргумент `alternative`.

Для одной выборки — то есть метода `ttest_1samp` — `alternative` задаёт направление сравнения истинного среднего (μμ) совокупности, из которой взята выборка, с заранее выбранным числом aa.

`alternative='less'` задаёт альтернативную гипотезу H1:μ<aH1​:μ<a.
