# Строки
## Длина строки
Длина строки — это количество символов в ней. Для подсчёта длины используют функцию `len()`. В качестве аргумента `len()` принимает строку, а возвращает её длину — целое число.
```python
some_string = 'В этой строке 25 символов'

print(len(some_string)) 

25 
```

Строка может быть и пустой. Длина пустой строки равна нулю.

## Индексация строк
Символы в строке идут друг за другом, при этом у каждого — свой порядковый номер или индекс. **Отсчёт начинается с нуля**, а не с единицы.
![[Pasted image 20260227132643.png]]

Обратиться к символу по его индексу:
```python
this_is_string = 'Это строка'

print(this_is_string[0])
print(this_is_string[4]) 

Э
с 
```

Для получения символа можно использовать и **обратный индекс**. В таком случае мы указываем его отрицательным.
![[Pasted image 20260227132804.png]]
```python
this_is_string = 'Это строка'

print(this_is_string[-1]) # -1 это первый символ с конца, то есть самый последний в строке
print(this_is_string[-2]) # -2 это второй символ с конца или предпоследний в строке 

а
к 
```

Если нужно обратиться к **последнему символу строки**, то используют либо длину строки `len() - 1`, либо обратный индекс `-1`.
```python
this_is_string = 'Это строка'

print(this_is_string[-1])
print(this_is_string[len(this_is_string) - 1]) 

a
a 
```

> [!important]
> Если обратиться к индексу, который **больше** длины строки, произойдёт ошибка.
Другая ошибка возникнет, если попытаться **изменить** символ строки, обратившись к нему по индексу, так как строки — *неизменяемый тип данных*. Можно только сохранить новую строку вместо старой. Но изменить существующую строку не получится.

## Срезы строк
Срезы — почти как индексы. Только они позволяют получить не один, а сразу несколько идущих подряд символов строки.
![[Pasted image 20260227133148.png]]

Чтобы получить срез, используют конструкцию: `[start:end:step]`:
 - `start` — индекс элемента, с которого начинается срез. В нашем примере `start = 0`.
 - `end` — индекс элемента, до которого продолжается срез. Важно, что элемент с этим индексом не включается в сам срез. В нашем примере последний элемент среза под индексом 2, а значит, `end = 3`.
 - `step` — шаг, с которым выбираются символы. По умолчанию `step=1`, и его можно не использовать.
```python
password = 'qwerty'
print(password[0:3]) 

qwe 
```
> [!important]
> Обратите внимание, вывелось только три символа. Четвёртый по порядку и третий по индексу символ r не вывелся. Последний символ диапазона не входит в результат.

Если делать срез с самого начала строки, то необязательно указывать нулевой индекс, можно просто пропустить эту часть:
```python
# Если начало среза не указано, то срез начинается с нулевого символа.
print(city[:3]) # то же, что и city[0:3] 

Par 
```

Если конец подстроки совпадает с концом исходной строки, можно пропустить индекс конца среза:
```python
# Если конец среза не указан, то срез продолжается до конца строки.
print(city[2:]) # тоже, что и city[2:5] 

ris 
```

Если не указывать ни конец, ни начало строки, то вернётся её точная копия:
```python
print(city[:]) # то же, что и city[0:5] 

Paris  
```

В срезах можно использовать обратные индексы.
```python
print(city[-3:]) # тоже, что и city[2:5] 

ris  
```

Используя `step`, мы можем указать шаг, с которым получаем символы. 
```python
city = 'Paris'
substring = city[::2] # Из всей строки выбираем каждый нечётный элемент
print(substring) 

Prs 
```

Если указать `step = -1`, то вернётся строка в обратном порядке:
```python
city = 'Paris'
reverse_string = city[::-1]
print(reverse_string) 

siraP 
```

## Как работать со строками
В Python есть набор встроенных **методов**, которые предназначены для работы со строками. Методы — это функции, как, например, `print()` или `len()`, но которые могут работать только с определённым типом данных (в нашем случае — это строки `str`). Методы вызываются через точку после самой строки:
```python
word = 'привет'

# метод upper сделает все буквы прописными
new_word = word.upper()
print(new_word) 

ПРИВЕТ 
```

Несколько полезных методов:
- `replace()` — заменит часть строки;
- `find()` — ищет подстроку в строке;
- `upper()` — заменяет все буквы в строке на прописные;
- `lower()` — заменяет все буквы на строчные.

## Замена части строки
Метод `replace()` принимает два аргумента:
1. подстроку, которую нужно заменить;
2. подстроку, которая заменит исходную.
```python
some_string = 'leaving'
print(some_string)

# заменяем подстроку leav строкой stay
some_string = some_string.replace('leav', 'stay')
print(some_string) 

leaving
staying 
```

## Поиск подстроки
Метод `find()` ищет подстроку в строке. Если найдёт — вернёт индекс символа, начиная с которого подстрока встречается в исходной строке.
Чаще всего методу передают один аргумент — подстроку, которую он должен найти:
```python
word = 'hey, hello, hello, hello, how low?'
# подстрока hello входит в строку несколько раз,
# но метод вернёт место начала первого вхождения
ind = word.find('hello')

print(ind) 

5 
```

Для `find()` можно задать диапазон индексов, в котором он должен искать. Инструкцию в этом коде можно прочитать так: «Найди первое вхождение `hello` в строке из переменной `word`, начиная с пятнадцатого символа и заканчивая двадцать четвёртым».
```python
word = 'hey, hello, hello, hello, how low?'

# Три аргумента: подстрока, индекс начала поиска, индекса завершения, увеличенный на 1
ind = word.find('hello', 15, 25)

print(ind) 

19 
```

Если `find()` ничего не найдёт, он вернёт `-1`:
```
hi = 'Привет!'
bye_ind = hi.find('Пока!')

print(bye_ind)

-1 
```

У метода `find()` есть аналог — `rfind()`, тут поиск подстроки начинается с правой стороны. Если `find()` ищет первое вхождение подстроки, то `rfind()` ищет последнее вхождение.
```
word = 'hey, hello, hello, hello, how low?'
# подстрока hello входит в строку несколько раз,
# но метод вернёт номер последнего вхождения
ind = word.rfind('hello')

print(ind) 

19 
```

## Изменение регистра
Методы `upper()` и `lower()` возвращают строки, в которых исходная строка приведена к верхнему и нижнему регистру соответственно:
```python
word = 'AbRacaDABRA'
lower_word = word.lower()
upper_word = word.upper()
print(lower_word, upper_word)

abracadabra ABRACADABRA
 
```

## Экранирование специальных символов
Чтобы Python воспринимал кавычки как часть строки, нужно сделать экранирование кавычек. При экранировании Python считывает специальный символ и подставляет вместо него то, что считает нужным. Символы для экранирования начинаются со слэша `\`.
```
phrase = 'I don\'t want to stop'
print(phrase)
print(len(phrase)) 

I don't want to stop
20 
```

Есть и другие специальные символы, которые используют для экранирования:
- `\n` — перенос строки;
- `\t` — табуляция;
- `\\` — слэш.

Такие сочетания считаются за один символ.

Метод `.strip` удаляет лишние пробелы в начале и в конце строки.
```
print('       Иван Иванович Иванов       '.strip()) 

Иван Иванович Иванов 
```

> [!important]
> Пробелы в середине строки остались нетронутыми.

## Форматирование строк
Форматирование — это способ создания строки, в которой значения переменных подставляются в определённых местах, как по шаблону.

1. Метод `format()`. В строке в фигурных скобках `{ }` указывают название переменной для подстановки. Аргументами метода `format` будут эти самые переменные.
```python
cr_percent = 3
dau = 1000
dau_delta = 5

s = """
По итогам квартала конверсия составила {cr_percent}%.
В тоже время показатель DAU составил {dau} пользователей, 
что на {dau_delta}% выше прогноза.
"""
print(s.format(cr_percent = cr_percent, dau = dau, dau_delta = dau_delta)) 

По итогам квартала конверсия составила 3%.
В тоже время показатель DAU составил 1000 пользователей, 
что на 5% выше прогноза. 
```

2. `f-строки`. Мы пишем символ `f` перед строкой и точно так же используем фигурные скобки `{ }` и именованные переменные, но использовать метод `format` уже необязательно.
```
cr_percent = 3
dau = 1000
dau_delta = 5

s = f"""
По итогам квартала конверсия составила {cr_percent}%.
В тоже время показатель DAU составил {dau} пользователей, 
что на {dau_delta}% выше прогноза.
"""
print(s) 

По итогам квартала конверсия составила 3%.
В тоже время показатель DAU составил 1000 пользователей, 
что на 5% выше прогноза. 
```

# Списки
## Что такое список?
Список — это упорядоченный набор элементов.

## Создание списка
Для создания списка используют квадратные скобки `[ ]`, а элементы списка перечисляют через запятую.
```python
years = [1994, 1972, 2008] #Список с годами
print(years) 

[1994, 1972, 2008] 
```

Список может содержать элементы разных типов данных:
```python
# список из строки str, целого int и вещественного числа float
items = ['какая-то строка', 1234, 3.1415] 
```

Список может содержать другие списки, тогда его называют **вложенным**:
```python
items_list = [[1, 2, 3], [1, 2, 3, 4], [5, 6]] 
```

Список — это отдельный тип объекта `list`:
```python
years = [1994, 1972, 2008] # Список с годами
print(type(years)) # Выведем тип объекта 

class <list> 
```

Каждый элемент имеет свой порядковый номер или индекс. Как и в строках, нумерация начинается с `0`. Используя индекс, можно получить элемент из списка, обычно это называют — «обратиться к элементу списка».
![[Pasted image 20260302161816.png]]

Количество элементов в списке называют его длиной. Как и в случае строк, длину списка вернёт функция `len()`:
```python
first_film = ['Побег из Шоушенка', 'США', 1994, 'драма', 142, 9.111]
print(len(first_film)) 

6 
```

Каждый элемент списка связан со своим индексом — порядковым номером. По индексу получают отдельные значения и их группы — срезы. Всё как со строками.
```python
first_film = ['Побег из Шоушенка', 'США', 1994, 'драма', 142, 9.111]
print(first_film[0], first_film[1], first_film[-1]) 

Побег из Шоушенка США 9.111 
```

## Срезы
С помощью срезов обращаются к диапазону элементов с помощью конструкции `[start:end:step]`.
```python
first_film = ['Побег из Шоушенка', 'США', 1994, 'драма', 142, 9.111]
#срезаются элементы с первого по третий
print(first_film[1:4])
#срезаются элементы с нулевого по второй
print(first_film[:3])
#срезаются элементы со второго по последний
print(first_film[2:]) 

['США', 1994, 'драма']
['Побег из Шоушенка', 'США', 1994]
[1994, 'драма', 142, 9.111] 
```

## Операции над списками
В отличие от строк, списки можно изменять. Например, мы можем изменить элемент в списке, обращаясь к нему по индексу:
```python
numbers = [1, 2, 3, 4, 5] # Создадим список
print(numbers) # Выведем содержимое списка

numbers[0] = 0 # Присвоим элементу с индексом 0 новое значение
print(numbers) # Снова выведем содержимое списка 

[1, 2, 3, 4, 5]
[0, 2, 3, 4, 5] 
```

|**Строки**|**Списки**|
|---|---|
|Нельзя изменить символ в строке, обращаясь по индексу.|Можно изменить элемент в списке, обращаясь по индексу.|
|Нельзя добавить к имеющейся строке новый символ. Операция сложения + создаст новую строку.|Можно добавить новый элемент к списку.|

## Добавление элементов в список
Самый простой способ добавить элемент в список — вызвать метод `append()`. На вход метод принимает элемент, который нужно добавить. Метод добавляет элемент в конец списка.
```python
first_film = ['Побег из Шоушенка', 'США', 1994, 'драма', 142, 9.111]
first_film.append('Фрэнк Дарабонт')

print(first_film) 

['Побег из Шоушенка', 'США', 1994, 'драма', 142, 9.111, 'Фрэнк Дарабонт'] 
```

> [!important]
> Обратите внимание, что метод `append()` **изменяет** изначальный список, а не создаёт новый. То есть результатом выполнения инструкции `first_film.append('Фрэнк Дарабонт')` уже будет добавление элемента списка. Таким образом, нам не нужно ничего переприсваивать переменной `first_film`. После выполнения этой инструкции список уже будет изменённым.

## Добавление нескольких элементов в список
Иногда нам нужно расширить список другим списком — прибавить к одному списку другой.
Один из способов расширить список — вызвать метод `extend()`. На вход он принимает другой список, которым нужно расширить изначальный.
```python
second_film = ['Крёстный отец', 'США', 1972] # Создаём список с описанием фильма
directors = ['Фрэнсис Форд Коппола', 'Нино Рота'] # Создаём список с режиссёрами фильма
# Расширим список second_film элементами из списка directors
second_film.extend(directors)

print(second_film)

['Крёстный отец', 'США', 1972, 'Фрэнсис Форд Коппола', 'Нино Рота'] 
```

Но есть и другой способ расширить список другим списком, более привычный — использовать оператор сложения `+`:
```python
second_film = ['Крёстный отец', 'США', 1972] # Фильм
directors = ['Фрэнсис Форд Коппола', 'Нино Рота'] # Режиссёр и композитор фильма
# в конец списка добавим имя режиссёра и имя композитора

print(second_film + directors) 

['Крёстный отец', 'США', 1972, 'Фрэнсис Форд Коппола', 'Нино Рота'] 
```

## Удаление элемента из списка
метод `pop()` принимает индекс элемента, который нужно удалить, и возвращает сам элемент, который был удалён.
```python
numbers = [1, 2, 3, 4, 5]
droped_element = numbers.pop(0) # Удалим первый элемент с индексом 0

# В переменной droped_element хранится элемент, который был удалён
print(droped_element) 
print(numbers) 

1
[2, 3, 4, 5] 
```

## Сортировка списка
Если список состоит из чисел, то их можно отсортировать в порядке возрастания или убывания. Для этого можно вызвать метод `sort()`:
```python
numbers = [3, 2, 5, 4, 1]
numbers.sort()

print(numbers) 

[1, 2, 3, 4, 5] 
```

По умолчанию список будет отсортирован в порядке возрастания — от меньшего к большему. Чтобы отсортировать наоборот в порядке убывания, используют параметр `reverse=True`:
```python
numbers = [3, 2, 5, 4, 1]
numbers.sort(reverse=True)

print(numbers) 

[5, 4, 3, 2, 1] 
```

## Получение минимального и максимального элемента в списке
функциями `min()` и `max()` найти самый большой или самый маленький элемент в списке
```python
numbers = [3, 2, 5, 4, 1]

print(max(numbers)) #Выведем максимальный элемент в списке
print(min(numbers)) #Выведем минимальный элемент в списке 

5
1 
```

# Строки и списки
## Разделить строку на несколько частей
Используя функцию `list()`, можно преобразовать строку в список, где элементами списка станут символы строки:
```python
text = 'Привет!'
list_text = list(text)

print(list_text) 

['П', 'р', 'и', 'в', 'е', 'т', '!'] 
```

Однако чаще встречается немного другая задача — разделить строку не на отдельные символы, а на отдельные подстроки, то есть разделить строку на несколько частей по какому-нибудь разделителю. Например, разбить абзац на отдельные предложения. Для этого используем метод `split()`. На вход метод принимает символ-разделитель — это символ, которым мы «дробим» строку. Метод возвращает список из разделённых строк.
```python
text = 'Предложение первое. Второе предложение. Предложение номер три.'
sent_text = text.split('.') #Разделим строку на отдельные части по точке

print(type(sent_text)) #Выведем тип полученного результата
print(sent_text) #Выведем содержимое списка 

<class 'list'>
['Предложение первое', ' Второе предложение', ' Предложение номер три'] 
```
Метод `split()` возвращает список строк, разъединённых указанным разделителем

## Объединить в строку
Мы можем сделать и наоборот — собрать целую строку из элементов списка. Для этого используем метод `join()`:
```python
sent_text = ['Предложение первое', 'Второе предложение', 'Предложение номер три']
sep = '.'
text = sep.join(sent_text)

print(text) 

Предложение первое.Второе предложение.Предложение номер три 
```

Метод `join()` объединяет элементы списка в одну строку, разделяя сами элементы строкой `sep`.
