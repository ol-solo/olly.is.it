Компилятор

Переменная — это имя для объекта. Когда вы первый раз указываете название переменной, это называется _объявить переменную_.

Свойство, при котором одни и те же операторы работают с разными типами объектов, но по-разному, называют **полиморфизмом**. Полиморфизм — один из принципов объектно-ориентированного подхода в программировании.

## Статическая типизация
Статическая типизация — свойство, при котором переменная может хранить значение только одного типа. При статической типизации, как правило, тип переменной задаётся вручную при её объявлении.

## Динамическая типизация
Динамическая типизация — свойство, при котором переменная может содержать значения разных типов. При объявлении переменной тип не указывается.

Область видимости определяет, в какой части программы переменные могут быть назначены, изменены и прочитаны. Существует две основные области:
- глобальная, в которой задаются переменные, доступные в любой части кода, и
- локальные — которые выполняются в строго определённой части кода, например, в цикле или функциях.
