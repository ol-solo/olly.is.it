### `int()`
превращает объект в целое число. отбрасывает дробную часть. Положительные числа округляются в меньшую сторону, а отрицательные — в большую сторону

### `float()`
превращает объект в вещественное число

### `str()`
превращает объект в строку

### `round()`
функцию математического округления округляет числа к ближайшему целому, в пограничных случаях (1.5, 2.5, 3.5 и т.п) работает в большую сторону

Можно указать точность округления (по умолчанию округляет до целого числа)
```python
round(temperature, 2)
```

### `min()`

### `max()`

### `list()`
преобразовать строку в список, где элементами списка станут символы строки

### `bool()`
возвращает значение `False`, если в исходных данных:
- число 0;
- что-то пустое: строка, список или пустой кортеж;
- значение None.

В остальных случаях функция вернёт `True`.

### `all()`
возвращает `True`, если все значения списка `True`. Например, функция поможет проверить, что заполнены все обязательные поля в форме регистрации. Она вернёт `True`, если всё заполнено, и `False`, если чего-то недостаёт.

### `any()`
возвращает `True`, если хотя бы одно значение списка `True`. Например, клиент может указать разные способы связи: номер телефона, имейл, ник в Телеграме. Эта функция поможет проверить, что указан хоть один способ связи. 

### `range()`
по мере обращения к ней возвращает последовательность целых чисел длиной `n` в диапазоне от `0` до `n-1`. Правая граница — значение `n` исключено из диапазона значений, возвращаемого функцией.

### `sum()`
суммирования значений

### `enumerate()`
возвращает индекс и значение всех элементов списка или другого итерируемого объекта.
дополнительный параметр `start`, который позволяет задать начало нумерации значений. По умолчанию этот параметр равен `0`.

### `zip()`
принимает в качестве аргумента список итерируемых объектов, элементы которых с одинаковым индексом будут объединены в кортеж. В результате будет создано множество кортежей, объединённых в `zip`-объект
можно объединять более двух списков.

### `next()`

### `iter()`
