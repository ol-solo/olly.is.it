# Статическая типизация
Свойство, при котором переменная может хранить значение только одного типа. При статической типизации, как правило, тип переменной задаётся вручную при её объявлении.

# Динамическая типизация 
Свойство, при котором переменная может содержать значения разных типов. При объявлении переменной тип не указывается.
