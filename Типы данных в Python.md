# Какие бывают типы данных
Тип объекта, или тип данных, определяет, в каком виде значения хранятся и записываются. Для записи строки используют кавычки, а число записывают без них. 
> [!hint]
> Узнать, какой тип Python использует для хранения объекта, поможет функция `type()`:
>```python
>number_int = 123 #Это целое число
>number_float = 1.2345 #Это число с плавающей точкой
>
>print(type(number_int))
>print(type(number_float)) 
>
><class 'int'>
><class 'float'> 
>```

|**Тип**|**Примеры**|**Как обозначается**|
|---|---|---|
|Вещественный|`-10.0`, `.1`, `1.23`|float|
|Целочисленный|`-1`, `12`, `123`|int|
|Логический|`True` / `False`|bool|
|Строка|`'Я строка'`|str|

Также тип данных определяет, какие операции можно проводить над значениями, а какие нельзя. Например, с помощью оператора `/` можно делить числа, но нельзя разделить одну строку на другую.

## Числа
- Целые числа хранят в типе `int`.
- Дробные числа — в типе `float`. Этот тип называют **вещественным**, или **числом с плавающей точкой**.

## Строки
Внутри строки могут быть любые символы: буквы, цифры и прочие символы. Строки всегда записывают в кавычках.
Если текст не умещается в одной строке кода, можно использовать тройные кавычки:
```python
pushkin_text = """Я помню чудное мгновенье:
Передо мной явилась ты,
Как мимолетное виденье,
Как гений чистой красоты.

В томленьях грусти безнадежной,
В тревогах шумной суеты,
Звучал мне долго голос нежный
И снились милые черты."""
```

Объединение ("сложение" и "умножение") нескольких строк называют **конкатенацией**:
```python
title = 'Название магазина'
shop_name = 'Моторизированный котокраб'

print(title + shop_name)

print("=" * 20) # Повторим символ 20 раз
```

> [!important]
> Невозможно, однако, сложить строку и число!

Свойство, при котором одни и те же операторы работают с разными типами объектов, но по-разному, называют **полиморфизмом**. Полиморфизм — один из принципов объектно-ориентированного подхода в программировании.

## Логический тип
Объекты логического типа (булевый, bool) хранят значения `True` или `False` и представляют результаты логических выражений. Для работы с таким типом данных используют специальные логические операторы. 

|**Оператор**|**Что означает**|**Как работает**|
|---|---|---|
|and|И|даёт истину, только если и справа и слева от него истина|
|or|ИЛИ|даёт в результате истину, если справа или слева от него есть истина|
|not|НЕ|меняет истину на ложь, и наоборот|

Логические значения часто представляют как числа: `True` — это 1, а `False` — 0. В Python логические значения можно явно преобразовать в целочисленные (`int`):
```python
print(int(False), int(True)) 

0 1 
```

В других операциях логические значения тоже ведут себя как числа.
```python
is_buyer = [True, True, False, True, False, False, False, False, True, True, False]
print(f'Всего посетителей: {len(is_buyer)}, из них совершили покупку {sum(is_buyer)}') 

Всего посетителей: 11, из них совершили покупку 5
```

В Python не нужно специально указывать, какой тип данных ему использовать — он сам справляется с этой задачей, когда создаёт переменную. Это свойство называют [[Динамическая типизация#Динамическая типизация|динамической типизацией]]. Благодаря этой особенности писать код на Python удобнее.

# Преобразование типов данных
## Основные функции для преобразования типов в Python
- `int()` — превращает объект в целое число;
- `float()` — превращает объект в вещественное число;
- `str()` — превращает объект в строку.

## Целые числа
Результатом операции с `int` и `float` будет `float`. Но иногда дробная часть равна нулю, чтобы убрать его, нужно преобразовать тип данных.
Когда переменная хранит вещественное число с ненулевой дробной частью, функция `int()` просто отбрасывает дробную часть. **Положительные** числа округляются в **меньшую** сторону, а *отрицательные* — в *большую* сторону.

## Дробные числа
Если нужно просто округлить дробное число, то как правило используют функцию математического округления `round()` — она округляет числа к ближайшему целому, в пограничных случаях (1.5, 2.5, 3.5 и т.п) работает в большую сторону.
![[Pasted image 20260226132152.png]]
Для функции `round()` можно указать точность, с которой округляется число. По умолчанию — до целого числа, но можно указать, чтобы округление было, например, до двух знаков после запятой:
```python
temperature = 36.6666 # Очень точная температура здорового человека

print(round(temperature), type(round(temperature))) # Округлим до ближайшего целого
print(round(temperature, 2), type(round(temperature, 2))) # Округлим до двух знаков после запятой 

37 <class 'int'>
36.67 <class 'float'> 
```

## Числа и строки
Если в результате операций ожидается число, то приведите все переменные к числу, используя `float()` или `int()`. Если ожидается строка, то приведите всё к строке, используя `str()`.
Любое число можно преобразовать к строке, но строку можно преобразовать к числу только, если она не содержит никаких лишних символов, кроме цифр.
Исключения:
- знак `+` или `-` перед цифрами если число положительное или отрицательное соответственно
- Если число дробное, то можно использовать символ точки `.` для разделения целой и дробной части.
- Пробелы по краям (справа или слева)

## Логические переменные
ЕФункция `bool()` возвращает значение `False`, если в исходных данных:
- число 0;
- что-то пустое: строка, список или пустой кортеж;
- значение None.

В остальных случаях функция `bool()` вернёт `True`.
![[Pasted image 20260302193824.png]]

**Функции-предикаты** возвращают булево значение (`True` или `False`) и помогают проверить условия или выполнить логические операции.

- `islower()` — возвращает `True`, если в строке нет прописных букв;
- `isdigit()` — возвращает `True`, если строка состоит только из числовых знаков;
- `isalpha()` — возвращает `True`, если в строке только буквы, без пробелов и знаков препинания;
- `isspace()` — возвращает `True`, если в строке только пробелы.

Если условие не соблюдено, эти функции вернут `False`.
![[Pasted image 20260302200119.png]]

Строковые методы, которые проверяют соответствие начала и окончания строки заданному набору символов:
- `startswith(prefix)` — возвращает `True`, если строка начинается с указанного набором символов `prefix`;
- `endswith(suffix)` — возвращает `True`, если строка заканчивается указанным набором символов `suffix`;
```python
print('hello'.startswith('h')) # проверяем, начинается ли hello с h
print('hello'.endswith('O')) # проверяем, заканчивается ли hello символом O 

True
False 
```
> [!important]
> Хотя `hello` заканчивается `o`, но в функции мы уточняем про другой символ — заглавную `O`. Эта функция чувствительна к регистру.

Функции `all()` и `any()` работают с пустыми значениями в списках и других множествах:
- `all()` — возвращает `True`, если все значения списка `True`. Например, функция поможет проверить, что заполнены все обязательные поля в форме регистрации. Она вернёт `True`, если всё заполнено, и `False`, если чего-то недостаёт.
- `any()` — возвращает `True`, если хотя бы одно значение списка `True`. Например, клиент может указать разные способы связи: номер телефона, имейл, ник в Телеграме. Эта функция поможет проверить, что указан хоть один способ связи. Она вернёт `True`, если клиент указал, как с ним можно связаться, и `False`, если клиент не указал ни одного способа.
```python
print(all(['A','B',None,'D'])) 
print(any(['A','B',None,'D'])) 

False
True 
```
