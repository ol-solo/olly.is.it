# Встроенные функции Python
### `print()`
```python
print(*objects, sep=" ", end="\n")
```
- **Основные аргументы**:
    - `*objects` — одно или несколько значений для вывода.
- **Частые именованные параметры**:
    - `sep` — строка‑разделитель между объектами (по умолчанию пробел).
    - `end` — строка, добавляемая в конце вывода (по умолчанию `"\n"`).

### `int()`
```python
int(x)
int(x, base=10)
```
- **Основные аргументы**:
    - `x` — число, строка или другой объект, который нужно привести к целому.
- **Частые параметры**:
    - `base` — система счисления для строкового аргумента (по умолчанию 10).
- **Поведение**:
    - при `x` типа `float` отбрасывает дробную часть (движение к нулю).

### `float()`
```python
float(x)
```
- **Основные аргументы**:
    - `x` — число или строка, которую нужно привести к вещественному числу.
- **Поведение**:
    - принимает строки вида `"3.14"`, `"1e-3"` и т.п.

### `str()`
```python
str(x)
```
- **Основные аргументы**:
    - `x` — объект, который нужно представить как строку.
- **Поведение**:
    - возвращает человеко‑читаемое строковое представление объекта.

### `round()`
```python
round(number)
round(number, ndigits)
```
- **Основные аргументы**:
    - `number` — число для округления.
- **Частые параметры**:
    - `ndigits` — количество знаков после запятой (по умолчанию до целого).
- **Поведение**:
    - округляет к ближайшему числу, в пограничных случаях — по правилам реализации (в учебных задачах воспринимается как «в большую сторону»).

### `len()`
```python
len(obj)
```
- **Основные аргументы**:
    - `obj` — объект с длиной (строка, список, кортеж, словарь и т.п.).
- **Поведение**:
    - возвращает количество элементов / символов.

### `type()`
```python
type(obj)
```
- **Основные аргументы**:
    - `obj` — объект, тип которого нужно узнать.
- **Поведение**:
    - возвращает объект‑тип (`int, list, str, ...`).

### `min()`
```python
min(iterable)
min(a, b, c, ...)
```
- **Основные аргументы**:
    - `iterable` — итерируемый объект, из которого ищем минимум;
    - или несколько позиционных аргументов.
- **Частые параметры**:
    - `key` — функция для вычисления ключа сравнения.
- **Поведение**:
    - возвращает минимальный элемент по обычному или заданному сравнению.

### `max()`
```python
max(iterable)
max(a, b, c, ...)
```
- **Основные аргументы**:
    - аналогично `min()`, но для максимума.
- **Частые параметры**:
    - `key` — функция ключа.
- **Поведение**:
    - возвращает максимальный элемент.

### `list()`
```python
list(iterable)
```
- **Основные аргументы**:
	- `iterable` — объект, по которому можно итерироваться.
- **Поведение**:
    - создаёт список из элементов `iterable`;
    - для строки — список символов строки.

### `bool()`
```python
bool(x)
```
- **Основные аргументы**:
    - `x` — значение для приведения к логическому типу.
- **Поведение**:
    - `False` для 0, пустых контейнеров, `None`;
    - `True` во всех остальных случаях.

### `all()`
```python
all(iterable)
```
- **Основные аргументы**:
    - `iterable` — последовательность значений, приводимых к `bool`.
- **Поведение**:
    - `True`, если все элементы «истинные»;
    - `True` для пустой последовательности.

### `any()`
```python
any(iterable)
```
- **Основные аргументы**:
    - `iterable` — последовательность значений.
- **Поведение**:
    - `True`, если хотя бы один элемент «истинный»;
    - `False` для пустой последовательности.

### `range()`
```python
range(stop)
range(start, stop, step=1)
```
- **Основные аргументы**:
    - `start` — начало диапазона (по умолчанию 0);
    - `stop` — конец (не включительно);
    - `step` — шаг (по умолчанию 1).
- **Поведение**:
    - создаёт объект диапазона целых чисел, часто используется в циклах.

### `sum()`
```python
sum(iterable, start=0)
```
- **Основные аргументы**:
    - `iterable` — итерируемый объект с числами;
- **Частые параметры**:
    - `start` — начальное значение суммы (по умолчанию 0).
- **Поведение**:
    - возвращает сумму элементов + `start`.

### `enumerate()`
```python
enumerate(iterable, start=0)
```
- **Основные аргументы**:
    - `iterable` — итерируемый объект;
- **Частые параметры**:
    - `start` — начальный индекс (по умолчанию 0).
- **Поведение**:
при обходе возвращает пары (`index`, `value`).

### `zip()`
```python
zip(iter1, iter2, *iterables)
```
- **Основные аргументы**:
    - два и более итерируемых объектов одинаковой (или разной) длины.
- **Поведение**:
    - на каждой итерации отдаёт кортеж из элементов с одинаковыми индексами;
    - длина результата равна длине самого короткого итерируемого.

### `next()`
```python
next(iterator)
next(iterator, default)
```
- **Основные аргументы**:
    - `iterator` — объект‑итератор.
- **Частые параметры**:
    - `default` — значение, которое вернуть, когда элементы закончились.
- **Поведение**:
    - возвращает следующий элемент;
    - без `default` при завершении итерации выбрасывает `StopIteration`.

### `iter()`
```python
iter(obj)
```
- **Основные аргументы**:
    - `obj` — итерируемый объект или объект с протоколом итератора.
- **Поведение**:
    - возвращает итератор, который можно обходить в `for` или через `next()`.

# Встроенные методы Python
## Методы строк (str)
> [!info]
> Методы вызываются через точку: `s.method(...)`, где `s` — строка.
### `.replace()`
```python
s.replace(old, new, count=-1)
```
- **Основные аргументы**:
    - `old` — подстрока, которую нужно заменить.
    - `new` — подстрока, на которую нужно заменить.
- **Частые параметры**:
    - `count` — максимальное количество замен (по умолчанию заменяет все вхождения).

### `.find()`
```python
s.find(sub, start=0, end=len(s))
```
- **Основные аргументы**:
    - `sub` — подстрока для поиска.
- **Частые параметры**:
    - `start` — индекс, с которого начинать поиск.
    - `end` — индекс, до которого выполнять поиск (не включительно).
- **Поведение**:
    - возвращает индекс первого вхождения `sub` или `-1`, если подстрока не найдена.

### `.rfind()`
```python
s.rfind(sub, start=0, end=len(s))
```
- **Основные аргументы**:
    - `sub` — подстрока для поиска.
- **Частые параметры**:
    - `start`, `end` — диапазон поиска, как в `.find()`.
- **Поведение**:
    - возвращает индекс последнего вхождения `sub` или `-1`, если не найдено.

### `.upper()`
```python
s.upper()
```
- **Поведение**:
    - возвращает копию строки, где все буквенные символы переведены в верхний регистр.

### `.lower()`
```python
s.lower()
```
- **Поведение**:
    - возвращает копию строки, где все буквенные символы переведены в нижний регистр.

### `.strip()`
```python
s.strip(chars=None)
```
- **Частые параметры**:
    - `chars` — строка символов, которые нужно удалить по краям (по умолчанию — пробельные символы).
- **Поведение**:
    - удаляет указанные символы в начале и в конце строки.

### `.split()`
```python
s.split(sep=None, maxsplit=-1)
```
- **Частые параметры**:
    - `sep` — разделитель (по умолчанию — любые пробельные символы).
    - `maxsplit` — максимальное число разбиений (по умолчанию без ограничения).
- **Поведение**:
    - возвращает список подстрок, на которые строка была разбита.

### `.join()`
```python
sep.join(iterable)
```
- **Основные аргументы**:
    - `iterable` — итерируемый объект из строк.
- **Поведение**:
    - объединяет строки из `iterable` в одну строку, вставляя между ними `sep`.

### `.islower()`
```python
s.islower()
```
- **Поведение**:
    - возвращает `True`, если есть хотя бы одна буква и все буквенные символы в нижнем регистре.

### `.isdigit()`
```python
s.isdigit()
```
- **Поведение**:
    - возвращает `True`, если все символы строки — цифровые и строка не пустая.

### `.isalpha()`
```python
s.isalpha()
```
- **Поведение**:
    - возвращает `True`, если все символы строки — буквы и строка не пустая.

### `.isspace()`
```python
s.isspace()
```
- **Поведение**:
    - возвращает `True`, если строка состоит только из пробельных символов и не пустая.

### `.startswith()`
```python
s.startswith(prefix, start=0, end=len(s))
```
- **Основные аргументы**:
    - `prefix` — подстрока или кортеж подстрок для проверки начала строки.
- **Частые параметры**:
    - `start`, `end` — диапазон в пределах строки, где делать проверку.
- **Поведение**:
    - возвращает `True`, если строка (или её срез) начинается с `prefix`.

### `.endswith()`
```python
s.endswith(suffix, start=0, end=len(s))
```
- **Основные аргументы**:
    - `suffix` — подстрока или кортеж подстрок для проверки конца строки.
- **Частые параметры**:
    - `start`, `end` — диапазон в пределах строки, где делать проверку.
- **Поведение**:
    - возвращает `True`, если строка (или её срез) заканчивается на `suffix`.
	
## Методы списков (list)
> [!info]
> Методы вызываются так: `lst.method(...)`, где `lst` — список.
### `.append()`
```python
lst.append(x)
```
- **Основные аргументы**:
    - `x` — элемент, который нужно добавить в список.
- **Поведение**:
    - добавляет `x` в конец списка;
    - изменяет исходный список, не создавая новый.

### `.extend()`
```python
lst.extend(iterable)
```
- **Основные аргументы**:
    - `iterable` — итерируемый объект (список, кортеж, множество, строка и т.п.), элементы которого нужно добавить.
- **Поведение**:
    - добавляет в конец списка все элементы из `iterable`;
    - изменяет исходный список.

### `.insert()`
```python
lst.insert(index, x)
```
- **Основные аргументы**:
    - `index` — позиция, куда нужно вставить элемент.
    - `x` — элемент для вставки.
- **Поведение**:
    - вставляет `x` перед указанным индексом, сдвигая элементы вправо.

### `.pop()`
```python
lst.pop(index=-1)
```
- **Частые параметры**:
    - `index` — индекс элемента, который нужно удалить; по умолчанию -1 (последний элемент).
- **Поведение**:
    - удаляет элемент по индексу и возвращает его;
    - при отсутствии индекса удаляет и возвращает последний элемент.

### `.remove()`
```python
lst.remove(x)
```
- **Основные аргументы**:
    - `x` — значение, которое нужно удалить из списка.
- **Поведение**:
    - удаляет первое вхождение `x`;
    - выбрасывает `ValueError`, если элемента нет.

### `.clear()`
```python
lst.clear()
```
- **Поведение**:
    - удаляет все элементы списка, делая его пустым.

### `.sort()`
```python
lst.sort(key=None, reverse=False)
```
- **Частые параметры**:
    - `key` — функция, которая получает элемент и возвращает ключ для сортировки.
    - `reverse` — если `True`, сортирует в обратном порядке.
- **Поведение**:
    - сортирует список на месте, возвращает `None`.

### `.reverse()`
```python
lst.reverse()
```
- **Поведение**:
    - разворачивает элементы списка в обратном порядке, изменяя его на месте.

### `.index()`
```python
lst.index(x, start=0, end=len(lst))
```
- **Основные аргументы**:
    - `x` — значение для поиска.
- **Частые параметры**:
    - `start` — начальный индекс поиска.
    - `end` — конечный индекс (не включительно).
- **Поведение**:
    - возвращает индекс первого вхождения x или выбрасывает `ValueError`, если элемента нет.

### `.count()`
```python
lst.count(x)
```
- **Основные аргументы**:
    - `x` — значение, количество вхождений которого нужно посчитать.
- **Поведение**:
    - возвращает число вхождений x в списке.

## Методы словарей (dict)
> [!info]
> Методы вызываются так: `d.method(...)`, где `d` — словарь.
### `.get()`
```python
d.get(key, default=None)
```
- **Основные аргументы**:
    - `key` — ключ, значение которого нужно получить.
- **Частые параметры**:
    - `default` — значение по умолчанию, если ключа нет (по умолчанию `None`).
- **Поведение**:
    - возвращает `d[key]`, если ключ есть, иначе — `default` без ошибки.

### `.keys()`
```python
d.keys()
```
- **Поведение**:
    - возвращает «представление» (`view`) всех ключей словаря, по которому можно итерироваться.

### `.values()`
```python
d.values()
```
- **Поведение**:
    - возвращает представление всех значений словаря.

### `.items()`
```python
d.items()
```
- **Поведение**:
    - возвращает представление пар (`key`, `value`);
    - удобно для перебора словаря в цикле `for key, value in d.items()`.

### `.update()`
```python
d.update(other)
d.update(other_dict)
d.update(key1=val1, key2=val2, ...)
```
- **Основные аргументы**:
    - `other` — другой словарь или итерируемый объект пар (`key`, `value`);
    - или именованные аргументы `key=value`.
- **Поведение**:
    - добавляет или обновляет пары `key: value` в словаре `d`;
    - существующие ключи перезаписываются;
    - изменяет словарь на месте, возвращает `None`.

### `.pop()`
```python
d.pop(key)
d.pop(key, default)
```
- **Основные аргументы**:
    - `key` — ключ, который нужно удалить.
- **Частые параметры**:
    - `default` — значение, которое вернуть, если ключа нет.
- **Поведение**:
    - удаляет ключ `key` и возвращает его значение;
    - если ключ не найден и `default` не указан — `KeyError`;
    - если `default` указан и ключа нет — возвращает `default`.

### `.popitem()`
```python
d.popitem()
```
- **Поведение**:
    - удаляет и возвращает пару (`key`, `value`);
    - в современных версиях  — последнюю добавленную пару (`LIFO`).

### `.setdefault()`
```python
d.setdefault(key, default=None)
```
- **Основные аргументы**:
    - `key` — ключ, значение которого нужно получить или создать.
- **Частые параметры**:
    - `default` — значение, которое будет записано и возвращено, если ключа нет.
- **Поведение**:
    - если ключ есть — возвращает его значение;
    - если ключа нет — создаёт `d[key] = default `и возвращает `default`.

### `.clear()`
```python
d.clear()
```
- **Поведение**:
    - удаляет все элементы словаря, делая его пустым.
# Модуль math
```python
# Подключение модуля
import math
```
### `math.pi`
- Число $π$ (отношение длины окружности к диаметру).

### `math.e`
- Основание натурального логарифма ≈ 2.71828.

### `math.ceil()`
```python
math.ceil(x)
```
- **Основные аргументы**:
    - $x$ — число.
- **Поведение**:
    - возвращает наименьшее целое число, которое не меньше $x$ (округление вверх).

### `math.floor()`
```python
math.floor(x)
```
- **Основные аргументы**:
    - $x$ — число.
- **Поведение**:
    - возвращает наибольшее целое число, которое не больше $x$ (округление вниз). 

### `math.trunc()`
```python
math.trunc(x)
```
- **Основные аргументы**:
    - $x$ — число.
- **Поведение**:
    - отбрасывает дробную часть, двигаясь к нулю (для положительных как `floor`, для отрицательных как `ceil`).

### `math.prod()`
```python
math.prod(iterable, *, start=1)
```
- **Основные аргументы**:
    - `iterable` — итерируемый объект с числами (список, кортеж и т.п.).
- **Частые параметры**:
    - `start` — начальное значение произведения (по умолчанию 1).​
- **Поведение**:
    - возвращает произведение всех элементов `iterable`, умноженных на `start`.

### `math.factorial()`
```python
math.factorial(n)
```
- **Основные аргументы**:
    - $n$ — целое неотрицательное число.
- **Поведение**:
    - возвращает $n!$ (произведение чисел от $1$ до $n$).

### `math.gcd()`
```python
math.gcd(a, b)
```
- **Основные аргументы**:
    - $a$, $b$ — целые числа.
- **Поведение**:
    - возвращает наибольший общий делитель чисел $a$ и $b$.

### `math.sqrt()`
```python
math.sqrt(x)
```
- **Основные аргументы**:
    - $x$ — неотрицательное число.
- **Поведение**:
    - возвращает квадратный корень из $x$.

### `math.pow()`
```python
math.pow(x, y)
```
- **Основные аргументы**:
    - $x$ — основание;
    - $y$ — показатель степени.
- **Поведение**:
    - возвращает $x$ в степени $y$ как число с плавающей точкой.

### `math.exp()`
```python
math.exp(x)
```
- **Основные аргументы**:
    - $x$ — число.
- **Поведение**:
    - возвращает $e**x$.

### `math.log()`
```python
math.log(x, base=math.e)
```
- **Основные аргументы**:
    - $x$ — положительное число.
- **Частые параметры**:
    - `base` — основание логарифма (по умолчанию $e$).
- **Поведение**:
    - натуральный логарифм при `base` по умолчанию, либо логарифм по указанному основанию.

### `math.log10()` / `math.log2()`

# Модуль itertools
```python
# Подключение модуля
import itertools
```

### `itertools.zip_longest()`
```python
itertools.zip_longest(*iterables, fillvalue=None)
```
- **Основные аргументы**:
    - `*iterables` — несколько итерируемых объектов.
- **Частые параметры**:
    - `fillvalue` — значение, которым заполняются «дыры», если один из итераторов закончился раньше (по умолчанию `None`).
- **Поведение**:
    - итерируется до длины самого длинного входного итератора;
    - возвращает кортежи, где отсутствующие элементы заменены `fillvalue`.

### `itertools.chain()`
```python
itertools.chain(*iterables)
```
- **Основные аргументы**:
    - `*iterables` — несколько итерируемых объектов.
- **Поведение**:
    - «склеивает» несколько последовательностей в одну цепочку:
    - сначала все элементы первого, затем второго и т.д.

### `itertools.chain.from_iterable()`
```python
itertools.chain.from_iterable(iterable_of_iterables)
```
- **Основные аргументы**:
    - `iterable_of_iterables` — итерируемый объект, содержащий другие итерируемые.
- **Поведение**:
    - удобен для «расплющивания» списка списков.

### `itertools.count()`
```python
itertools.count(start=0, step=1)
```
- **Частые параметры**:
    - `start` — начальное значение последовательности;
    - `step` — шаг между значениями.
- **Поведение**:
    - бесконечная арифметическая прогрессия: `start, start + step, start + 2*step, ....`

### `itertools.cycle()`
```python
itertools.cycle(iterable)
```
- **Основные аргументы**:
    - `iterable` — итерируемый объект (список, строка и т.п.).
- **Поведение**:
    - бесконечно повторяет элементы переданного `iterable` по кругу.

### `itertools.repeat()`
```python
itertools.repeat(object, times=None)
```
- **Основные аргументы**:
    - `object` — значение, которое нужно повторять.
- **Частые параметры**:
    - `times` — количество повторов (если `None`, повторяет бесконечно).
- **Поведение**:
    - возвращает один и тот же объект много раз (конечно или бесконечно).


### `itertools.product()`
```python
itertools.product(*iterables, repeat=1)
```
- **Основные аргументы**:
    - `*iterables` — коллекции (списки, кортежи и др.), над которыми берётся декартово произведение.
- **Частые параметры**:
    - `repeat` — количество повторений всего набора `iterables` (как будто переданы несколько раз).
- **Поведение**:
    - генерирует кортежи всех возможных комбинаций элементов (декартово произведение).

### `itertools.permutations()`
```python
itertools.permutations(iterable, r=None)
```
- **Основные аргументы**:
    - `iterable` — исходная коллекция;
- **Частые параметры**:
    - `r` — длина перестановки (по умолчанию длина `iterable`).
- **Поведение**:
    - перестановки элементов с учётом порядка, без повторов элементов в одном кортеже.

### `itertools.combinations()`
```python
itertools.combinations(iterable, r)
```
- **Основные аргументы**:
    - `iterable` — исходная коллекция;
    - `r` — длина сочетания (обязательный аргумент).
- **Поведение**:
    - сочетания без повторений, порядок внутри кортежа не важен;
    - каждый набор элементов встречается один раз.

### `itertools.combinations_with_replacement()`
```python
itertools.combinations_with_replacement(iterable, r)
```
- **Основные аргументы**:
    - `iterable` — исходная коллекция;
    - `r` — длина сочетания.
- **Поведение**:
    - сочетания с возможностью повторения элементов;
    - порядок внутри кортежа не важен.

### `itertools.compress()`
```python
itertools.compress(data, selectors)
```
- **Основные аргументы**:
    - `data` — итерируемый объект с данными;
    - `selectors` — итерируемый объект с булевыми значениями.
- **Поведение**:
    - пропускает элементы из `data`, оставляя только те, для которых соответствующий элемент в `selectors` является истинным.

### `itertools.filterfalse()`
```python
itertools.filterfalse(predicate, iterable)
```
- **Основные аргументы**:
    - `predicate` — функция, принимающая элемент и возвращающая `True`/`False` (или `None` для всегда `False`);
    - `iterable` — итерируемый объект.
- **Поведение**:
    - пропускает только те элементы, для которых `predicate`(`element`) ложно (инверсный `filter`).

### `itertools.groupby()`
```python
itertools.groupby(iterable, key=None)
```
- **Основные аргументы**:
    - `iterable` — отсортированная по ключу последовательность;
- **Частые параметры**:
    - `key` — функция, извлекающая ключ группировки из элемента.
- **Поведение**:
    - на каждой итерации отдаёт пары (`key`, `group_iter`), где `group_iter` — итератор по подряд идущим элементам с одним и тем же ключом.
    - Важно: `groupby` группирует только подряд идущие элементы с одинаковым ключом, так что обычно `iterable` сортируют по этому ключу заранее.

### `itertools.pairwise()`
```python
itertools.pairwise(iterable)
```
- **Основные аргументы**:
    - iterable — итерируемый объект.
- **Поведение**:
    - генерирует пары «(текущий, следующий)»: `[s0, s1, s2, ...] → (s0, s1), (s1, s2), (s2, s3), ...`

### `itertools.islice()`
```python
itertools.islice(iterable, stop)
itertools.islice(iterable, start, stop[, step])
```
- **Основные аргументы**:
    - `iterable` — итерируемый объект;
    - `start`, `stop`, `step` — как в срезах.
- **Поведение**:
    - ленивый «срез» итератора, не создаёт промежуточный список.

### `itertools.accumulate()`
```python
itertools.accumulate(iterable, func=operator.add)
```
- **Основные аргументы**:
    - `iterable` — итерируемый объект.
- **Частые параметры**:
    - `func` — бинарная функция для накопления (по умолчанию сложение).
- **Поведение**:
    - генерирует последовательность накопленных значений (префиксные суммы, произведения и т.п. при другой `func`).
