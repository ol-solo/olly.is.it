## Длина строки
Длина строки — это количество символов в ней. Для подсчёта длины используют функцию `len()`. В качестве аргумента `len()` принимает строку, а возвращает её длину — целое число.
```python
some_string = 'В этой строке 25 символов'

print(len(some_string)) 

25 
```

Строка может быть и пустой. Длина пустой строки равна нулю.

## Индексация строк
Символы в строке идут друг за другом, при этом у каждого — свой порядковый номер или индекс. **Отсчёт начинается с нуля**, а не с единицы.
![[Pasted image 20260227132643.png]]

Обратиться к символу по его индексу:
```python
this_is_string = 'Это строка'

print(this_is_string[0])
print(this_is_string[4]) 

Э
с 
```

Для получения символа можно использовать и **обратный индекс**. В таком случае мы указываем его отрицательным.
![[Pasted image 20260227132804.png]]
```python
this_is_string = 'Это строка'

print(this_is_string[-1]) # -1 это первый символ с конца, то есть самый последний в строке
print(this_is_string[-2]) # -2 это второй символ с конца или предпоследний в строке 

а
к 
```

Если нужно обратиться к **последнему символу строки**, то используют либо длину строки `len() - 1`, либо обратный индекс `-1`.
```python
this_is_string = 'Это строка'

print(this_is_string[-1])
print(this_is_string[len(this_is_string) - 1]) 

a
a 
```

> [!important]
> Если обратиться к индексу, который **больше** длины строки, произойдёт ошибка.
Другая ошибка возникнет, если попытаться **изменить** символ строки, обратившись к нему по индексу, так как строки — *неизменяемый тип данных*. Можно только сохранить новую строку вместо старой. Но изменить существующую строку не получится.

## Срезы строк
Срезы — почти как индексы. Только они позволяют получить не один, а сразу несколько идущих подряд символов строки.
![[Pasted image 20260227133148.png]]

Чтобы получить срез, используют конструкцию: `[start:end:step]`:
 - `start` — индекс элемента, с которого начинается срез. В нашем примере `start = 0`.
 - `end` — индекс элемента, до которого продолжается срез. Важно, что элемент с этим индексом не включается в сам срез. В нашем примере последний элемент среза под индексом 2, а значит, `end = 3`.
 - `step` — шаг, с которым выбираются символы. По умолчанию `step=1`, и его можно не использовать.
```python
password = 'qwerty'
print(password[0:3]) 

qwe 
```
> [!important]
> Обратите внимание, вывелось только три символа. Четвёртый по порядку и третий по индексу символ r не вывелся. Последний символ диапазона не входит в результат.

Если делать срез с самого начала строки, то необязательно указывать нулевой индекс, можно просто пропустить эту часть:
```python
# Если начало среза не указано, то срез начинается с нулевого символа.
print(city[:3]) # то же, что и city[0:3] 

Par 
```

Если конец подстроки совпадает с концом исходной строки, можно пропустить индекс конца среза:
```python
# Если конец среза не указан, то срез продолжается до конца строки.
print(city[2:]) # тоже, что и city[2:5] 

ris 
```

Если не указывать ни конец, ни начало строки, то вернётся её точная копия:
```python
print(city[:]) # то же, что и city[0:5] 

Paris  
```

В срезах можно использовать обратные индексы.
```python
print(city[-3:]) # тоже, что и city[2:5] 

ris  
```

Используя `step`, мы можем указать шаг, с которым получаем символы. 
```python
city = 'Paris'
substring = city[::2] # Из всей строки выбираем каждый нечётный элемент
print(substring) 

Prs 
```

Если указать `step = -1`, то вернётся строка в обратном порядке:
```python
city = 'Paris'
reverse_string = city[::-1]
print(reverse_string) 

siraP 
```

## Как работать со строками
В Python есть набор встроенных **методов**, которые предназначены для работы со строками. Методы — это функции, как, например, `print()` или `len()`, но которые могут работать только с определённым типом данных (в нашем случае — это строки `str`). Методы вызываются через точку после самой строки:
```python
word = 'привет'

# метод upper сделает все буквы прописными
new_word = word.upper()
print(new_word) 

ПРИВЕТ 
```

Несколько полезных методов:
- `replace()` — заменит часть строки;
- `find()` — ищет подстроку в строке;
- `upper()` — заменяет все буквы в строке на прописные;
- `lower()` — заменяет все буквы на строчные.

## Замена части строки
Метод `replace()` принимает два аргумента:
1. подстроку, которую нужно заменить;
2. подстроку, которая заменит исходную.
```python
some_string = 'leaving'
print(some_string)

# заменяем подстроку leav строкой stay
some_string = some_string.replace('leav', 'stay')
print(some_string) 

leaving
staying 
```

## Поиск подстроки
Метод `find()` ищет подстроку в строке. Если найдёт — вернёт индекс символа, начиная с которого подстрока встречается в исходной строке.
Чаще всего методу передают один аргумент — подстроку, которую он должен найти:
```python
word = 'hey, hello, hello, hello, how low?'
# подстрока hello входит в строку несколько раз,
# но метод вернёт место начала первого вхождения
ind = word.find('hello')

print(ind) 

5 
```

Для `find()` можно задать диапазон индексов, в котором он должен искать. Инструкцию в этом коде можно прочитать так: «Найди первое вхождение `hello` в строке из переменной `word`, начиная с пятнадцатого символа и заканчивая двадцать четвёртым».
```python
word = 'hey, hello, hello, hello, how low?'

# Три аргумента: подстрока, индекс начала поиска, индекса завершения, увеличенный на 1
ind = word.find('hello', 15, 25)

print(ind) 

19 
```

Если `find()` ничего не найдёт, он вернёт `-1`:
```
hi = 'Привет!'
bye_ind = hi.find('Пока!')

print(bye_ind)

-1 
```

У метода `find()` есть аналог — `rfind()`, тут поиск подстроки начинается с правой стороны. Если `find()` ищет первое вхождение подстроки, то `rfind()` ищет последнее вхождение.
```
word = 'hey, hello, hello, hello, how low?'
# подстрока hello входит в строку несколько раз,
# но метод вернёт номер последнего вхождения
ind = word.rfind('hello')

print(ind) 

19 
```

## Изменение регистра
Методы `upper()` и `lower()` возвращают строки, в которых исходная строка приведена к верхнему и нижнему регистру соответственно:
```python
word = 'AbRacaDABRA'
lower_word = word.lower()
upper_word = word.upper()
print(lower_word, upper_word)

abracadabra ABRACADABRA
 
```

## Экранирование специальных символов
Чтобы Python воспринимал кавычки как часть строки, нужно сделать экранирование кавычек. При экранировании Python считывает специальный символ и подставляет вместо него то, что считает нужным. Символы для экранирования начинаются со слэша `\`.
```
phrase = 'I don\'t want to stop'
print(phrase)
print(len(phrase)) 

I don't want to stop
20 
```

Есть и другие специальные символы, которые используют для экранирования:
- `\n` — перенос строки;
- `\t` — табуляция;
- `\\` — слэш.

Такие сочетания считаются за один символ.

Метод `.strip` удаляет лишние пробелы в начале и в конце строки.
```
print('       Иван Иванович Иванов       '.strip()) 

Иван Иванович Иванов 
```

> [!important]
> Пробелы в середине строки остались нетронутыми.

## Форматирование строк
Форматирование — это способ создания строки, в которой значения переменных подставляются в определённых местах, как по шаблону.

1. Метод `format()`. В строке в фигурных скобках `{ }` указывают название переменной для подстановки. Аргументами метода `format` будут эти самые переменные.
```python
cr_percent = 3
dau = 1000
dau_delta = 5

s = """
По итогам квартала конверсия составила {cr_percent}%.
В тоже время показатель DAU составил {dau} пользователей, 
что на {dau_delta}% выше прогноза.
"""
print(s.format(cr_percent = cr_percent, dau = dau, dau_delta = dau_delta)) 

По итогам квартала конверсия составила 3%.
В тоже время показатель DAU составил 1000 пользователей, 
что на 5% выше прогноза. 
```

2. `f-строки`. Мы пишем символ `f` перед строкой и точно так же используем фигурные скобки `{ }` и именованные переменные, но использовать метод `format` уже необязательно.
```
cr_percent = 3
dau = 1000
dau_delta = 5

s = f"""
По итогам квартала конверсия составила {cr_percent}%.
В тоже время показатель DAU составил {dau} пользователей, 
что на {dau_delta}% выше прогноза.
"""
print(s) 

По итогам квартала конверсия составила 3%.
В тоже время показатель DAU составил 1000 пользователей, 
что на 5% выше прогноза. 
```
