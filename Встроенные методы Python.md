> [!info]
> Методы вызываются через точку после самой строки

### `.replace()`
заменит часть строки. Принимает два аргумента:
1. подстроку, которую нужно заменить;
2. подстроку, которая заменит исходную.
```python
some_string = 'leaving'
# заменяем подстроку leav строкой stay
some_string = some_string.replace('leav', 'stay')
```

### `.find()`
ищет подстроку в строке. Если найдёт — вернёт индекс символа, начиная с которого подстрока встречается в исходной строке. Принимает аргументы:
1. подстроку, которую он должен найти
```python
word = 'hey, hello, hello, hello, how low?'
# подстрока hello входит в строку несколько раз,
# но метод вернёт место начала первого вхождения
ind = word.find('hello')
```
2. диапазон индексов, в котором он должен искать
```python
word = 'hey, hello, hello, hello, how low?'
# Три аргумента: подстрока, индекс начала поиска, индекса завершения, увеличенный на 1
ind = word.find('hello', 15, 25)
```


### `.rfind()`
Аналог `.find()`, поиск подстроки начинается с правой стороны. Если `find()` ищет первое вхождение подстроки, то `rfind()` ищет последнее вхождение.


### `.upper()`
заменяет все буквы в строке на прописные

### `.lower()`
заменяет все буквы на строчные

### `.strip`
удаляет лишние пробелы в начале и в конце строки

### `format()`
В строке в фигурных скобках `{ }` указывают название переменной для подстановки. Аргументами метода `format` будут эти самые переменные.
```python
cr_percent = 3
dau = 1000
dau_delta = 5

s = """
По итогам квартала конверсия составила {cr_percent}%.
В тоже время показатель DAU составил {dau} пользователей, 
что на {dau_delta}% выше прогноза.
"""
print(s.format(cr_percent = cr_percent, dau = dau, dau_delta = dau_delta))
```

### `.append()`
На вход метод принимает элемент, который нужно добавить. Метод добавляет элемент в конец списка.
> [!important]
> изменяет изначальный список, а не создаёт новый

### `.extend()`
На вход он принимает другой список, которым нужно расширить изначальный.

### `.pop()`
принимает индекс элемента, который нужно удалить, и возвращает сам элемент, который был удалён.

### `.sort()`
Если список состоит из чисел, то их можно отсортировать в порядке возрастания или убывания. По умолчанию список будет отсортирован в порядке возрастания — от меньшего к большему. Чтобы отсортировать наоборот в порядке убывания, используют параметр `reverse=True`

### `.split()`
На вход метод принимает символ-разделитель — это символ, которым мы «дробим» строку. Метод возвращает список из разделённых строк.

### `.join()`
собрать целую строку из элементов списка. объединяет элементы списка в одну строку, разделяя сами элементы строкой `sep`.

### `.islower()`
возвращает `True`, если в строке нет прописных букв

### `.isdigit()`
возвращает `True`, если строка состоит только из числовых знаков

### `.isalpha()`
возвращает `True`, если в строке только буквы, без пробелов и знаков препинания

### `.isspace()` 
возвращает `True`, если в строке только пробелы

### `.startswith()` 
`startswith(prefix)` возвращает `True`, если строка начинается с указанного набором символов `prefix`

### `.endswith()`
`endswith(suffix)` возвращает `True`, если строка заканчивается указанным набором символов `suffix`
