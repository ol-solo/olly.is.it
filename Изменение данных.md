# Действия над названиями столбцов
При работе с данными в pandas аналитик часто использует названия или метки столбцов. Метки столбцов следует оформлять в одной нотации и на одном языке. Названия столбцов не должны дублировать названия встроенных функций Python

## Стиль названия столбцов и переменных
|**Нотация**|**Пример**|**Описание**|
|---|---|---|
|Snake Case (змеиный_регистр)|`column_name`|Все слова пишутся в нижнем регистре и разделяются не пробелом, а символом подчёркивания _|
|Camel Case (верблюжийРегистр):|`columnName`|Каждое слово, кроме первого, начинается с заглавной буквы, слова не разделяются символами и пишутся слитно|
|Pascal Case (ПаскальРегистр)|`ColumnName`|Каждое слово, включая первое, начинается с заглавной буквы, слова не разделяются символами и пишутся слитно|
|Upper Case (ВЕРХНИЙ_РЕГИСТР)|`COLUMN_NAME`|Все буквы — заглавные, а слова разделяются символом подчёркивания|
|Kebab Case (кебаб-регистр)|`column-name`|Все слова пишутся в нижнем регистре и разделяются не пробелом, а дефисом -|

Обычно, для названия переменных и меток столбцов при работе с датафреймами в pandas используют Snake Case. Этому же стилю следуют стандартные и распространённые библиотеки Python. Это легко запомнить, так как питон — это змея. Но в специальных случаях при написании кода в Python используются и другие стили — Pascal Case для названия классов (например, `CarModel`) и Upper Case для названия констант (например, `TIME`).

## Атрибут датафрейма `columns`
У датафреймов в pandas есть специальные атрибуты, которые представляют собой список названий всех столбцов или строк в виде объекта Index.
```python
# Используем атрибут columns для вывода названия столбцов music_df:
print(music_df.columns)

# В результате получаем список всех столбцов объекта Index
Index(['user_id', 'total_play', 'artist', 'genre', 'track'], dtype='object')
```
Этот объект похож на список: используя индексацию, можно вывести определённое значение
```python
# Выведем название второго по счёту столбца:
print(music_df.columns[1])
total_play
```
в отличие от обычного списка, в объекте Index нельзя изменить какое-то определённое значение, поскольку Index представляет собой неизменяемую структуру данных, аналогичную массиву или множеству. Можно изменить только весь Index сразу.
```python
# Зададим данные для датафрейма: названия столбцов и содержание строк:
columns = ['Product ID', 'Category', 'Description', 'Price']
data = [[10001, 'Телефоны','Смартфон', 15999],
       [10002, 'Компьютеры', 'Монитор', 17200],
       [10003, 'Кухонная техника', 'Микроволновка', 6350]]

# Создадим новый датафрейм df:
df = pd.DataFrame(data=data, columns=columns)
print(df)

   Product ID          Category    Description  Price
0       10001          Телефоны       Смартфон  15999
1       10002        Компьютеры        Монитор  17200
2       10003  Кухонная техника  Микроволновка   6350

# Зададим список с новыми названиями столбцов:
new_columns = ['product_id','category','description','price']

# Используя атрибут columns, зададим новый индекс столбцов df:
df.columns = new_columns
print(df)

   product_id          category    description  price
0       10001          Телефоны       Смартфон  15999
1       10002        Компьютеры        Монитор  17200
2       10003  Кухонная техника  Микроволновка   6350
```
> [!important]
> список `new_columns` должен содержать такое же количество значений, как количество столбцов. Такой подход позволяет изменить все названия столбцов в одно действие. В этом случае мы должны быть уверены как в последовательности столбцов, так и в их количестве. Если вдруг изменится последовательность столбцов, например, при обновлении данных, то столбцы могут получить неверное название, что повлияет на работоспособность кода.

Поскольку Index это неизменяемый массив данных, то для изменения одного из его значений Index можно перевести в список. Затем изменить необходимое значение, обратившись к нему по его индексу, и используя атрибут `columns` завершить изменения.
```python
# Создадим список на основе индекса столбцов датафрейма df:
columns = df.columns.to_list()

# Изменим третий по счёту элемент списка columns:
columns[2] = 'info'

# Используя атрибут columns зададим новый индекс столбцов df:
df.columns = columns
print(df)

   product_id          category           info  price
0       10001          Телефоны       Смартфон  15999
1       10002        Компьютеры        Монитор  17200
2       10003  Кухонная техника  Микроволновка   6350
```
## Метод `.rename()`
Для изменения названия столбцов необходимо использовать аргумент `columns` метода `.rename()`, в котором передаётся словарь, содержащий в качестве ключа исходное название столбца, а в качестве значения — новое
```python
# Зададим данные для датафрейма: названия столбцов и содержание строк:
columns = ['Product ID', 'Category', 'Description', 'Price']
data = [[10001, 'Телефоны','Смартфон', 15999],
       [10002, 'Компьютеры', 'Монитор', 17200],
       [10003, 'Кухонная техника', 'Микроволновка', 6350]]

# Создадим новый датафрейм df:
df = pd.DataFrame(data=data, columns=columns)

# Используя метод rename() изменим названия столбцов df:
df = df.rename(columns={'Product ID': 'product_id',
                        'Category': 'category',
                        'Description': 'description',
                        'Price': 'price'})

print(df)

   product_id          category    description  price
0       10001          Телефоны       Смартфон  15999
1       10002        Компьютеры        Монитор  17200
2       10003  Кухонная техника  Микроволновка   6350
```
Используя метод `.rename()`, мы можем изменить названия как всех столбцов, так и только их часть. При этом нет необходимости соблюдать последовательность столбцов
```python
# Изменим название столбца 'description' на 'info':
# Используя атрибут inplace=True, сохраним изменения в текущем датафрейме:
df.rename(columns={'description': 'info'}, inplace=True)

print(df)

   product_id          category           info  price
0       10001          Телефоны       Смартфон  15999
1       10002        Компьютеры        Монитор  17200
2       10003  Кухонная техника  Микроволновка   6350
```
в методе `.rename()` мы используем атрибут `inplace=True`. Этот параметр, как и в некоторых других методах pandas, позволяет внести изменения в исходный `DataFrame` без дополнительной перезаписи переменной датафрейма.

Метод `.rename()`, в сравнении с ручным присвоением новых значений Index, используя `columns`, помогает избежать возможных ошибок, если вдруг по какой-то причине изменится последовательность столбцов. Также этот метод не требует перечисления названия столбцов, которые не используются при изменении названий, что оптимизирует код и упрощает его чтение. А если какого-то столбца, который есть в словаре, не окажется в датафрейме, то метод `.rename()` пропустит этот столбец, и при этом работа кода не вызовет ошибку.

## Перевод названия столбцов в Snake Case
```python
# Зададим исходную строку для перевода в нижний регистр:
name = 'Column Name'

# Используем метод .lower() для перевода всех символов в нижний регистр:
lower_name = name.lower()
print(lower_name)

# Используем метод .replace() для замены пробела на нижнее подчёркивание:
lower_name = lower_name.replace(' ', '_')
print(lower_name)

column_name
```

мы можем совместить два метода `.lower()` и `.replace()` в одну цепочку действий, что позволит изменить нотацию названия на Snake Case:
```python
# Зададим исходную строку для перевода в нижний регистр:
name = 'Column Name'

# Используем метод .lower() для перевода всех символов в нижний регистр:
snake_case_name = name.lower().replace(' ', '_')
print(snake_case_name)

column_name
```
Однако эти методы работают только со строками и их нельзя применить к массивам. А названия столбцов представлены в виде Index.

Эту задачу можно решить, если применять методы `.lower()` и `.replace()` для каждого значения массива индексов столбцов. С этим отлично справится цикл в сочетании с атрибутом датафрейма `columns` и методом `.rename()`:
```python
# Зададим данные для датафрейма: названия столбцов и содержание строк:
columns = ['Product ID', 'Category', 'Description', 'Price']
data = [[10001, 'Телефоны','Смартфон', 15999],
       [10002, 'Компьютеры', 'Монитор', 17200],
       [10003, 'Кухонная техника', 'Микроволновка', 6350]]

# Создадим новый датафрейм df:
df = pd.DataFrame(data=data, columns=columns)

# Создадим пустой словарь для записи переименований
rename_dict = {}

# Заполним словарь в цикле
for column in df.columns:
    rename_dict[column] = column.lower().replace(' ', '_')

# Произведём переименование
df.rename(columns=rename_dict, inplace=True)

    
print(df)

     product_id          category    description  price
0       10001          Телефоны       Смартфон  15999
1       10002        Компьютеры        Монитор  17200
2       10003  Кухонная техника  Микроволновка   6350
```

## Изменение названия столбцов с помощью аксессора `str`
`.str` позволяет использовать различные векторизованные строковые методы, в том числе `.lower()`, `.replace()`, описанные выше, с текстовыми данными внутри объектов Series и Index библиотеки pandas, которые имеют тип данных `object`.
```python
# Зададим данные для датафрейма: названия столбцов и содержание строк:
columns = ['Product ID', 'Category', 'Description', 'Price']
data = [[10001, 'Телефоны','Смартфон', 15999],
       [10002, 'Компьютеры', 'Монитор', 17200],
       [10003, 'Кухонная техника', 'Микроволновка', 6350]]

# Создадим новый датафрейм df:
df = pd.DataFrame(data=data, columns=columns)
    
print(df.columns)

Index(['Product ID', 'Category', 'Description', 'Price'], dtype='object')

# Измененим все объекты внутри Index при помощи метода lower():
print(df.columns.str.lower())

Index(['product id', 'category', 'description', 'price'], dtype='object')
```
позволил нам обратиться к объектам Index и изменить их с помощью метода `.lower()`. Поскольку этот метод векторный, то изменения произошли для всего массива Index сразу. Перебирать каждое значение, как в случае с использованием цикла, не понадобилось.

важно отметить, что после использования `.str.lower()` тип объекта `Index` не изменился.

поскольку `str` предоставляет доступ к данным только для одного строкового метода, то использовать этот атрибут в цепочке методов следует перед каждым методом:
```python
# Зададим данные для датафрейма: названия столбцов и содержание строк:
columns = ['Product ID', 'Category', 'Description', 'Price']
data = [[10001, 'Телефоны','Смартфон', 15999],
       [10002, 'Компьютеры', 'Монитор', 17200],
       [10003, 'Кухонная техника', 'Микроволновка', 6350]]

# Приведем названия столбцов к Snake Case:
df.columns = df.columns.str.lower().str.replace(' ', '_')

print(df.columns)

Index(['product_id', 'category', 'description', 'price'], dtype='object')
```

## Изменение названия столбцов: общий подход
- Атрибут DataFrame `columns`
Замена названий всех столбцов на заранее подготовленный список. Оптимально использовать, когда последовательность и количество столбцов известно. Например, при изменении названий столбцов после создания сводных таблиц.

```
# Синтаксис:
df.columns = [элементы через запятую] 

# Пример:
df.columns = ['column_a','columns_b'] 
```

- Метод pandas `.rename()`

```
# Синтаксис:
df.rename(columns={пары значений}, inplace = True) 
```

```
# Пример:
df.rename(columns={'A': 'column_a', 'B': 'column_b'}, inplace = True) 
```

Замена названий всех или только некоторых столбцов. Используется словарь, содержащий старое и новое названия. Не зависит от последовательности столбцов и их количества. Часто используется при изучении динамических данных и при замене названия единичных столбцов. Делает код более читаемым и минимизирует ошибки при обновлении данных. Параметр `inplace = True` указывает на замену значений индекса в датафрейме сразу при выполнении инструкции.

- Атрибут Index `str` и строковые методы `.lower()` и `.replace()`

```
# Синтаксис:
df.columns = df.columns.str.lower().str.replace() 
```

```
# Пример:
df.columns = df.columns.str.lower().str.replace(' ', '_') 
```

Позволяет быстро перевести названия всех столбцов в нотацию Snake Case. При этом, будут заменяться все названия столбцов, даже неизвестные на момент написания кода. Использовать атрибут `str` необходимо перед каждым строковым методом — `.lower()` и `.replace()`. Используются векторные преобразования, без перебора значений Index.

# Методы замены данных
## Замена данных с помощью метода `.replace()`
С помощью метода `.replace()` можно заменить одни данные на другие как во всём датафрейме, так и в отдельных столбцах или строках.
```python
DataFrame.replace(to_replace=None, value=None, inplace=False, limit=None, regex=False)
```
Метод имеет несколько атрибутов:
- `to_replace` — значение или список значений, которые необходимо заменить.
- `value` — значение или их список, на которые будет проводиться замена.
- `inplace` — логический атрибут, который указывает, изменять датафрейм на месте или нет. По умолчанию стоит значение `False` — не изменять на месте.
- `limit` — устанавливает максимальное количество значений, которые необходимо заменить в датафрейме. По умолчанию ограничений нет.
- `regex` — логический атрибут, который указывает, являются ли `to_replace` и `value` регулярным выражением.

```python
# Заменим название категории 'Электроника' на англоязычное название 'electronics':
df.replace('Электроника', 'electronics', inplace=True)

print(df)
```

> [!important]
> Метод `.replace()` чувствителен к регистру значений. Например, в инструкции `df.replace('электроника', 'electronics', inplace=True)` заменяемое значение не учитывает регистр, поэтому её выполнение ничего не изменит.

```python
# Заменим название категории 'Смартфоны' на название 'electronics':
df['category'].replace('Смартфоны', 'electronics', inplace=True)

print(df)
```

Если применить метод `.replace()` из этого примера ко всему датафрейму, то изменятся ещё и значения в столбце `group`. Это будет ошибкой, поскольку `Смартфоны` — корректное название группы. Поэтому здесь уместна замена данных только в столбце `category`.

В методе `.replace()` для замены значений можно использовать списки.
```python
# Создадим два списка – один со старым названием категорий, другой – с новым:
old_category = ['Бытовая техника', 'Компьютерная техника']
new_category = ['appliances', 'computers']

# Заменим название категории согласно двум спискам:
df['category'].replace(old_category, new_category, inplace=True)

print(df)
```

можно использовать словарь, где ключ представлен старым названием, а значение — новым:

```python
# Создадим словарь для замены значений:
replace_dict = {'Бытовая техника':'appliances',
                'Компьютерная техника':'computers'}

# Заменим название категории согласно словарю:
df['category'].replace(replace_dict, inplace=True)

print(df) 
```

На промежуточной стадии анализа использовать `inplace=True` стоит осторожно — чтобы вернуть изменения, придётся перезагрузить все исходные данные и запустить скрипт с начала. В этом случае удобнее сохранять изменение значений в виде нового объекта — датафрейма или серии. Для этого не прописывают в методе атрибут `inplace` — по умолчанию возможность перезаписи исходных данных отключена (`inplace=False`).
```python
# Создадим список с названием групп товаров для замены названия:
group_list = ['Смартфоны','Умные часы','Планшет','Мобильный телефон']

# Заменим все значения списка group_list на gadgets:
df['new_group'] = df['group'].replace(group_list, 'gadgets')

print(df)
```

Код выше можно оптимизировать, если при создании прописать логическое условие:

```
# Создадим новый столбец 'is_popular' с помощью логической маски:
df['is_popular'] = (df['votes'] > 500) & (df['rating'] > 4.7)
```

Вместе с логической индексацией можно использовать метод `.replace()`.
```python
# Создадим новый столбец 'is_popular' со значением по умолчанию False:
df['is_popular'] = False

# Используя логическую индексацию, найдём популярные товары:
mask = (df['votes'] > 500) & (df['rating'] > 4.7)
df.loc[mask, 'is_popular'] = df['is_popular'].replace(False, True)
```
## Использование `query()` в условии при замене данных
Для замены данных также можно использовать запросы `query()` в сочетании с оператором `.loc[]`.
```python
# Создадим новый столбец 'is_popular' со значением по умолчанию False:
df['is_popular'] = False

# Используя запрос query, найдём необходимые товары, результат сохраним в mask:
mask = df.query('votes > 1500 and rating > 4.5').index
df.loc[mask, 'is_popular'] = True

# Выведем на экран часть данных:
print(df.iloc[:, [1,4,5,6,7]])
```
> [!important]
> Если передать оператору `.loc[]` для фильтрации данных `mask = df.query('votes > 1500 and rating > 4.5')` с помощью `df.loc[mask, 'is_popular']`, то интерпретатор выдаст ошибку. Оператор `.loc[]` ожидает значения индексов строк, а не срез данных.
