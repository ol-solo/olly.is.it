# Данные, но еще не совсем. Словари. Практика
### Шпаргалка
Словари — это структуры данных, состоящие из пар ключ-значение. Значениями могут быть числа, строки, списки и словари, а ключами, как правило, строки или числа.

Чтобы создать словарь:
- запишите элементы в фигурных скобках `{ }` через запятую;
- разделяйте ключ и значение двоеточием `:`.

Чтобы обратиться к значению в словаре, укажите ключ в квадратных скобках.

Проверить, есть ли ключ в словаре, можно с помощью метода `get()`. Найти пару ключ-значение поможет метод `items()`.

Чтобы добавить новый ключ в словарь, нужно присвоить ему значение. Например:
```
country_capitals = {
    'США': 'Вашингтон',
    'Италия': 'Рим',
    'Англия': 'Лондон'
    }
country_capitals['Россия'] = 'Москва'

print(country_capitals) 
```

Значения менять можно, ключи — нет. Чтобы удалить элемент словаря, используйте метод `pop()`.

# Фильтрация по строкам при помощи индексации. Практика
### Шпаргалка
В предыдущем уроке вы изучили, как при помощи индексации и оператора `[]` вывести необходимые строки или срез строк и столбцов. Вспомним, что оператору `[]` можно передать один аргумент:

- если передать срез целочисленных значений индексов, произойдёт фильтрация по строкам;
- если передать метку столбца или их список, произойдёт фильтрация по столбцам.

Для этого задания добавили небольшую шпаргалку:

|**Выборка (срез) данных**|**оператор `[]`**|
|---|---|
|Один столбец (Series)|`df['column']`|
|Один столбец (DataFrame)|`df[['column']]`|
|Несколько столбцов|`df[['column_A','column_B']]`|
|Все строки, начиная с заданной|`df[0:]`|
|Все строки до заданной|`df[:5]` (не включая индекс 5)|
|Срез строк|`df[2:5]` (не включая индекс 5)|
|Одна строка|`df[4:5]`|
|Одна ячейка|`df[4:5]['artist']`|
|Логическая индексация|`df[mask]`|

Также в прошлом уроке вы познакомились с методом `.sort_values()`, который помогает сортировать данные. Напомним, что у этого метода есть два параметра, которые обычно используются для сортировки значений в столбце:

- `by` — в нём передаются названия столбцов (столбец или серия столбцов), по которым проводится сортировка данных.
- `ascending` — метод, который по умолчанию проводит сортировку от меньшего значения к большему (`ascending=True`). Так, для сортировки в обратном порядке, по убыванию значений используется параметр `ascending=False`.

В этом уроке вы продолжите изучать данные датасета `orders_shop_rel.csv` и потренируетесь использовать оператор `[]` для индексации данных по строкам и столбцам, а также с обращением с логическим маскам.

|**Выборка (срез) данных**|**оператор `[]`**|**оператор `.loc[]`**|**оператор `.iloc[]`**|
|---|---|---|---|
|Один столбец|`df['column']`|`.loc[:, 'column']`|`.iloc[:, index]`|
|Несколько столбцов|`df[['column_A','column_B']]`|`.loc[:, ['column_A','column_B']]`|`.iloc[:, [index_1, index_2]]`|
|Срез столбцов|-|`.loc[:, 'column_A':'column_B']`|`.iloc[:, index_1: index_2]`|

|Все строки до заданной|`df[:5]` (не включая 5)|`.loc[:5]` (включая 5)|`.iloc[:5]` (не включая 5)|
|Срез строк|`df[2:5]` (не включая 5)|`.loc[2:5]` (включая 5)|`.iloc[2:5]` (не включая 5)|
|Несколько строк (не последовательный срез)|-|`.loc[[2,4,6]]`|`.iloc[[2,4,6]]`|
|Одна ячейка|`df[4:5]['column']`|`.loc[4, 'column']`|`.iloc[4, index]`|
|Логическая индексация (строки)|`df[mask]`|`.loc[mask]`|-|
|Логическая индексация (строки + фильтрация по столбцам)|`df[mask][['column_A','column_B']]`|`.loc[mask, ['column_A','column_B']]`|-|

# Фильтрация при помощи query. Практика
### Шпаргалка
В запросе с `query()` можно использовать различные операции сравнения: `!=`, `>`, `>=`, `<`, `<=`, функции-предикаты, а также математические вычисления.

Метод `.query()` позволяет использовать запросы, содержащие два и более условных выражения.

В запросе `.query()` логические условия можно соединять логическими операторами `or` — «или», `and` — «и», `not` — «не». Приводить отдельные условия в скобках в этом случае нет необходимости, поскольку у логических операторов сравнения `not`, `and` и `or` наименьший приоритет при выполнении.

В запросе `query()` можно сравнивать значения с элементами списка. Для этого используются логические операторы `==` и `!=`, или операторы сравнения `in` и `not in`, которые представляют собой краткую запись метода `isin`.

В запросе `query()` можно обращаться к индексу датафрейма, используя ключевое слово `index`.

Если необходимо вывести диапазон строк, то в запросе можно использовать несколько операторов сравнения. Также с использованием метода `.query()` можно вывести только строки с заданным индексом.

Если запрос `query()` помещён в одиночные кавычки, то все другие кавычки, которые используются в запросе должны быть двойными.

В этом уроке вы потренируетесь составлять срезы данных с помощью метода `query` и решите несколько практических задач интернет-магазина полезных девайсов. # Действия над названиями столбцов. Практика
### Шпаргалка
- Атрибут датафрейма `columns`
```
# Синтаксис:
df.columns = [элементы через запятую] 
```
```
# Пример:
df.columns = ['column_a','columns_b'] 
```
Замена названий всех столбцов на заранее подготовленный список. Оптимально использовать, когда последовательность и количество столбцов известно. Например, при изменении названий столбцов после создания сводных таблиц.

- Метод pandas `.rename()`
```
# Синтаксис:
df.rename(columns={пары значений}, inplace = True) 
```
```
# Пример:
df.rename(columns={'A': 'column_a', 'B': 'column_b'}, inplace = True) 
```
Замена названий всех или только некоторых столбцов. Используется словарь, содержащий старое и новое названия. Не зависит от последовательности столбцов и их количества. Часто используется при изучении динамических данных и при замене названия единичных столбцов. Делает код более читаемым и минимизирует ошибки при обновлении данных. Параметр `inplace = True` указывает на замену значений индекса в датафрейме сразу при выполнении инструкции.
- Атрибут Index `str` и строковые методы `.lower()` и `.replace()`
```
# Синтаксис:
df.columns = df.columns.str.lower().str.replace() 
```
```
# Пример:
df.columns = df.columns.str.lower().str.replace(' ', '_') 
```
Позволяет быстро перевести названия всех столбцов в нотацию Snake Case. При этом, будут заменяться все названия столбцов, даже неизвестные на момент написания кода. Использовать атрибут `str` необходимо перед каждым строковым методом — `.lower()` и `.replace()`. Используются векторные преобразования, без перебора значений Index.
