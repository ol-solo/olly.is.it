## Математические вычисления
```python
movies_duration = [142, 175, 152, 195, 201, 154, 178, 139, 133, 163, 136]
total_duration = 0 # создадим переменную-счётчик

for duration in movies_duration: # В этом цикле перебираем каждый элемент из списка `movies_duration`. На каждой итерации цикла переменной `duration` присваивается значение очередного элемента списка.
    total_duration += duration # На каждой итерации цикла текущее значение `duration` добавляется к значению счётчика. Таким образом, на выходе из цикла в `total_duration` будет сумма всех значений списка.

print(total_duration) # Здесь функции `print()` мы передаём переменную `total_duration`, которую создали с самого начала и потом меняли в цикле.

1768 
```
Произведение элементов вычисляют тем же образом, но есть несколько важных особенностей
```python
movies_duration = [142, 175, 152, 195, 201, 154, 178, 139, 133, 163, 136]

mult_duration = 1 # при создании новой переменной счётчика начальное значение `0` необходимо изменить на `1`, чтобы на первом шаге цикла в счётчик сохранился первый элемент списка, умноженный на единицу, а не ноль. В противном случае общий результат будет равен нулю.
for duration in movies_duration:
    mult_duration *= duration # значение счётчика будем менять оператором `*=`, а не `+=`.

print(mult_duration) 

1663161131763038875968000 
```

```python
movies_duration = [142, 175, 152, 195, 201, 154, 178, 139, 133, 163, 136]
total_duration = sum(movies_duration)
print(total_duration)
```

**Использование функции** `prod()` **для произведения значений**. Она встроена в модуль `math` — придётся его импортировать
```python
import math 

movies_duration = [142, 175, 152, 195, 201, 154, 178, 139, 133, 163, 136]
mult_duration = math.prod(movies_duration)
print(mult_duration) 

1663161131763038875968000 
```

```python
movies_duration = [142, 175, 152, 195, 201, 154, 178, 139, 133, 163, 136]
max_value = movies_duration[0] # Объявим переменную `max_value`, которая будет фиксировать максимальное значение, и установим в качестве начального значения первое значение списка `movies_duration[0]`

for duration in movies_duration: # перебирает элементы списка `movies_duration`, присваивая значения переменной `duration`
    if duration > max_value: # сравним очередной элемент с результатом
        max_value = duration

print(max_value)

201 
```

```python
values = [142, 175, 152, 195, 201, -142, -175, -152, -195]
```

Найдём самую низкую стоимость товара среди значений, состоящих из четырёх и более знаков:
```python
values = [750, 341, 950, 899, 159, 3550, 1500, 1105, 1999]
min_value = max(values) # назначено максимальное значение из списка `values`

for value in values:
    if value > 1000: # сравниваем значение списка `value` с `1000`, поскольку мы должны учитывать значения с четырьмя и более знаками.
        if value < min_value: # будет искать минимальное значение из списка
            min_value = value

print(min_value)

```

На вход программе поступает список чисел. Каждый элемент — сторона квадрата. Нужно создать новый список, где каждому исходному элементу соответствует площадь квадрата
```python
sides = [2, 10, 20, 3]

# создаём пустой список
squares = [] 
# объявляем цикл for, 
# который перебирает элементы списка sides
for side in sides: 
    # добавляем к squares 
    # квадрат очередного элемента
    squares.append(side ** 2) 
    
# после цикла «печатаем» результат
print('Список длин сторон квадратов:', sides)
print('Список площадей квадратов:', squares)

Список длин сторон квадратов: [2, 10, 20, 3]
Список площадей квадратов: [4, 100, 400, 9]
```

Если бы задача состояла в изменении исходного списка, помогла бы функция `range()`
```python
sides = [2, 10, 20, 3]

# выведем значения списка до цикла
print('Список длин сторон квадратов:', sides) 

# пройдёмся по индексам списка sides: 0, 1, 2, 3
for i in range(len(sides)):  
    # на каждом шаге обращаемся 
    # к очередному элементу списка и заменяем его 
    # на квадрат этого же элемента
    sides[i] = sides[i] ** 2 

# после цикла «печатаем» результат
print('Список превратился в список площадей квадратов:', sides) 

Список длин сторон квадратов: [2, 10, 20, 3]
Список превратился в список площадей квадратов: [4, 100, 400, 9]
```

## Расчёт факториала
Факториал числа `n` — произведение всех целых чисел от 1 до `n`. Факториал означают так: `!`. Факториалы важны в комбинаторике
```python
number = 8 # задали необходимое число для нахождения факториала в переменной `number`
fact = 1 # Задали переменную-счётчик `fact`, поскольку расчёт факториала начинается с 1, то и значение `fact` зададим равное `1`
for i in range(1, number + 1): # перебираем значения от 1 до `number + 1`. Добавление 1 к `number` обеспечивает включение `number` в диапазон
    fact *= i # значение переменной-счётчика домножается на текущее значение `i`
print(fact) 

40320 
```

функцию `factorial()` — «факториал» из модуля `math`:
```python
import math

print(math.factorial(8)) 

40320 
```

## Подсчёт количества значений в строке
```python
text = 'Коровье молоко'
# счётчик, подсчитывающий общее количество символов
total_letters = 0 

# счётчик, подсчитывающий количество букв о
letter_o_cnt = 0 

for letter in text:
    # на каждом шаге увеличиваем счётчик символов
    total_letters += 1 
    # проверяем в условном выражении, является ли итератор буквой о
    if letter.lower() == 'о': 
        letter_o_cnt += 1       
    
print(f'Всего в строке {total_letters} знаков, из них {letter_o_cnt} букв "о"') 

Всего в строке 14 знаков, из них 5 букв "о" 
```

## Индекс и значение элемента из списка. Функция `enumerate()`
Функция помогает извлечь из списка и индекс элемента, и его значение. Результат работы  — объект-перечисление `enumerate object`, содержащий множество кортежей из двух элементов — индекса элемента исходного списка и значения этого элемента.
`enumerate object` — это объект-генератор. В Python так называется специальный тип объекта, который можно итерировать (перебирать, запрашивая у него следующие значения). При этом на каждой итерации перебираемые значения генерируются из списка, на основе которого был создан объект. Это позволяет сохранять вычислительные ресурсы при работе с большими объёмами данных. При этом в памяти хранится только последнее значение.

Разберём особенности работы объекта-генератора на жизненных примерах. Представьте, вы собрались на обед в классное кафе. Сегодня в меню вкусный витаминный салат, первое блюдо, второе блюдо, десерт и яблочный компот. Все блюда вместе можно представить как набор элементов. Теперь у нас есть два сценария:
- В первом случае все блюда приносят сразу вместе, и пустые тарелки не убирают. В таком случае стол должен быть большой, и при этом пустые тарелки остаются, а второе блюдо и вовсе может остыть.
- Во втором случае каждое блюдо приносят по мере употребления предыдущего. Это экономит место на обеденном столе, процесс обеда оптимизируется — пустые тарелки убираются со стола, снижается одновременно задействованное количество тарелок. С таким подходом проще придерживаться определённой заранее последовательности событий. При этом, если кто-то не захочет яблочный компот, то его не нужно будет специально подавать к столу.

Первый случай больше соответствует простому перебору значений из определённого множества, когда все элементы из этого множества зафиксированы и хранятся в памяти, даже если не используются. А второй пример больше характеризует работу объекта генератора в цикле: каждый новый элемент множества извлекается по мере необходимости его использования.

Мы уже сталкивались с такими объектами ранее — функция `range` генерирует последовательность чисел, которая воспроизводится по мере обращения.

```python
trees = ['Дуб','Берёза','Сосна','Ель']
enumerate_result = enumerate(trees) # сохраним результат работы функции enumerate()

print(enumerate_result) 

<enumerate object at 0x00000177BB5CE110> # если обратиться к этому объекту напрямую, то мы получим только общую информацию о нём
```

Как выглядит содержимое объекта `enumerate`, перебрав его значения в цикле:
```python
trees = ['Дуб','Береза','Сосна','Ель']

for i in enumerate(trees):
      print(i) 

(0, 'Дуб')
(1, 'Берёза')
(2, 'Сосна')
(3, 'Ель') 
```

Каждый элемент исходного списка представлен кортежем с индексом этого элемента и его значением.

Поскольку в каждом кортеже два значения, то для их использования в цикле следует вместо одной переменной-итератора задать две:
- первая будет отображать индекс элемента (`i`);
- а вторая — его значение (`tree`).

Например:
```python
trees = ['Яблоня','Ясень','Черёмуха','Ирга']

for i, tree in enumerate(trees): # зададим две переменные-итератора
    print(f"Дерево с индексом {i} в списке - {tree}") 

Дерево с индексом 0 в списке - Яблоня
Дерево с индексом 1 в списке - Ясень
Дерево с индексом 2 в списке - Черёмуха
Дерево с индексом 3 в списке - Ирга 
```

Можно изменять значения исходного списка по индексу
```python
prices = [1500, 3100, 7500, 4200]
print('Значения списка prices до изменения:', prices)

# уменьшим каждое значение списка prices на 100 единиц:
for i, price in enumerate(prices):
     prices[i] = price - 100
print('Результат изменения значений списка prices:', prices) 

Значения списка prices до изменения: [1500, 3100, 7500, 4200]
Результат изменения значений списка prices: [1400, 3000, 7400, 4100] 
```

Функция `enumerate()` возвращает индекс и значение всех элементов списка или другого итерируемого объекта. Это позволяет уже в самом начале цикла, в его заголовке определить как индекс элемента, так и его значение. Поэтому `enumerate()` предпочтительнее использовать при переборе и изменении элементов по их индексу вместо конструкции `range(len(some_list))`
```python
prices_a = [1500, 3100, 7500, 4200]
prices_b = [1500, 3100, 7500, 4200]

# уменьшим каждое значение списка prices на 100 единиц, используя enumerate:
for i, price in enumerate(prices_a):
     prices_a[i] = price - 100

# уменьшим каждое значение списка prices на 100 единиц, используя range(len(prices)):
for i in range(len(prices_b)):
     prices_b[i] = prices_b[i] - 100
        
print(prices_a)
print(prices_b) 

[1400, 3000, 7400, 4100]
[1400, 3000, 7400, 4100] 
```

Функция `enumerate()`имеет дополнительный параметр `start`, который позволяет задать начало нумерации значений. По умолчанию этот параметр равен `0`.
```python
funnel = ['registration','login','main_page','product_page','purchase_page']

for i, stage in enumerate(funnel, start=1):
    print(f'Этап воронки {i}: {stage}')

Этап воронки 1: registration
Этап воронки 2: login
Этап воронки 3: main_page
Этап воронки 4: product_page
Этап воронки 5: purchase_page
```
> [!important]
> Однако в этом случае мы не сможем использовать переменную итератор `i` для перебора элементов списка по их индексу, так как индексы, возвращаемые `enumerate()` с `start=1`, не будут соответствовать индексам списка, и они не начнутся с `0`.

## Использование в цикле нескольких списков. Функция `zip()`
В цикле можно использовать несколько списков, если их предварительно объединить.
Функция `zip()` принимает в качестве аргумента список итерируемых объектов, элементы которых с одинаковым индексом будут объединены в кортеж. В результате будет создано множество кортежей, объединённых в `zip`-объект
```python
trees = ['Яблоня','Ясень','Черёмуха','Ирга']
prices = [10000, 15000, 13000, 14000]

# выведем значения zip-объекта, объединяющие два списка:
for i in zip(trees, prices):
     print(i)

('Яблоня', 10000)
('Ясень', 15000)
('Черёмуха', 13000)
('Ирга', 14000)
```
Как и в случае с функцией `enumerate()`, возвращаемый zip-объект будет объектом- генератором.

Для перебора кортежей, сформированных функцией `zip()` при объединении нескольких списков, можно использовать цикл `for` с несколькими переменными итераторами в заголовке:

```python
trees = ['Яблоня','Ясень','Черёмуха','Ирга']
prices = [10000, 15000, 13000, 14000]

for tree, price in zip(trees, prices):
     print(f'{tree}, стоимость {price} руб.') 

Яблоня, стоимость 10000 руб.
Ясень, стоимость 15000 руб.
Черёмуха, стоимость 13000 руб.
Ирга, стоимость 14000 руб. 
```

С помощью функции `zip()` можно объединять более двух списков. У функции `zip()` есть важная особенность: если количество элементов в циклах будет разным, то итоговое количество кортежей будет соответствовать количеству элементов в наикратчайшем из списков.
```python
trees = ['Яблоня','Ясень','Черёмуха','Ирга']
prices = [10000, 15000, 13000, 14000]
markets = ['Алексеевка','Заречное']

# объединим списки:
for tree, price, market in zip(trees, prices, markets):
    print(f'{tree}, стоимость {price} руб., магазин: {market}')

Яблоня, стоимость 10000 руб., магазин: Алексеевка
Ясень, стоимость 15000 руб., магазин: Заречное
```

Если необходимо сложить все списки, когда они не равны по количеству элементов, можно использовать функцию `zip_longest()` из модуля `itertools`. Действие этой функции аналогично функции `zip()`, только в случае недостающих значений будет использовано значение `None`.
```python
import itertools # импортируем модуль itertools 

trees = ['Яблоня','Ясень','Черёмуха','Ирга']
prices = [10000, 15000, 13000, 14000]
markets = ['Алексеевка','Заречное']

for tree, price, market in itertools.zip_longest(trees, prices, markets):
    print(f'{tree}, стоимость {price} руб., магазин: {market}')

Яблоня, стоимость 10000 руб., магазин: Алексеевка
Ясень, стоимость 15000 руб., магазин: Заречное
Черёмуха, стоимость 13000 руб., магазин: None
Ирга, стоимость 14000 руб., магазин: None
```

В функцию `zip_longest()` можно передать аргумент, который будет заполнять пропущенные значения: `fillvalue`:
```python
import itertools

trees = ['Яблоня','Ясень','Черёмуха','Ирга']
prices = [10000, 15000, 13000, 14000]
markets = ['Алексеевка','Заречное']

# Выведем совместно значения списков:
for tree, price, market in itertools.zip_longest(trees, prices, markets, fillvalue='Нет данных'):
    print(f'{tree}, стоимость {price} руб., магазин: {market}') 

Яблоня, стоимость 10000 руб., магазин: Алексеевка
Ясень, стоимость 15000 руб., магазин: Заречное
Черёмуха, стоимость 13000 руб., магазин: Нет данных
Ирга, стоимость 14000 руб., магазин: Нет данных 
```

Рассмотрим работу цикла на примере с новогодними подарками
```python
length = 100 # переменная, которая задаёт общую длину бумаги

# объявим цикл while
while length > 32: # условие, которое проверяет хватит ли бумаги на подарок
    length -= 32 # изменение переменной на каждой итерации
    print(f'Осталось {length} см упаковочной бумаги') 

Осталось 68 см упаковочной бумаги
Осталось 36 см упаковочной бумаги
Осталось 4 см упаковочной бумаги 
```

```python
length = 0 # зададим переменную, которая будет учитывать затраченную длину бумаги
n = 1 # зададим начальное количество итераций - количество подарков

# объявим цикл
while n <= 5: # условие, которое проверяет значение перменной 'n'
    length += 32 # на каждой итерации к значению length будет добавлено 32
    print(f'Подарок {n}, израсходовано {length} см упаковочной бумаги')
    n += 1 # увеличиваем счётчик итераций на 1

print(f'Результат: для {n-1} подарков нужно {length} см упаковочной бумаги')
```

## Использование цикла `for` или `while`
С помощью цикла `while` можно рассчитать проценты по вкладу с капитализацией (начислением процентов на проценты) за 6 месяцев при годовой ставке 6.8%
```python
# зададим исходные значения вклада:
deposit = 100000 # сумма вклада
annual_rate = 0.068 # годовая процентная ставка в долях 
time = 6 # срок вклада в месяцах

month = 1 # установим счётчик месяцев

while month <= time: # объявим цикл
    dep_interest = deposit * annual_rate / 12 # рассчитаем начисленный процент по вкладу
    deposit += dep_interest # увеличим размер вклада на значение процента по вкладу
```

Можно также использовать цикл `for`:

```python
# зададим исходные значения вклада:
deposit = 100000 # сумма вклада
annual_rate = 0.068 # годовая процентная ставка в долях 
time = 6 # срок вклада в месяцах

for month in range(6): # объявим цикл
    dep_interest = deposit * annual_rate / 12 # рассчитаем начисленный процент по вкладу
    deposit += dep_interest # увеличим размер вклада на значение процента по вкладу 
    print(f'Месяц {month+1}, процент по вкладу {round(dep_interest, 2)}, сумма вклада с капитализацией {round(deposit, 2)}') 
```

> [!tip]
> Обратите внимание, что при использовании цикла `for` нам не пришлось задавать переменную счётчик `month`. Это значение мы инициировали в качестве переменной-итератора в заголовке цикла.

Так же и при решении различных аналитических задач количество итераций, как правило, заранее известно и в этом случае предпочтительнее использовать цикл `for`.

Циклы `while` чаще используются, когда необходимо ожидать наступления определённого события.

```python
month = 1 # установим счётчик месяцев

    dep_interest = deposit * annual_rate / 12 # рассчитаем начисленный процент по вкладу
    deposit += dep_interest # увеличим размер вклада на значение процента по вкладу

    month += 1
    
print(f'Количество месяцев, через которое сумма начисленных процентов по вкладу превысить 10000 = {month-1}')
```
