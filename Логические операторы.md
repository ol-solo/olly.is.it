## Логические операторы сравнения
|**Логический оператор**|**Значение**|**Действие логического оператора**|**Когда возвращает `True`**|**Пример выражения с результатом `True`**|
|---|---|---|---|---|
| $==$ |равно|сравнивает два значения между собой|когда значения равны|$5 == 5$|
| $!=$ |не равно|сравнивает два значения между собой|когда значения не равны|$4 != 5$|
| $>$ |больше|сравнивает два значения между собой|если первое больше второго|$5 > 1$|
| $<$ |меньше|сравнивает два значения между собой|если первое меньше второго|$1 < 5$|
| $>=$ |больше или равно|сравнивает два значения между собой|если первое больше второго или значения равны|$5 >= 1$ ; $5 >= 5$|
| $<=$ |меньше или равно|сравнивает два значения между собой|если первое меньше второго или значения равны|$1 <= 1$  $5 <= 5$|

Значения, сравнивающиеся в логическом выражении, называются **операнд**. Например, в выражении `1 < 5` операнды — это 1 и 5.

## Простое логическое выражение
Логические выражения, в котором используется только два операнда и один логический оператор сравнения, называют **простыми логическими выражениями**.

Можно сравнивать не только числа, но и строки, списки и другие типы данных. 
```python
(16*5) > 99 # False
20 <= 20 # True
['A','B','C'] == ['A','B','C'] # True
'одно слово' != 'второе слово' # True 
```

## Сравнение данных разных типов
Данные разных типов можно сравнивать с помощью операторов равенства (`==`) и неравенства (`!=`). 
```python
'20' == 20 

False 
```

Результат `False`, поскольку строка `'20'` не равна числу 20.
Если попытаться сравнить данные разных типов с помощью операторов `>`, `<`, `>=` или `<=`, то получим ошибку.
Исключение — сравнение `int` и `float`. Это разные типы данных, но это числа, так что их можно сравнивать с помощью любых операторов сравнения: `==`, `!=`, `>`, `<`, `>=` и `<=`.

## Сложные логические выражения
Сложное логическое выражение состоит из нескольких простых логических выражений, сочетание которых определяет результат всего выражения.

|**Выражение А**|**Выражение В**|**A and B**|**A or B**|**not A**|**not B**|
|---|---|---|---|---|---|
|True|True|True|True|False|False|
|True|False|False|True|False|True|
|False|True|False|True|True|False|
|False|False|False|False|True|True|

## Операторы `and` и `or`
Оператор `and` называют ещё «логическое И» или «логическое умножение». Он возвращает `True`, только если **оба** логических значения — `True`. В остальных случаях оператор `and` вернёт `False`.

Оператор `or` называют также «логическое ИЛИ», «логическое сложение». Он возвращает `True`, если **хотя бы** одно логическое значение тоже `True`. Оператор `or` вернёт `False` только если оба логические значения будут `False`.

Логические операторы `and` и `or` являются **бинарными** логическими операторами, поскольку применяются к двум операндам, стоящим слева и справа.

## Оператор `not`
Логический оператор `not` меняет одно значение на противоположное:

- `True` на `False`,
- `False` на `True`.

Оператор `not` является **унарным**, поскольку применяется к одному операнду, стоящему после него. Этим он отличается от бинарных `and` и `or`.

## Битовые операторы `&` и `|`
Помимо стандартных числовых операций, есть операторы для побитовых операций над целыми числами в их двоичном представлении.

Битовые операторы соответствуют логическим операторам:

|битовый оператор|значение|
|---|---|
|`&`|и (`and`)|
| `\|` |или (`or`)|
|`~`|не (`not`)|

Логические значения `True` и `False` можно рассматривать как 1 и 0. Битовый оператор берёт последовательно бит из одного числа и соответствующий бит из другого и получает снова 1 или 0. В результате получается последовательность единичек и нулей, которую можно обратно перевести в десятичную систему.

#### Пример работы `&`
Оператор `&` выполняет логическое «и» для каждой пары битов:
```python
a = 5 # в двоичной системе - это 101
b = 3 # в двоичной системе - это 011

result = a & b
print(result) # Вывод: 1 (в двоичной системе - это 001) 
```
![[Pasted image 20260302182321.png]]
Результат содержит только тот бит, который присутствует в a и в b. В двоичной системе получается `001`, в десятичной это 1.

#### Пример работы `|`
Оператор `|` работает как побитовое логическое «или»:
```python
a = 5 # 101
b = 3 # 011

result = a | b
print(result) # Вывод: 7 (111) 
```
![[Pasted image 20260302182424.png]]
Здесь результат содержит те биты, которые были установлены в 1 хотя бы в одном из чисел a или b, это `111`, что в десятичной системе соответствует числу 7.

## Последовательность выполнения условий
Стандартный порядок в Python такой:
1. операции в скобках;
2. возведение в степень `**`;
3. битовый оператор `~`;
4. математические операторы — умножение `*` и деление `/`, затем сложение `+` и вычитание `-`;
5. битовые операторы `&` и `|`;
6. операторы сравнения — `<`, `<=`, `>`, `>=`, `!=`, `==`;
7. логические операторы сравнения `not`, `and` и `or`.

Как обрабатываются сложные условия.
```python
x = 2
y = 15
z = 15

print((x > 5) and (y < 10) or (z == y)) 

True 
```
