# Данные, но ещё не совсем. Словари

Структуры данных позволяют хранить данные так, чтобы человеку было удобно с ними работать. С одной такой структурой — списками — вы уже познакомились. Списки помогают собрать несколько объектов вместе.

Теперь вы готовы познакомиться с ещё одной, не менее полезной структурой данных в Python — словарями.

#### **В этом уроке вы узнаете:**

- Что такое словари в Python и зачем они нужны;
- Какие данные можно хранить в словаре и как с ними работать;
- Когда может пригодиться список со словарями.

## Что такое словарь?

Словари в Python очень похожи на словари и энциклопедии в обычной жизни.

- В энциклопедии, обращаясь к термину, мы получаем его объяснение.
- В англо-русском словаре, обращаясь к слову на английском языке, мы получаем его перевод на русский.

Так выглядит любая словарная статья: это пара значений, где первое помогает найти второе. А сам словарь — это просто набор таких пар: слово — перевод.

Словари в Python устроены так же. Каждый элемент словаря — пара из ключа и значения:

- **ключ** — то, что мы ищем в словаре. Ключом может быть строка или число;
- **значение** — данные, к которым ведёт этот ключ. В отличие от ключа, значением может быть объект любого типа: не только строка или число, но и, например, список.

Для словарей в Python есть свой тип объекта — `dict`.

## Зачем нужен словарь, когда есть списки?

Часто удобнее искать значения не по его индексу, а по ключу, который имеет собственный смысл.
## Создание словаря

Есть несколько правил синтаксиса для создания словаря:

- словари записываются в фигурных скобках `{ }`;
- элементы словаря перечисляются через запятую;
- ключ и значение разделяют двоеточием `:`.

Создадим словарь с авиарейсами. Ключом станет номер рейса, а значением — дата и время вылета:

```
schedule = {                    # элементы словаря заключают в фигурные скобки
    'SU2222': '12.06.23 12:30', # слева от двоеточия — ключ, справа — значение
    'SU1111': '12.06.23 14:05', # каждый элемент отделяется запятой
    'SU0777': '12.06.23 17:00',
    'SU5555': '15.06.23 11:25'
} 
```

Каждую пару ключ-значение мы записали в новой строке. В правилах синтаксиса мы ничего не сказали о переносах. Если использовать переносы, код становится проще читать. Но ничего не мешает записать словарь и в плоском виде.

```
schedule = {'SU2222': '12.06.23 12:30', 'SU1111': '12.06.23 14:05', 'SU0777': '12.06.23 17:00', 'SU5555': '15.06.23 11:25'} 
```

Оба варианта верны, но первый вариант выглядит удобнее.

Выведем содержимое словаря с помощью `print()`:

```
print(schedule) 
```

```
{'SU2222': '12.06.23 12:30', 'SU1111': '12.06.23 14:05', 'SU0777': '12.06.23 17:00', 'SU5555': '15.06.23 11:25'} 
```

Выведем тип полученного объекта:

```
print(type(schedule)) 
```

```
<class 'dict'> 
```

Мы разобрались, как создавать словари. Теперь давайте посмотрим, как с ними работать.

## Находим значение в словаре по ключу

Обратиться к значению в словаре можно при помощи квадратных скобок. Как со списками — только указываем не индекс, а ключ.

```
schedule = {                    
    'SU2222': '12.06.23 12:30', 
    'SU1111': '12.06.23 14:05', 
    'SU0777': '12.06.23 17:00',
    'SU5555': '15.06.23 11:25'
}

print(schedule['SU2222'])
print(schedule['SU1111']) 
```

```
12.06.23 12:30
12.06.23 14:05 
```

Если обратиться к словарю по ключу, которого нет, то вернётся ошибка, и программа остановится. Нужно быть уверенным в том, что ключ есть в словаре.

```
schedule = {                    
    'SU2222': '12.06.23 12:30', 
    'SU1111': '12.06.23 14:05', 
    'SU0777': '12.06.23 17:00',
    'SU5555': '15.06.23 11:25'
}

print(schedule['DP6831']) 
```

```
KeyError: 'DP6831' 
```

Если мы заранее не знаем, есть ли такой ключ, то нужно использовать метод `get()`:

```
schedule = {                    
    'SU2222': '12.06.23 12:30', 
    'SU1111': '12.06.23 14:05', 
    'SU0777': '12.06.23 17:00',
    'SU5555': '15.06.23 11:25'
}

print(schedule.get('DP6831')) 
```

```
None 
```

Этот способ вернёт значение `None`, если такого ключа не нашлось. При этом ошибки не будет, и программа не остановится.

Что будет, если в словаре для одного слова будет два разных определения, то есть ключ будет встречаться несколько раз?

```
print({
    0: 'ноль',
    1: 'один',
    2: 'два',
    1: 'единица'
}) 
```

```
{0: 'ноль', 1: 'единица', 2: 'два'} 
```

Одно значение пропало! Это потому, что ключи словаря должны быть уникальными.


## Какие данные может хранить словарь?

Значениями словаря могут быть числа и строки, но не только. Списки тоже можно хранить. Создадим ещё один словарь с рейсами, но для каждого сохраним расписание сразу на несколько лет вперёд. Для этого используем списки.

```
schedule = {                   
    'SU2222': ['12.06.23 12:30', '12.06.24 18:30', '12.06.25 22:30'], 
    'SU1111': ['12.06.23 14:05', '12.06.24 20:05', '12.06.25 23:05'], 
    'SU0777': ['12.06.23 17:00', '12.06.24 21:00', '12.06.25 21:00'],
    'SU5555': ['15.06.23 11:25', '12.06.24 17:25', '12.06.25 22:25'], 
}

print(schedule['SU2222']) 
```

```
['12.06.23 12:30', '12.06.24 18:30', '12.06.25 22:30'] 
```

А ещё значениями словаря могут быть другие словари. Их называют вложенными. Создадим словарь с рейсами, но ключом станет не название рейса, а название компании-перевозчика. Значением будет словарь со всеми рейсами и их расписанием.

```
schedule = {
    'Аэрофлот': {
        'SU2222': '12.06.23 12:30',
        'SU1111': '12.06.23 14:05',
        'SU0777': '12.06.23 17:00',
        'SU5555': '15.06.23 11:25'},
    'Победа': {
        'DP6831': '12.06.23 09:30', 
        'DP6800': '12.06.23 11:05'},
    'Nordwind Airlines': {
        'N40001': '12.06.23 08:15', 
        'N48321': '12.06.23 19:15'}
    } 
```

Получим все рейсы компании `Аэрофлот`:

```
aeroflot_flights = schedule['Аэрофлот']

print(aeroflot_flights) 
```

```
{'SU2222': '12.06.23 12:30',
 'SU1111': '12.06.23 14:05',
 'SU0777': '12.06.23 17:00',
 'SU5555': '15.06.23 11:25'} 
```

А теперь получим дату вылета рейса `SU2222`:

```
current_flight = aeroflot_flights['SU2222']

print(current_flight) 
```

```
12.06.23 12:30 
```

То же самое мы могли бы сделать и одной строчкой кода, последовательно обращаясь сначала к верхнему словарю, а затем ко вложенному, используя несколько раз квадратные скобки:

```
current_flight = schedule['Аэрофлот']['SU2222'] 
```

Ключами словаря не могут стать изменяемые объекты, например списки или словари. Обычно ключами в словаре являются строки или числа, будем придерживаться этого.

## Поиск в словаре

Создадим словарь со странами и их столицами.

```
country_capitals = {
    'США': 'Вашингтон',
    'Италия': 'Рим',
    'Англия': 'Лондон'
} 
```

Как вы помните, словарь — это структура, которая состоит из ключей и значений. Ключи и значения связаны, но с ними можно работать по отдельности.

Иногда нам может потребоваться получить набор из всех ключей, которые есть в словаре. Например, найдём и выведем все страны (страна — это ключ), название которых состоит из трёх символов. Для этого воспользуемся методом `keys()`.

```
all_countries = country_capitals.keys()

for country in all_countries:
    if len(country) == 3:
        print(country) 
```

```
США 
```

Точно так же мы можем сразу найти и все значения, которые есть в словаре. Найдём все столицы (столица — это значение), название которых состоит из трёх символов. Для этого воспользуемся методом `values()`.

```
all_capitals = country_capitals.values()

for capital in all_capitals:
    if len(capital) == 3:
        print(capital) 
```

```
Рим 
```

Словарь отлично подходит для поиска **значения** по его **ключу**. Но что, если, наоборот, необходимо найти ключ по его значению? Для решения такой задачи придётся перебрать в цикле каждую пару ключ-значение, которые содержатся в словаре, пока не найдётся пара с нужным значением.

Решим похожую задачку на примере данных о городах и их столицах. Найдём страну, в которой столица — Рим.

```
country_capitals = {
    'США': 'Вашингтон',
    'Италия': 'Рим',
    'Англия': 'Лондон'
}
rome_country = ''

for country in country_capitals.keys(): # keys() вернёт список всех стран
    capital = country_capitals[country] # сохраним столицу текущей страны
    if capital == 'Рим':       # если столица Рим
        rome_country = country # то сохраним страну в переменной rome_country
        break                  # и завершим цикл
                               # если столица не Рим, то продолжаем цикл

print(rome_country) 
```

```
Италия 
```

Сейчас разберёмся, что тут происходит:

1. Первое, что мы сделаем, — создадим цикл, в котором будем перебирать все страны в словаре. Метод `keys()` возвращает набор всех ключей, а ключ — это и есть страна.

```
for country in country_capitals.keys():
    pass 
```

2. Найдём столицу для текущей страны, для этого используем наш словарь `country_capitals`. Теперь в переменной `capital` хранится столица, а в переменной `country` — страна.

```
capital = country_capitals[country] 
```

3. Проверим, является ли текущая столица Римом. Если является, то мы нашли нужную страну. Сохраним результат в переменной `rome_country` и завершим выполнение цикла командой `break`.

```
if capital == 'Рим':
    rome_country = country
    break  
```

4. Если столица не Рим, то продолжаем цикл и повторяем то же самое на следующей итерации.

Наше решение строилось на том, чтобы в цикле перебрать все пары ключ-значение. Для этого при создании цикла мы получили набор ключей методом `keys()`, а уже внутри цикла для каждого нового ключа находили его значения.

А что, если сразу получать эту пару — ключ-значение? Да, так тоже можно! Для этого воспользуемся методом `items()`:

```
country_capitals = {
    'США': 'Вашингтон',
    'Италия': 'Рим',
    'Англия': 'Лондон'
}

for country, capital in country_capitals.items():
    print(country, capital) 
```

```
США Вашингтон
Италия Рим
Англия Лондон 
```

Решим ту же самую задачу — найдём страну, столица которой — Рим. Но теперь воспользуемся методом `items()`:

```
country_capitals = {
    'США': 'Вашингтон',
    'Италия': 'Рим',
    'Англия': 'Лондон'
}
rome_country = ''

# запустим цикл: items() вернёт пары — страна и столица
for country, capital in country_capitals.items(): 
    if capital == 'Рим':       # если столица Рим
        # то сохраним страну в переменной rome_country
        rome_country = country 
        # и прервём цикл
        break                  
    # если столица не Рим, то продолжаем цикл

print(rome_country) 
```

```
Италия 
```

Наш код стал на строчку короче, а решает задачу ничуть не хуже!

## Действия со словарями

Работа со словарями очень похожа на работу со списками. В словарь можно добавлять новые элементы, изменять их и удалять.

Для того чтобы добавить в словарь новый ключ, просто присвоим ему новое значение:

```
country_capitals = {
    'США': 'Вашингтон',
    'Италия': 'Рим',
    'Англия': 'Лондон'
}
country_capitals['Россия'] = 'Москва'

print(country_capitals) 
```

```
{'США': 'Вашингтон', 'Италия': 'Рим', 'Англия': 'Лондон', 'Россия': 'Москва'} 
```

Если такой ключ уже есть в словаре, то его значение обновится.

```
country_capitals = {
    'США': 'Вашингтон',
    'Италия': 'Рим',
    'Англия': 'Лондон'
}

country_capitals['США'] = 'Washington'
country_capitals['Италия'] = 'Roma'
country_capitals['Англия'] = 'London'

print(country_capitals) 
```

```
{'США': 'Washington', 'Италия': 'Roma', 'Англия': 'London'} 
```

Таким способом, мы можем изменить значение в словаре по его ключу. Однако мы не можем изменить сам ключ. Если по какой-то причине нам нужно изменить его, то необходимо удалить элемент с этим ключом и создать новый с правильным ключом.

Для удаления элемента из словаря можно использовать метод `pop()`. Он работает так же, как и со списками — удаляет элемент и возвращает значение, которое было удалено.

```
country_capitals = {
    'США': 'Вашингтон',
    'Италия': 'Рим',
    'Англия': 'Лондон'
}

deleted_capital = country_capitals.pop('Англия')

print(country_capitals)
print(deleted_capital) 
```

```
{'США': 'Вашингтон', 'Италия': 'Рим'}
Лондон 
```

## Список со словарями

Представим, что у нас есть информация о пользователе: ФИО, дата регистрации, дата последнего заказа и количество заказов. Для хранения такой информации хорошо подходит словарь.

```
user_info = {
    'ФИО': 'Иван Иваныч',
    'Дата регистрации': '01.01.2021',
    'Дата последней покупки': '05.07.2023',
    'Количество заказов': 123
} 
```

А что делать, если таких покупателей будет несколько?

```
user_info_1 = {
    'ФИО': 'Иван Иваныч',
    'Дата регистрации': '01.01.2021',
    'Дата последней покупки': '05.07.2023',
    'Количество заказов': 123
}

user_info_2 = {
    'ФИО': 'Пётр Васильич',
    'Дата регистрации': '03.06.2020',
    'Дата последней покупки': '20.07.2023',
    'Количество заказов': 20
}

user_info_3 = {
    'ФИО': 'Марья Петровна',
    'Дата регистрации': '05.02.2018',
    'Дата последней покупки': '20.05.2021',
    'Количество заказов': 5
} 
```

Нам поможет список! Вы же помните, что любой тип объекта может стать элементом списка? Словарь не исключение.

```
users = [user_info_1, user_info_2, user_info_3] #список словарей 
```

Такой способ хранения данных напоминает привычные таблицы. Попробуем найти общее количество заказов от всех пользователей.

```
orders = 0
for user in users:
        orders += user['Количество заказов']

print(orders) 
```

```
148 
```

- Словарь состоит из ключа и значения. Значением может быть объект любого типа, а ключом, как правило, строка или число.
- Обращение к элементу словаря происходит по его ключу.
- В словарь можно добавлять новые элементы и удалять старые.
- Для того чтобы перебрать все элементы словаря в цикле, можно использовать метод `items()`.

## Шпаргалка

Словари — это структуры данных, состоящие из пар ключ-значение. Значениями могут быть числа, строки, списки и словари, а ключами, как правило, строки или числа.

Чтобы создать словарь:

- запишите элементы в фигурных скобках `{ }` через запятую;
- разделяйте ключ и значение двоеточием `:`.

Чтобы обратиться к значению в словаре, укажите ключ в квадратных скобках.

Проверить, есть ли ключ в словаре, можно с помощью метода `get()`. Найти пару ключ-значение поможет метод `items()`.

Чтобы добавить новый ключ в словарь, нужно присвоить ему значение. Например:

```
country_capitals = {
    'США': 'Вашингтон',
    'Италия': 'Рим',
    'Англия': 'Лондон'
    }
country_capitals['Россия'] = 'Москва'

print(country_capitals) 
```

Значения менять можно, ключи — нет. Чтобы удалить элемент словаря, используйте метод `pop()`.
