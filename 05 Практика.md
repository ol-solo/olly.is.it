# «Ящик с усами» и гистограмма. Практика

### Шпаргалка
Используйте `plt.hist()` для создания гистограммы. Доступные параметры:
- `data` — список значений для гистограммы;
- `bins` — количество интервалов;
- `color` — цвет корзин гистограммы;
- `edgecolor` — цвет края корзин;
- `alpha` — прозрачность.

Используйте `plt.boxplot()` для построения диаграммы размаха.
- `notch=True` — создаёт выемку на ящике диаграммы вокруг медианы, что указывает на доверительный интервал медианы.
- `vert=True` — указывает на вертикальную ориентацию диаграммы размаха. Если установить `False`, диаграмма будет горизонтальной.

Для оформления графиков укажите:
- `plt.title('Заголовок')`
- `plt.xlabel('Подпись по горизонтальной оси')`
- `plt.ylabel('Подпись по вертикальной оси')`

### Задание 1
Постройте гистограмму средних дневных температур за месяц, чтобы понять какие температурные диапазоны встречаются чаще всего. Используйте синий цвет для корзин гистограммы и чёрный для границ корзин.

```python
import matplotlib.pyplot as plt

# Данные о температурах
temperatures = [22, 25, 19, 21, 23, 27, 28, 22, 20, 18,
                24, 26, 22, 23, 24, 25, 26, 27, 28, 29,
                30, 19, 20, 21, 22, 23, 24, 25, 26, 27]

plt.hist(temperatures, bins=12, color='blue', edgecolor='black')
plt.title('Распределение температур за месяц')
plt.xlabel('Температура')
plt.ylabel('Количество дней')
plt.show()
```

# Линейный график и столбчатая диаграмма. Практика

### Шпаргалка
Для решения задач вам понадобятся следующие параметры:
- `plt.plot()` строит линейный график;
- `marker=` определяет вид маркеров на линейном графике (например, `marker='o'` для маркера в виде точки);
- `plt.bar()` создаёт вертикальные столбцы на диаграмме;
- `plt.barh()` создаёт горизонтальные столбцы на диаграмме;
- `plt.title()` добавляет заголовок к графику;
- `plt.xlabel()`, `plt.ylabel()` подписывает оси графика;
- `color=` позволяет изменить цвет заливки;
- `plt.grid()` добавляет сетку к графику для лучшей читаемости.

### Задание 1
В вашем распоряжении данные о средней месячной температуре в городе за год. Постройте линейный график, отражающий изменение температуры по месяцам.

```python
import matplotlib.pyplot as plt
  
months = ["Январь", "Февраль", "Март", "Апрель", "Май", "Июнь",
          "Июль", "Август", "Сентябрь", "Октябрь", "Ноябрь", "Декабрь"]
temperatures = [0, -2, 5, 10, 15, 20, 25, 24, 20, 14, 5, 0]
  
plt.plot(months, temperatures)
plt.title("Средняя месячная температура в городе")
plt.xlabel("Месяц")
plt.ylabel("Температура, °C")
plt.grid(True)
plt.xticks(rotation=45)
plt.show()
```

### Задание 2
У вас есть данные об изменении высоты растения за каждую неделю в течение 10 недель. Визуализируйте рост растения, построив линейный график с маркерами.

```python
import matplotlib.pyplot as plt
  
weeks = list(range(1, 11))
heights = [10, 12, 15, 17, 21, 24, 28, 31, 35, 40]
  
plt.plot(weeks, heights, marker='o')
plt.title("Рост растения за 10 недель")
plt.xlabel("Неделя")
plt.ylabel("Высота, см")
plt.grid(True)
plt.show()
```

### Задание 3.

У вас есть данные о продуктивности двух команд за четыре квартала.

Команда A должна быть представлена линией синего цвета, а команда B — зелёного. Постройте линейные графики для каждой команды на одном графике, используя для обозначения линий метки `'Команда A'` и `'Команда B'`.

```python
import matplotlib.pyplot as plt
  
quarters = ["Q1", "Q2", "Q3", "Q4"]
team_a = [5, 10, 15, 20]
team_b = [3, 7, 8, 18]

plt.xlabel("Квартал")
plt.ylabel("Продуктивность")
plt.legend()
plt.grid(True)
plt.show()
```
