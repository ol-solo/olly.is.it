# «Ящик с усами» и гистограмма. Практика

### Шпаргалка
Используйте `plt.hist()` для создания гистограммы. Доступные параметры:
- `data` — список значений для гистограммы;
- `bins` — количество интервалов;
- `color` — цвет корзин гистограммы;
- `edgecolor` — цвет края корзин;
- `alpha` — прозрачность.

Используйте `plt.boxplot()` для построения диаграммы размаха.
- `notch=True` — создаёт выемку на ящике диаграммы вокруг медианы, что указывает на доверительный интервал медианы.
- `vert=True` — указывает на вертикальную ориентацию диаграммы размаха. Если установить `False`, диаграмма будет горизонтальной.

Для оформления графиков укажите:
- `plt.title('Заголовок')`
- `plt.xlabel('Подпись по горизонтальной оси')`
- `plt.ylabel('Подпись по вертикальной оси')`

# Линейный график и столбчатая диаграмма. Практика

### Шпаргалка
Для решения задач вам понадобятся следующие параметры:
- `plt.plot()` строит линейный график;
- `marker=` определяет вид маркеров на линейном графике (например, `marker='o'` для маркера в виде точки);
- `plt.bar()` создаёт вертикальные столбцы на диаграмме;
- `plt.barh()` создаёт горизонтальные столбцы на диаграмме;
- `plt.title()` добавляет заголовок к графику;
- `plt.xlabel()`, `plt.ylabel()` подписывает оси графика;
- `color=` позволяет изменить цвет заливки;
- `plt.grid()` добавляет сетку к графику для лучшей читаемости.

### Задание 1
В вашем распоряжении данные о средней месячной температуре в городе за год. Постройте линейный график, отражающий изменение температуры по месяцам.

```python
import matplotlib.pyplot as plt
  
months = ["Январь", "Февраль", "Март", "Апрель", "Май", "Июнь",
          "Июль", "Август", "Сентябрь", "Октябрь", "Ноябрь", "Декабрь"]
temperatures = [0, -2, 5, 10, 15, 20, 25, 24, 20, 14, 5, 0]
  
plt.plot(months, temperatures)
plt.title("Средняя месячная температура в городе")
plt.xlabel("Месяц")
plt.ylabel("Температура, °C")
plt.grid(True)
plt.xticks(rotation=45)
plt.show()
```

### Задание 5
Вы анализируете аудиторию сайта из разных стран с разных устройств. Используйте горизонтальную столбчатую диаграмму, чтобы отобразить количество посетителей из каждой страны в разбивке по устройствам. Посещения с устройств upDroid отобразите цветом `'lightgreen'`, а с устройств yPhone цветом `'lightblue'`.

```python
import matplotlib.pyplot as plt
import pandas as pd
  
df = pd.DataFrame({
        'countries': ["Страна A", "Страна B", "Страна C", "Страна D"] * 2,
        'devices': ['yPhone'] * 4 + ['upDroid'] * 4,
        'visitors': [800, 650, 400, 300, 700, 600, 500, 400]})
  
# Группируем по странам и устройствам
grouped = df.pivot_table(index='countries', columns='devices', values='visitors', aggfunc='sum')
  
# Строим график
grouped.plot(kind='barh', stacked=False, color=['lightgreen', 'lightblue'])
  
plt.title("Посещаемость сайта из разных стран")
plt.xlabel("Количество посетителей")
plt.ylabel("Страны")
plt.legend(title='Устройства', loc='center left', bbox_to_anchor=(1, 0.5))
plt.show()
```

# Пироги и другие графики. Практика
## Шпаргалка
В этом уроке вам пригодятся следующие функции:
- `plt.pie()` создает круговую диаграмму;
- `plt.scatter()` создает диаграмму рассеивания;
- `plt.subplot()` создает сабплоты;
- `plt.plot()` создает линейный график;
- `plt.xlabel()` и `plt.ylabel()` добавляют подписи к осям;
- `plt.title()` добавляет заголовки к графику.

### Задание 1
В университете прошел опрос о том, какие специальности выбирают студенты. Постройте круговую диаграмму, чтобы визуализировать распределение студентов по специальностям. Подпишите в формате `'%1.1f%%'` сектора диаграммы, подчеркните сектор информационных технологий, выдвинув его. Укажите название графика `Распределение студентов по специальностям`.

```python
import matplotlib.pyplot as plt
  
labels = ['Информационные технологии', 'Экономика', 'Медицина', 'Инженерия']
sizes = [210, 130, 245, 210]
  
plt.pie(sizes, labels=labels, autopct='%1.1f%%', explode=[0.1, 0, 0, 0])
plt.title('Распределение студентов по специальностям')
plt.show()
```

### Задание 3
Проанализируйте изменения уровня загрязнения воздуха и температуры за месяц. Данные о загрязнении и температуре представлены ниже. Создайте сабплот с двумя графиками:
- Первый, сверху, для уровня загрязнения воздуха.

```python
import matplotlib.pyplot as plt

ax[0].plot(days, pollution, marker='o', color='red')
ax[0].set_title('Уровень загрязнения воздуха за месяц')

ax[1].set_xlabel('День')
ax[1].set_ylabel('Температура (°C)')
ax[1].grid(True)
  
plt.show()
```
