# «Ящик с усами» и гистограмма. Практика

### Шпаргалка
Используйте `plt.hist()` для создания гистограммы. Доступные параметры:
- `data` — список значений для гистограммы;
- `bins` — количество интервалов;
- `color` — цвет корзин гистограммы;
- `edgecolor` — цвет края корзин;
- `alpha` — прозрачность.

Используйте `plt.boxplot()` для построения диаграммы размаха.
- `notch=True` — создаёт выемку на ящике диаграммы вокруг медианы, что указывает на доверительный интервал медианы.
- `vert=True` — указывает на вертикальную ориентацию диаграммы размаха. Если установить `False`, диаграмма будет горизонтальной.

Для оформления графиков укажите:
- `plt.title('Заголовок')`
- `plt.xlabel('Подпись по горизонтальной оси')`
- `plt.ylabel('Подпись по вертикальной оси')`

### Задание 1
Постройте гистограмму средних дневных температур за месяц, чтобы понять какие температурные диапазоны встречаются чаще всего. Используйте синий цвет для корзин гистограммы и чёрный для границ корзин.
