# «Ящик с усами» и гистограмма. Практика

### Шпаргалка
Используйте `plt.hist()` для создания гистограммы. Доступные параметры:
- `data` — список значений для гистограммы;
- `bins` — количество интервалов;
- `color` — цвет корзин гистограммы;
- `edgecolor` — цвет края корзин;
- `alpha` — прозрачность.

Используйте `plt.boxplot()` для построения диаграммы размаха.
- `notch=True` — создаёт выемку на ящике диаграммы вокруг медианы, что указывает на доверительный интервал медианы.
- `vert=True` — указывает на вертикальную ориентацию диаграммы размаха. Если установить `False`, диаграмма будет горизонтальной.

Для оформления графиков укажите:
- `plt.title('Заголовок')`
- `plt.xlabel('Подпись по горизонтальной оси')`
- `plt.ylabel('Подпись по вертикальной оси')`

# Линейный график и столбчатая диаграмма. Практика

### Шпаргалка
Для решения задач вам понадобятся следующие параметры:
- `plt.plot()` строит линейный график;
- `marker=` определяет вид маркеров на линейном графике (например, `marker='o'` для маркера в виде точки);
- `plt.bar()` создаёт вертикальные столбцы на диаграмме;
- `plt.barh()` создаёт горизонтальные столбцы на диаграмме;
- `plt.title()` добавляет заголовок к графику;
- `plt.xlabel()`, `plt.ylabel()` подписывает оси графика;
- `color=` позволяет изменить цвет заливки;
- `plt.grid()` добавляет сетку к графику для лучшей читаемости.

# Гистограммы и диаграммы размаха в seaborn. Практика

### Шпаргалка
Загрузить данные из CSV можно, написав `pd.read_csv('путь_к_файлу.csv')`. Наш файл находится по адресу `/datasets/market_file.csv`.

Метод `info()` нужен для получения краткой сводки данных DataFrame, включая названия столбцов, количество записей, типы данных по каждой колонке и количество ненулевых значений.

Метод `sns.histplot()` используется для построения гистограмм — визуализаций для распределения одномерного набора данных. Метод отлично подходит для изучения формы распределения переменной. Его параметры:
- `data` — датафрейм с данными для построения;
- `x`, `y` — названия переменных в `data`, которые будут отображаться на осях;
- `bins` — количество столбцов гистограммы или границы бинов;
- `color` — цвет столбцов гистограммы.

Метод `sns.boxplot()` нужен для построения «ящика с усами» (boxplot), который используется для визуализации распределения данных через квартили и для выявления выбросов. Параметры:
- `data` — датафрейм с данными для построения;
- `x`, `y` — названия переменных в `data`, одна для категорий (ось Х) и одна для числовых значений (ось Y);
- `hue` — группирующая переменная, которая произведёт боксы с разными цветами;
- `palette` — набор цветов для отображения различных категорий.

# Столбчатые графики в seaborn. Практика

### Шпаргалка
В этом уроке вы будете работать со встроенным датасетом `flights` из библиотеки seaborn. Он содержит данные о количестве пассажиров авиалиний за разные месяцы в период с 1949 по 1960 годы.

Вот первые несколько строк этого датасета:

|**Index**|**year**|**month**|**passengers**|
|---|---|---|---|
|0|1949|Jan|112|
|1|1949|Feb|118|
|2|1949|Mar|132|
|3|1949|Apr|129|
|4|1949|May|121|

Для работы вам понадобятся функции:
- `sns.barplot()` — создание столбчатой диаграммы с помощью `seaborn`. Параметры:
    - `data` — данные в формате DataFrame, из которых будут взяты `x` и `y`;
    - `x` — данные на оси X, обычно категориальные;
    - `y` — данные на оси Y, обычно о количестве или среднем значении;
    - `hue` — параметр для разделения данных на подкатегории внутри каждой категории оси X;
    - `palette` — параметр для настройки цветовой схемы столбцов, принимает название цветовой палитры (`'deep'`, `'muted'`, `'bright'`, `'pastel'`, `'dark'`, `'colorblind'` и др.).
- `plt.xticks(rotation=)` — поворот меток на оси X для лучшей читаемости. В качестве аргумента принимает угол поворота в градусах.
- `plt.title()` — Добавление заголовка к графику. В качестве аргумента принимает текст заголовка.
- `plt.show()` — Отображение графика.
- `pd.DataFrame.groupby()` — Группировка данных по определенным критериям. Используется для агрегации данных перед визуализацией:
    - первый аргумент — имя столбца для группировки;
    - `as_index=False` — оставляет исходные индексы для удобства дальнейшего использования данных.
- `.mean()` — Расчет среднего значения по группам после применения `groupby()`
