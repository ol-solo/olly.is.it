Работа аналитика связана с расчётами и, пожалуй, это одни из самых часто повторяемых действий. Обработать данные, вычислить показатели, получить статистику и результаты внедрения процесса — для всего этого нужно последовательно выполнить математические операции.

Чем больше вопросов нужно решить в задаче и чем больше данных нужно для решения, тем сложнее формулы и вычисления. В Python много встроенных математических функций и есть библиотеки с математическим аппаратом — они автоматизируют, облегчают и ускоряют вычисления.

**Математическая функция** — это правило, по которому передаваемым параметрам ставится в соответствие выходной результат.

Это было формальное определение, теперь представим приземлённое.

- У нас есть аппарат, на вход которому передаётся число.
- Внутри аппарата происходит какое-то преобразование входного числа. Например, оно умножается на 2.
- Преобразованное число возвращается как результат.

Схематично наш аппарат-«умножатор» можно представить так:
![[Pasted image 20260304183715.png]]

В теме по основам синтаксиса вы научились выполнять с помощью Python простые операции: суммировать, вычитать, перемножать, делить и возводить числа в степень. Из этих операций складываются более сложные формулы.

Чтобы вспомнить, как работать с базовыми операциями, решим задачу.

**Задача.** Зарплата Павла была 70 денег в месяц. За год он сильно прокачал свои навыки в Python, смог автоматизировать многие рутинные процессы и стал справляться с задачами гораздо быстрее. В компании отметили достижения Павла и решили поощрить его новой зарплатой: 100 денег в месяц. С каждой зарплаты 13% удерживают как налог. Павел планирует бюджет понедельно и хочет понять, на сколько больше денег в неделю он стал получать?

Для простоты будем считать, что в месяце 4 недели.

**Решение.** Решить задачу можно несколькими способами. Мы предлагаем такой:

1. Найдём сумму, которая оставалась у Павла после оплаты налога со старой заработной платой. Для этого заявленную сумму нужно уменьшить на величину налога — 13%. Чтобы найти 13% от числа, нужно разделить это число на 100, а полученный результат умножить на 13.

```
old_salary = 70 # сумма до вычета налога
tax = 13 # налог
# сумма после вычета налога
old_profit = old_salary - old_salary/100 * tax 
print('Доход со старой зарплатой:',old_profit) 
```

Вывод:

```
Доход со старой зарплатой: 60.9 
```

2. Найдём сумму, которая остаётся у Павла после оплаты налога с новой заработной платой. Попробуйте самостоятельно повторить в Python действия — для новой зарплаты.
```
new_salary = 100 # сумма до вычета налога
tax = 13 # налог
# сумма после вычета налога
new_profit = new_salary - new_salary/100 * tax 
print('Доход с новой зарплатой:',new_profit) 
```

```
Доход с новой зарплатой: 87.0 
```

3. Переведём месячный доход в недельный. Для этого разделим фактические месячные доходы на количество недель, то есть на 4:

```

week_old_profit = old_profit/4
print('Старый доход в неделю:', week_old_profit) 
```

```
Старый доход в неделю: 15.225 
```

```
week_new_profit = new_profit/4 
print('Новый доход в неделю:', week_new_profit)
 
```

```
Новый доход в неделю: 21.75 
```

4. Осталось немного — определить разность между новым и старым доходом:

```
answer = week_new_profit - week_old_profit
print('С новой зарплатой доход Павла в неделю увеличится на:', answer)
 
```

```
С новой зарплатой, доход Павла в неделю увеличится на: 6.525 
```

Последний шаг — порадоваться за Павла: его недельный доход вырос на 6.525 денег.

Простые математические операции помогают решить и более сложные задачи, но бывает, наиболее удобная функция не встроена в Python, а находится в библиотеке.

## Как использовать библиотеки с математическими функциями

В прошлом уроке вы познакомились с двумя библиотеками, которые хранят математические функции: Math и Numpy.

Решим ещё одну задачу.

**Задача.** Компания «Феерические принтеры» производит детали для офисной техники. Одну из деталей создают так: берут шар, затем его пилят пополам. Площадь среза должна быть больше 7 см27 см2 и не превышать 10 см210 см2. Станки могут произвести шары таких диаметров: 1 см, 1,15 см, 2 см, 3,5 см, 5 см, 7 см. Какие шары подойдут?

**Решение.** Нам предстоит вычислить площади срезов доступных шаров и сравнить результаты с заданными ограничениями. Вариантов шаров много — есть смысл перейти к циклу. Освежить материал по циклам можно в теме «Циклы и их организация» в уроке «Понятие цикла. Элементы цикла».

**Вспомним формулы из геометрии.** Срез шара посередине — это круг, радиус которого равен радиусу самого шара. Площадь круга вычисляется по формуле: $S=π⋅R2$, где $S$ — площадь, $R$ — радиус, $π$ — константа. Радиус равен половине диаметра.
![[Pasted image 20260304184030.png]]
Получается, у нас есть 6 вариантов шаров, для каждого надо сделать три действия: по диаметру посчитать радиус, затем возвести его в степень и домножить на π.π. Итого 18 операций!
**Составим короткий алгоритм решения:**

- 1) Для доступного диаметра вычисляем радиус.
- 2) По формуле вычисляем площадь.

Здесь нам пригодится библиотека Math:

- В ней есть константа ππ: `math.pi`.
- Воспользуемся функцией возведения в степень: `math.pow(radius ,2)`, где `radius` — посчитанный радиус, `2` — степень, в которую возводим.

Тогда всю формулу $S=π⋅R2$ запишем так:

`math.pi * math.pow(radius ,2)`.

- 3) Повторяем действия для остальных радиусов — в цикле.

Напишем цикл:

```
import math

# создаём список с возможными диаметрами шара
values = [1, 1.15, 2, 3.5, 5, 7] 

# проходимся по списку диаметров
for value in values: 
    radius = value/2 # радиус - это половина диаметра
  
    # находим площадь круга по формуле
    s = math.pi * math.pow(radius ,2)
  
    # выводим результат
    print('диаметр:',value,', площадь:',s)  
```

```
# вывод
диаметр: 1 , площадь: 0.7853981633974483
диаметр: 1.15 , площадь: 1.0386890710931251
диаметр: 2 , площадь: 3.141592653589793
диаметр: 3.5 , площадь: 9.62112750161874
диаметр: 5 , площадь: 19.634954084936208
диаметр: 7 , площадь: 38.48451000647496 
```

Осталось учесть ограничения и выбрать подходящий шар. По условию площадь среза должна быть от 7 см^2 до 10 см^2. Выведенные результаты показывают, что в этом случае подойдёт только диаметр 3,5 см.
Задать условие поможет оператор `if`:

```
import math

# создаём список с возможными диаметрами шара
values = [1, 1.15, 2, 3.5, 5, 7] 

# проходим по списку диаметров
for value in values: 
  radius = value/2 # радиус - это половина диаметра
  
  # находим площадь круга по формуле
  s = math.pi * math.pow(radius ,2)
  
  # выводим результат
  if s > 7 and s <= 10:
      print('Подходящий диаметр:',value) 
  
> Подходящий диаметр: 3.5 
```

Так библиотека Math, цикл и условный оператор помогли нам избежать большого количества вычислений и быстро получить результат.

Особенность библиотеки NumPy в том, что она может работать сразу с массивом чисел. Например, чтобы возвести в степень массив чисел, не нужно писать цикл — можно применить функцию `.power(массив, степень)`. Она пройдёт по элементам массива и возведёт каждый в нужную степень.
Сначала создадим массив значений, с которым будем работать. Нам поможет `numpy.arange()`. Эта функция:

- принимает три параметра: начальное и крайнее значения, а также шаг,
- создаёт массив, в котором находятся элементы от начального до крайнего значений с заданным шагом.

```
import numpy as np

# определяем начало и конец диапазона, шаг
start, stop, step = 1, 5, 1

# создаём массив значений, с которым будем работать
array = np.arange(start, stop+step, step)

print('Исходный массив:', array)
 
```

```
Исходный массив: [1 2 3 4 5] 
```

Если указать последнее число, которое мы хотим видеть в массиве, как 3, а шаг 0,5, то получится массив `[1. , 1.5, 2. , 2.5, 3. ]`.

Теперь возведём все элементы массива в 3 степень:

```
# возводим каждый элемент в 3 степень
print('Возведение в степень 3:', np.power(array, 3)) 
```

```
Возведение в степень 3: [1 8 27 64 125] 
```

Вот так одной строкой мы возвели в куб элементы всего массива.

Также одной формулой можно выполнять и другие операции над элементами целого массива. Для примера извлечём квадратный корень при помощи `.sqrt(массив)` и посчитаем логарифм каждого числа `.log(массив)`:

```
# извлекаем квадратный корень из каждого элемента
print('Квадратный корень:', np.sqrt(array)) 
```

```
Квадратный корень: [1. 1.41421356 1.73205081 2. 2.23606798] 
```

```
# вычисляем натуральный логарифм каждого элемента
print('Натуральный логарифм:', np.log(array)) 
```

```
Натуральный логарифм: [0. 0.69314718 1.09861229 1.38629436 1.60943791] 
```

Вот так одним взмахом функции NumPy мы рассчитали много всего сложного.

Библиотека NumPy специально адаптирована для сложных вычислений.

Давайте. Например, преобразуем каждый элемент массива по намеренно усложнённой формуле x3−x4⋅ln⁡xx3−x​4⋅lnx.

```
# совместить полученные результаты:
print('Комбинация для каждого элемента', 
      np.power(array, 3) - 
      np.sqrt(array) ** 
      (4 * np.log(array))) 
```

```
Комбинация для каждого элемента [ 0. , 5.38593618, 15.82255485, 17.30560615, -52.786037 ] 
```

И снова быстро и без циклов мы преобразовали по сложным формулам элементы всего массива — и всё благодаря возможностям NumPy.

Подведём итоги:

- Математическая функция — правило, по которому передаваемым параметрам ставится в соответствие выходной результат.
- Соберём в таблицу изученные в уроке функции:
|**Функция**|**Описание**|
|---|---|
|`math.pow(число, степень)`|Возводит `число` в `степень`|
|`math.pi`|Хранит значение константы ππ|
|`numpy.arange(начальное_значение, конечное_значение, шаг)`|Создаёт массив чисел от `начального_значения` до `конечного_значения` с указанным `шагом`|
|`numpy.power(массив, степень)`|Возводит каждый элемент `массива` в `степень`|
|`numpy.sqrt(массив)`|Извлекает квадратный корень из каждого элемента `массива`|
|`numpy.log(массив)`|Применяет натуральный логарифм к каждому элементу `массива`|
