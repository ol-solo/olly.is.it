# Переменные. Практика
## Шпаргалка
Переменная — это имя для объекта.
Когда вы первый раз указываете название переменной, это называется объявить переменную. Это можно сделать с помощью оператора присваивания `=`.

Чтобы вывести значение переменной на экран, используйте функцию `print()`.
`end` — аргумент, в котором можно задать текст, который будет добавлен в конце вывода строки.
`sep` — аргумент, в котором можно задать разделитель между значениями.
Внутри строк `\n` — это специальный символ перевода на следующую строку.

# Арифметические операции. Практика
## Шпаргалка
Арифметические операции в Python:
- `+` — сложение,
- `-` — вычитание,
- `*` — умножение,
- `/` — деление;
- `//` — деление нацело;
- `%` — остаток от деления;
- `**` — возведение в степень.
- 
`items -= 5` — то же самое, что и `items = items - 5`;
`items /= 3` то же, что `items = items / 3` — и так далее.

Переменным можно присваивать числовые значения, и с ними же можно проделывать арифметические операции:
`desktop_users = 5402`
`mobile_users = 1103`
`all_users = desktop_users + mobile_users`
`mobile_share = mobile_users / all_users`

# Преобразование типов. Практика
## Шпаргалка
Основные функции для преобразования типов в Python:
- `int()` — превращает объект в целое число;
- `float()` — превращает объект в вещественное число;
- `str()` — превращает объект в строку.

Если в результате операций ожидается число, то приведите все переменные к числу, используя `float()` или `int()`. Если ожидается строка, то приведите всё к строке, используя `str()`.

Логические данные — это те, которые принимают значения True или False. Если преобразовать логическую переменную к числу, то True — это всегда 1, а False — всегда 0. А если преобразовать в строку, то получим `'True'` и `'False'` соответственно.

Если преобразовать число к типу `bool`, то не нулевое число станет `True`. Это работает как с целыми, так и с вещественными числами.

# Строки. Практика
## Шпаргалка
Для подсчёта длины используют функцию `len()`. В качестве аргумента `len()` принимает строку, а возвращает её длину — целое число.

К каждому символу можно обращаться по его индексу, а для указания индекса используют квадратные скобки `[]`.

Срезы позволяют получить сразу несколько идущих подряд символов строки. Чтобы получить срез, используют конструкцию: `[start:end:step]`.

Несколько полезных методов:
- `replace()` — заменит часть строки;
- `find()` — ищет подстроку в строке;
- `upper()` — заменяет все буквы в строке на прописные;
- `lower()` — заменяет все буквы на строчные.

Для форматирования строки можно использовать метод `format()`. В строке в фигурных скобках `{ }` указывают название переменной для подстановки.

Иногда удобнее использовать `f-строки`: напишите символ `f` перед строкой и точно так же используйте фигурные скобки `{ }` и именованные переменные, но без метода `format`.

# Списки. Практика
## Шпаргалка
Для создания списка используют квадратные скобки `[ ]`, а элементы списка перечисляют через запятую. Список может содержать другие списки, тогда его называют вложенным. Например: `items_list = [[1, 2, 3], [1, 2, 3, 4], [5, 6]]`.

Список — это отдельный тип объекта `list`.
Каждый элемент имеет свой порядковый номер или индекс. Как и в строках, нумерация начинается с `0`.
С помощью срезов обращаются к диапазону элементов с помощью конструкции `[start:end:step]`.
Изменить элемент в списке можно, обращаясь к нему по индексу.
Чтобы добавить элемент в список — вызовите метод `append()`. Для удаления элемента используйте `pop()`.
Один из способов расширить список — вызвать метод `extend()`. На вход он принимает другой список, которым нужно расширить изначальный.
Объединить два списка можно также при помощи оператора `+`.
Чтобы узнать длину списка, используйте функцию `len()`.

# Логика. Практика
## Шпаргалка
Логическое выражение — выражение, в котором сравниваются данные.

Результатом логического выражения могут быть только два логических значения:
- `True` — указывает на истинность выражения,
- `False` — подразумевает, что выражение ложно.

Логическое выражение с одним оператором сравнения называют простым логическим выражением.

Часто используемые операторы сравнения:

|**Логический оператор**|**Значение**|**Действие логического оператора**|**Когда возвращает `True`**|**Пример выражения с результатом `True`**|
|---|---|---|---|---|
|`==`|равно|сравнивает два значения между собой|когда значения равны|`5 == 5`|
|`!=`|не равно|сравнивает два значения между собой|когда значения не равны|`4 != 5`|
|`>`|больше|сравнивает два значения между собой|если первое больше второго|`5 > 1`|
|`<`|меньше|сравнивает два значения между собой|если первое меньше второго|`1 < 5`|
|`>=`|больше или равно|сравнивает два значения между собой|если первое больше второго или значения равны|`5 >= 1`  <br>`5 >= 5`|
|`<=`|меньше или равно|сравнивает два значения между собой|если первое меньше второго или значения равны|`1 <= 1`  <br>`5 <= 5`|

# Сложные логические выражения. Практика
## Шпаргалка
Логические операторы:
 - Оператор `and` возвращает `True`, только если оба логических значения — `True`. В остальных случаях возвращает `False`.
 - Оператор `or` возвращает `True`, если **хотя бы** одно логическое значение тоже `True`. В случае, если оба значения `False`, возвращает `False`.
 - Оператор `not` меняет одно значение на противоположное.

Таблица истинности логических операторов

|**Выражение А**|**Выражение В**|**A and B**|**A or B**|**not A**|**not B**|
|---|---|---|---|---|---|
|True|True|True|True|False|False|
|True|False|False|True|False|True|
|False|True|False|True|True|False|
|False|False|False|False|True|True|

Битовые операторы

|**битовый оператор**|**значение**|
|---|---|
|`&`|и (`and`)|
|`\|`|или (`or`)|
|`~`|не (`not`)|
|`^`|исключающее или|

Порядок выполнения операторов
1. операции в скобках;
2. возведение в степень `**`;
3. битовый оператор `~`;
4. математические операторы — умножение `*`, деление `/`, сложение `+`, вычитание `-`;
5. битовые операторы `&` и `|`;
6. операторы сравнения — `<`, `<=`, `>`, `>=`, `!=`, `==`;
7. логические операторы сравнения `not`, `and` и `or`.

Операции с одинаковым приоритетом выполняются последовательно слева направо.

# Циклы. Практика
## Шпаргалка
**Цикл** — управляющая конструкция, многократно исполняющая определённый набор инструкций.

Часто применяемый цикл Python для анализа данных — это `for`. Он повторяет заданную последовательность инструкций для каждого итерируемого объекта.

**Заголовок цикла** — его первая строка, которая должна содержать:
- ключевое слово `for`;
- переменную-итератор, которая перемещается по множеству значений слева направо, принимая на каждом шаге значение очередного элемента списка;
- ключевое слово `in`;
- множество значений, по которому движется итератор;
- знак `:`. Без него будет синтаксическая ошибка, а сам цикл не будет объявлен.

**Тело цикла** — набор инструкций, которые выполняются на каждом шаге цикла. В теле должна быть как минимум одна инструкция.

Циклом можно перебирать строки. В этом случае переменная-итератор будет последовательно принимать значение каждого элемента строки.

Такой формат используется для перевода данных с плавающей точкой в целочисленный тип, когда происходит умножение значений на 10 или 100.

Скорректируйте значения температур списка `temperature`, разделив каждое значение на 10. Новые значения сохраните в исходном списке и затем выведите их вместе на экран. Для корректировки значений используйте цикл `for`.
```python
temperature = [-77, -83, -43, 7, 65, 119, 162, 147, 104, 39, -5, -56]

for i in range(len(temperature)):
    temperature[i] = temperature[i] / 10

print(temperature) # напишите цикл for здесь
# выведите на экран обновлённый список температуры
```

# Циклы и строки. Практика
## Шпаргалка
К буквам или любым другим символам строки можно обращаться через их индексы.

С помощью функции `range()` можно выбрать только часть строки.

Метод `append` позволяет добавлять элемент в конец списка, к которому метод применяется.

## Задание 1
Создайте с помощью цикла `for` новую строку, которая будет содержать только часть исходной строки `text` с инвентарным номером товара. Используйте индексы чтобы подсчитать начало и конец значений строки.
```python
text = 'Фанера ФК 4 мм, размер 1525х1525 мм, сорт 3/4, инв.н. 354228, 5 шт.'

number = ''

right = len(text) - 7      # отбрасываем запятую и пробел перед "5 шт."
left = len(text) - 13      # ещё 6 символов номера 

for i in range(left, right):
    number += text[i]

print(number)  # 354228
```
